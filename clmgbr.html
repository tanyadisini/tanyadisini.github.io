7. Tuliskan reaksi elektrolisis yang terjadi jika lelehan MgBr 2 dielektrolisis dengan elektroda Emas (Au)​


Jawaban:

Reaksi elektrolisis dari :

1. Lelehan MgBr₂ dengan elektroda C

Katoda  :  Mg²⁺  + 2e  ⇒ 2Mg

Anoda   : 2Br⁻ ⇒ Br₂ + 2e

_________________________________ +

              Mg²⁺ + 2Br⁻ ⇒ 2Mg + Br₂

Ketentuan :  

Anoda : Kation langsung direduksi

Katoda : Anion langsung dioksidasi

Karena dalam bentuk lelehan dengan elektroda inert maka kation dan anion dapat langsung di reduksi dan oksidasi.

2. Larutan ZnSO₄ dengan elektroda Ag

Katoda  :  Zn²⁺  + 2e  ⇒ 2Zn     ║×1

Anoda   : Ag⁺ ⇒ Ag + e              ║×2

Katoda  :  Zn²⁺  + 2e  ⇒ 2Zn

Anoda   : 2Ag ⇒ 2Ag⁺ + 2e

_________________________________ +

              Zn²⁺ + 2Ag ⇒ 2Zn + 2Ag⁺

Ketentuan :

Katoda : Kation golongan transisi yang direduksi adalah kation tersebut menjadi : Mˣ⁺ + 2e ⇒  M.

Anoda : Yang di oksidasi adalah elektroda tersebut tanpa memperhatikan anionnya.

Zn merupakan unsur golongan transisi sehingga direduksi menjadi kationnya. Sedangkan Ag merupakan elektroda non inert sehingga dapat langsung di oksidasi.

26. Bang mo tanya dong apakah clogb/cloga = c log (b-a) tolong di bantu beserta penjelasanya​


Jawab:

Tidak, melainkan ᶜlog b / ᶜlog a = ᵃlog b

Penjelasan dengan langkah-langkah:

Definisi logaritma dengan eksponen ᵃlog b = c ↔ aᶜ = b

Misal ᵃlog b = m↔ aᵐ = b

ᶜlog aᵐ = ᶜlog b

m ᶜlog a = ᶜlog b

m = ᶜlog b / ᶜlog b

ᵃlog b = ᶜlog b / ᶜlog b

28. membuktikan sifat logaritma alogb=clogb per cloga


Misal  A log B = x
B = A pangkat x
^C log B = ^C log A^x
^C log B = x ^C log A
x = ^C log B per ^C log A
jadi, A log B = ^C log B per ^C log A

34. Gambarkan rumus lewis dan senyawa berikut dan tentukan jenis ikatannya masing masing. A.C2H6 B.KBr C.MgBr2 D.HCI E.PCI3


Rumus lewis dari senyawa dapat digunakan untuk menggambarkan dan  menentukan jenis ikatan masing - masing senyawa .

Rumus lewis adalah lambang unsur yang disertai dengan jumlah elektron valensi.

Pembahasan

Pembentukan senyawa merupakan salah satu cara dari suatu atom dalam mencapai kestabilannya. Unsur yang stabil memiliki elektron valensi sejumlah 8 seperti konfigurasi dari unsur gas mulia, atom-atom saling bergabung. Penggabungan atom-atom karena adanya gaya tarik menarik yang disebut dengan ikatan kimia.

Ada dua cara pembentukan ikatan kimia, yaitu :  

Ikatan ion : penggabungan suatu atom dengan cara serah terima elektron, melalui pembentukan kation dan anion.Ikatan kovalen : penggabungan suatu atom dengan jalan pemakaian bersama pasangan elektron.A. Senyawa C₂H₆

Untuk molekul C₂H₆ yang terbentuk antara 2 atom C, dan 6 atom H, dengan konfigurasi elektron unsur pembentuknya :

C : Golongan IV-A (elektron valensi 4) - Non logam

H : Golongan I-A (elektron valensi 1) - Non logam

Pembentukan ikatan antara unsur logam dengan unsur non logam adalah ikatan kovalen.

        H        H                        H   H

       xо       xо                        |     |  

H оx C  xx  C  xо H     →  H - C - C - H

       xо       xо                         |     |

        H        H                         H    H

B. Senyawa KBr

Pembentukan senyawa antara unsur K dan Br adalah sebagai berikut :

K, unsur dari golongan IA dengan elektron valensi 1 termasuk golongan logam membentuk kation : K → K⁺ + 1e⁻Br, unsur dari golongan VII-A dengan elektron valensi 7 termasuk golongan non logam membentuk anion : Br + 1e⁻ → Br⁻

Pembentukan ikatan antara unsur logam dengan non logam adalah ikatan ionik.

K → K⁺ + 1e⁻

Br + 1e⁻ → Br⁻

___________ +

K + Br → K⁺ + Br⁻ {KBr}

C. Senyawa MgBr₂

Pembentukan senyawa antara unsur Mg dan Br adalah sebagai berikut :

Mg, unsur dari golongan IIA dengan elektron valensi 2 termasuk golongan logam membentuk kation : Mg → Mg²⁺ + 2e⁻Br, unsur dari golongan VII-A dengan elektron valensi 7 termasuk golongan non logam membentuk anion : Br + 1e⁻ → Br⁻

Pembentukan ikatan antara unsur logam dengan non logam adalah ikatan ionik.

Mg → Mg²⁺ + 2e⁻   | × 1    Mg → Mg²⁺ + 2e⁻

Br + 1e⁻ → Br⁻         | × 2   2Br + 2e⁻ → 2Br⁻

_____________            ______________+

                                       Mg + 2Br → Mg²⁺ + 2Br⁻ {MgBr₂}

D. HCl

Untuk molekul HCl yang terbentuk antara 1 atom H, dan 1 atom Cl, dengan konfigurasi elektron unsur pembentuknya :

H : Golongan IA (elektron valensi 1) - Non logam

Cl : Golongan VIIA (elektron valensi 7) - Non logam
