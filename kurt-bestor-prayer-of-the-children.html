24. 182 sweets are equally divided among a certain number of children at a party. If the number of sweets each child receives is one more than the number of children, find the number of children at the party.


Misalkan jumlah anak = x
maka tiap anak akan mendapat bagian = (x + 1) sweets

dengan demikian:

x(x + 1) = 182
x² + x - 182 = 0
(x -13)(x + 14) = 0
x = 13 atau x = -14 (tidak memenuhi)

Jadi jumlah anak di dalam party adalah 13 anak
Anggap Jumlah anak adalah a

Maka :
a(a + 1) = 182
a² + a = 182
a² + a - 182 = 0
(a - 13)(a + 14) = 0

a = 13  atau  a = -14
Karena, jumlah anak tidak mungkin minus
maka jumlah anak adalah 13

Semoga membantu :)
