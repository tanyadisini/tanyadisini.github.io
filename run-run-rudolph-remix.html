26. Apa pengertian boomerang run


Apa pengertian boomerang run, boomerang run adalah suatu gerakan pada olahraga untuk mengukur kelincahan seseorang. Boomerang run bisa dilakukan dimana saja, boomerang run berfungsi untuk mengukur kelincahan seorang atlet. Boomerang run dilakukan dengan cara memasang tiga titik dari titik mulai berlari, yaitu di seberang gari mulai dengan jarak sekitar 5 meter, kemudian dari garis mulai tanda lain dipasang dengan jarak yang hampir sama yaitu 5 meter di sebelah kiri dari titik pertama dan titik terakhir adalah di sebelah kanan dari titik mulai. Gerakan boomerang run dimulai dengan berlari ke arah titik pertama, kemudian berlari kembali ke arah titik yang kedua, kemudian ke titik ketiga, setelah itu kembali memutari titik pertama dan kembali ke garis awal.

Demikian pembahasan mengenai boomerang run, boomerang run memiliki fungsi untuk melatih kelincahan. Selain boomeran run, kita juga bisa melakukan zig zag run.
