4. Perbandingan tinggi badan Ariq dan Barik adalah 3 : 7, sedangkan perbandingan tinggi badan Barik dan Yusuf adalah 7 : 9. Berapakah perbandingan tinggi badan Ariq, Barik, dan Yusuf?tolong jawab kak​


Misalkan:

Arik (A)

Barik (B)

Yusuf (C)

Perhatikan perbandingan A dan B

A : B = 3 : 7

Perhatikan juga perbandingan B dan C

B : C = 7 : 9

Dari pernyataan di atas kita memperoleh nilai perbandingan dari masing - masing pihak

A = 3

B = 7

C = 9

Maka, A : B : C = 3 : 7 : 9

Jadi, perbandingan tinggi badan Ariq, Barik, dan Yusuf

adalah 3 : 7 : 9

Pupuk padi (P) : pupuk jagung (J) = 7 : 5
Pupuk padi (PP) = 84 kg
Pupuk jagung (PJ)?

P/J = PP/PJ
7/5 = 84/PJ
PJ = 84 x 5/7
PJ = 12 x 5
PJ = 60 kg pupik utk jagung
