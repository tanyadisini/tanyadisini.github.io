36. tentukan jumlah dari deret berikut ini.a. 47+49+51+...+101b. 72+70+68+...+40


Matematika
Pola, Barisan, dan Deret

a) 47 + 49 + 51 + ... + 101

a = U1 = 47
b = U2 - U1 = 49 - 47 = 2

Un = a + (n - 1)b = 47 + 2(n - 1) = 45 + 2n

101 = 45 + 2n
56 = 2n
n = 28
101 = U28

Sn = n/2 × (a + Un)
S28 = 28/2 × (47 + 101) = 14 × 148 = 2072

b) 72 + 70 + 68 + ... + 40

a = U1 = 72
b = U2 - U1 = 70 - 72 = -2

Un = a + (n - 1)b = 72 - 2(n - 1) = 74 - 2n

40 = 74 - 2n
2n = 74 - 40 = 34
n = 17
40 = U17

Sn = n/2 × (a + Un)
S17 = 17/2 × (72 + 40)
= 17/2 × 112
= 952
