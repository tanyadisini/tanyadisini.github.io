5. nilai dari ³log 1/81 + ³log 24 - ³log 30 adalah​


Jawaban:

Sehingga, nilai dari ³log 1/81 + ³log 24 - ³log 30 = -log₁₈ (67.5).

Penjelasan dengan langkah-langkah:

Untuk menyelesaikan ekspresi ³log₁₈₁ + ³log₂₄ - ³log₃₀, kita dapat menggunakan properti-logaritma dan aturan logaritma untuk menyederhanakan ekspresi tersebut.

Pertama, gunakan properti-logaritma log aⁿ = n log a, untuk menyederhanakan ekspresi:

³log₁₈₁ + ³log₂₄ - ³log₃₀ = log₁₈ (1/81) + log₂ (24) - log₃ (30)

Kedua, gunakan aturan logaritma log a + log b = log (a * b), dan log a - log b = log (a/b):

log₁₈ (1/81) + log₂ (24) - log₃ (30) = log₁₈ [(1/81) * 24 / 30]

Ketiga, kita dapat menyederhanakan logaritma pada basis yang sama:

log₁₈ [(1/81) * 24 / 30] = log₁₈ (24/1620)

Keempat, gunakan aturan logaritma log aⁿ = n log a, untuk menyederhanakan ekspresi:

log₁₈ (24/1620) = log₁₈ (1/67.5)

Kelima, gunakan aturan logaritma log a/b = log a - log b:

log₁₈ (1/67.5) = log₁₈ 1 - log₁₈ 67.5

Keenam, kodekan log dalam basis yang sama:

log₁₈ (1/67.5) = 0 - log₁₈ (67.5)

Terakhir, gunakan aturan logaritma log aⁿ = n log a:

0 - log₁₈ (67.5) = -log₁₈ (67.5)

Sehingga, nilai dari ³log 1/81 + ³log 24 - ³log 30 = -log₁₈ (67.5).

9. tentukan dP/dQ dan dQ/dP ketika 30P+30-81 = 0​


Jawaban:

Diketahui :

Qd = 4 - P

Qd = 3

Ditanya :

Ed = ?

Jawab :

jika kita mengetahui fungsi permintaan dalam bentuk Qd = a - bP, maka kita dapat menghitung koefisien elastisitas permintaan dengan menggunakan rumus :

Ed = Q' × P/Q

ket :

Ed = Koefisien elastisitas permintaan

P = tingkat harga

Q = kuantitas pada tingkat harga tertentu

Q’ = turunan pertama dari fungsi

karena kita belum mengetahui harganya (P) maka kita cari dahulu dengan mensubtitusikan Q yang telah kita ketahui ke dalam fungsi permintaan :

Qd = 4- P

3 = 4 - P

P = 4 - 3

P = 1

kemudian kita cari juga turunan pertama fungsi permintaannya :

Q = 4 - P

Q' = -1

setelah mengetahui harganya (P), barulah kita dapat menghitung koefisien elastisitas permintaannya :

Ed = Q' × P/Q

Ed = -1 × 1/3

Ed = -0,34

gunakan nilai mutlak, Jadi, Ed = 0,34

Pembahasan

Pengertian Elastisitas Permintaan

Elastisitas permintaan adalah sebuah ukuran seberapa besar derajat kepekaan permintaan terhadap perubahan harga.

Jenis-jenis Elastisitas Permintaan

Elastisitas permintaan ada 5 jenis, yaitu:

Permintaan elastis, yaitu apabila koefisien elastisitasnya lebih besar dari 1 (Ed > 1)

Permintaan inelastis, yaitu apabila koefisien elastisitasnya lebih kecil dari 1 (Ed < 1)

Permintaan elastis uniter, yaitu apabila koefisien elastisitasnya sama dengan 1 (Ed = 1)

Permintaan elastis sempurna, yaitu apabila koefisien elastisitasnya sama dengan tidak terhingga (Ed = ∞)

Permintaan inelastis sempurna, yaitu apabila koefisien elastisitasnya sama dengan nol (Ed = 0)

Menghitung Koefisien Elastisitas Permintaan

rumus untuk menghitung koefisien elastisitas permintaan adalah :

Ed = ∆Q/∆P×P/Q

Ket :

Ed : koefisien elastisitas permintaan

ΔP : perubahan harga (selisih P1 dan P2)

ΔQ : perubahan kuantitas (selisih Q1 dan Q2)

P : harga awal

Q : kuantiatas awal yang diminta

Jika diketahui fungsi permintaan, maka koefisien elastisitasnya di hitung dengan cara :

Jika diketahui fungsi permintaan Qd=a –bP, maka rumusnya: Ed=Q'×P/Q

Jika diketahui fungsi permintaan Pd=a –bQ, maka rumusnya: Ed=1/P'×P/Q

Ket :

Ed = Koefisien elastisitas permintaan

Q = kuantitas pada tingkat harga tertentu

Sifat bilangan ada 3 yaitu

1. Sifat komunitatif atau pertukaran

contohnya a + b = b + a

c × d = d × c

2. Sifat asosiatif atau pengelompokkan

contohnya (a + b) + c = a + (b + c)

(a × t) × m = a × (t × m)

3. Sifat distribusi atau penjabaran
