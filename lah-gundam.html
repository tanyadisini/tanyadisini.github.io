________

• Menentukan nilai a dan b

a + b = 15

a = 7,5 dan b = 7,5

...

• Menentukan nilai c

b + c = 25

7,5 + c = 25 ➞ Substitusikan nilai b

c = 25 - 7,5

c = 17,5✔

...

• Menentukan nilai c - a

= c - a

= 17,5 - 7,5

= 10✔

Kesimpulan

Jadi, Nilai c - a adalah 10✔
