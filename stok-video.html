Penjelasan dengan langkah-langkah:

barisan aritmatika

Un = a + ( n - 1 ) b

i. U₂ + U₄ = 26

ii. U₈ - U₅ = 9

dari ( i ) ;

a + b + a + 3b = 26

2a + 4b = 26

a + 2b = 13

dari ( ii ) ;

a + 7b - ( a - 4b ) = 9

a + 7b - a + 4b = 9

11b = 9

b = 9/11

a + 2b = 13

a + 2(9/11) = 13

a + 18/11 = 13

a = 13 - 18/11

a = 125/11

U₁₀ = 125/11 + ( 10 - 1 ) 9/11

.      = 125/11 + ( 9 ) 9/11

.      = 125/11 + 81/11

23. Contoh sda stok dan flow


Secara umum sumber daya alam dapat diklasifikasikan ke dalam dua kelompok yang disebut sebagai kelompok stock dan kelompok flow. Pada kelompok stok sumber daya ini dianggap memiliki cadangan yang terbatas sehingga eksploitasi terhadap sumber daya tersebut akan menghabiskan cadangan sumber daya. Apa yang kita manfaatkan sekarang mungkin tidak lagi tersedia di masa mendatang. Dengan demikian sumber daya stok tidak dapat diperbaharui (non reneweble) atau terhabiskan (exhaustible). Termasuk dalam kelompok ini antara lain sumber daya mineral, logam, minyak dan gas bumi.

32. Stok pupuk untuk petani setiap minggunya membentuk barisan aritmatika jumlah stok pada Minggu ke-2 dan ke-4 adalah 26 ton namun selisih stok pada minggu ke-8 dan ke-5 adalah 9 ton banyaknya stok pupuk pada minggu ke-10 adalah??


Jawab:

stok pupuk pada minggu ke-10 adalah 50 ton

Penjelasan dengan langkah-langkah:

Minggu ke-2 dan ke-4

64 = a + b + a + 3b

64 = 2a + 4b ..... (i)

minggu ke-8 dan ke-5

9 = a + 7b - a - 4b

9 = 3b ..... (ii)

Eliminasi persamaan (i) dan (ii)

64 = 2a + 4b    | x 3 | 192 = 6a + 12b

9 =           3b -  | x 4 | 36 =           12b -

                                 ---------------------

                                  156 = 6a

                                  156/ 6 = a

                                  26       = a

Subtitusika a = 26 ke persamaan (i)

64 = 2a + 4b

64 = 2(26) + 4b

64 = 52 + 4b

64 - 52 = 4b

12 = 4b

12/4 = b

b = 3

a = 26 dan b = 3

Un = a + (n - 1) b

U10 = 26 + (9 - 1) 3

U10 = 26 + 8 x 3

U10 = 26 + 24

U10 = 50 ton

Jadi, stok pupuk pada minggu ke-10 adalah 50 ton
