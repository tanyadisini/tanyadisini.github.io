Harga barang asli adalah $1720

Diskon yg didapatkan ialah 30%

Nilai diskon yaitu = [tex]\frac{30}{100}[/tex] x $1720

Nilai diskon yaitu = $516

Maka harga yg harus dibayarkan = $1720 - $516

Maka harga yg harus dibayarkan = $1204

27. Sbuah mini market berukuran panjang 27 m, lebar 18 m dan tinggi 9 m. jika mini market tersebut dibuat model dengan ukuran lebAR 6 CM, maka panjang dan tinggi paada model adalah.....


Mapel : Matematika
Kelas : IX SMP
Bab : Kesebangunan dan Kekongruenan

Pembahasan :
Pakai Kesebangunan

Panjang sebenarnya / Panjang model = Lebar sebenarnya / Lebar Model
27/Panjang Model = 18/0,06
Panjang Model = 27 x 0,06 / 18
Panjang Model = 0,09 m
Panjang Model = 9 cm

Panjang sebenarnya / Panjang model = Tinggi sebenarnya / Tinggi model
27/0,09 = 9/Tinggi Model
Tinggi Model = 9 x 0,09 / 27
Tinggi Model = 0,03 m
Tinggi Model = 3 cm

32. harga 3 laptop dan 2 ipad adalah Rp. 17.000.000,- sedangkan harga 2 laptop dan 5 ipad adalah Rp.20.500.000,-. tentukanlah harga 1 laptop dan harga 1 ipad

pers. 1 --> 3L + 2P = 17.000.000  dikali 2
pers. 2 --> 2L + 5P = 20.500.000  dikali 3

6L + 4P   = 34.000.000
6L + 15P = 61.500.000
-------------------------------  -
       -11P = -27.500.000
         P = 2.500.000

3L + 2P = 17.000.000
3L  = 17.000.000 - 2P
3L  = 17.000.000 - 2(2.500.000)
3L  = 17.000.000 - 5.000.000
3L  = 12.000.000
L = 4.000.000

Harga 1 laptop = 4.000.000
harga 1 ipad 2.500.000
