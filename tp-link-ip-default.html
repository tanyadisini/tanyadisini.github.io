2. Ada TP-LINK tl-wr840n.1. Sebutkan cara setting IP Network!2. Sebutkan cara setting IP LAN!3. Sebutkan cara setting routing!​


1. Cara setting IP Network pada TP-LINK TL-WR840N:

Buka browser web dan masuk ke halaman admin router dengan mengetik alamat IP router (biasanya 192.168.0.1 atau 192.168.1.1) di URL.Masukkan username dan password default yang tercantum pada manual atau label pada router.Pilih "Network" atau "LAN" dari menu utama.Ubah pengaturan IP Network sesuai kebutuhan, misalnya dengan mengubah alamat IP, subnet mask, dan gateway.Klik "Save" atau "Apply" untuk menyimpan perubahan.

2. Cara setting IP LAN pada TP-LINK TL-WR840N:

Buka browser web dan masuk ke halaman admin router.Masukkan username dan password default.Pilih "LAN" dari menu utama.Ubah pengaturan IP LAN sesuai kebutuhan, misalnya dengan mengubah alamat IP dan subnet mask.Klik "Save" atau "Apply" untuk menyimpan perubahan.

3. Cara setting routing pada TP-LINK TL-WR840N:

Buka browser web dan masuk ke halaman admin router.Masukkan username dan password default.Pilih "Routing" dari menu utama.Pilih tipe routing yang dibutuhkan, seperti Static Routing atau Dynamic Routing.Masukkan informasi yang dibutuhkan untuk setiap tipe routing, misalnya destination IP, subnet mask, gateway, dan metric.Klik "Save" atau "Apply" untuk menyimpan perubahan.

note: Langkah-langkah ini mungkin berbeda tergantung dari versi firmware router. Pastikan untuk membaca manual pengguna atau dokumentasi produk untuk informasi lebih rinci.

13. Default Subnetmask ip addres clas c adalah?


Dalam istilah teknologi informasi, subnet mask dan IP address adalah hal yang berbeda tapi serupa. Pada subnetmask sendiri adalah pembeda antara Network ID dengan Host ID dan sebagai penentu terhadap tujuan pengiriman paket-paket data yang dikirimkan, apakah local atau remote. Kemudian, pengertian dari IP address sendiri adalah sebuah alamat pada sebuah device yang terhubung pada sebuah jaringan, kegunaannya sendiri adalah agar device dapat berkomunikasi dengan device lainnya. 

Dalam penjabarannya, kelas-kelas IP address dibagi kedalam 3 kelompok yang dibuat berdasarkan subnet mask default. Berikut adalah subnet mask default dari masing-masing IP kelas berdasarkan biner dan desimal:

- IP Address Kelas A
Subnet Mask (biner) : 11111111.00000000.00000000.00000000
Subnet Mask (desimal) : 255.0.0.0

- IP Address Kelas B
Subnet Mask (biner) : 11111111.11111111.00000000.00000000
Subnet Mask (desimal) : 255.255.0.0

- IP Address Kelas C
Subnet Mask (biner) : 11111111.11111111.11111111.00000000
Subnet Mask (desimal) : 255.255.255.0

Kesimpulan:
Default subnet mask untuk IP address kelas C adalah 255.255.255.0

18. 1. diketahui ip address : 203.202.101.10 tentukan : a. konversi ip tersebut ke bilangan biner (32 bit)! b. jenis kelas dari ip tersebut dan jelaskan! c. network id dan host id ! d. berapa subnet mask default dari ip tersebut!


Jawaban ringkas:

1. a. Hasil konversinya ialah:

11001011.11001010.01100101.00001010

b. termasuk IP kelas C

c. Network ID = 203.202.101

Host ID = 101

d. Subnet mask default = 255.255.255.0

Penjelasan panjang:

a. Bila dipisahkan, hasil konversinya yaitu:

203 = 11001011

202 = 11001010

101 = 1100101

10 =  1010

Karena yg diinginkan adalah konversi biner 32 bit, maka masing-masing angka di atas harus ditulis dg 8 digit. Bila digabung menjadi:

11001011.11001010.01100101.00001010

Cara perhitungan konversinya bisa dilihat di bagian lampiran.

b. Perlu diketahui bahwa IP Address memiliki kelas jaringan sesuai rentang alamatnya, antara lain:

1 - 126 : IP Kelas A

128 - 191 : IP Kelas B

192 - 223 : IP Kelas C

Karena soal di atas memperlihatkan 203 (sebelum titik pertama), maka dapat disimpulkan bahwa IP Address tersebut termasuk dalam rentang kelas C

c. Pada IP kelas C, bagian yg menjadi network ID ialah 3 angka pertama yg masing-masingnya dipisahkan tanda titik.

Sementara host ID ialah angka terakhir yg tampil. Jadi kesimpulannya:

Network ID = 203.202.101

Host ID = 10

d. Subnet mask default IP kelas C yaitu 255.255.255.0


19. Jelaskan perintah dalam router Cisco untuk mengatur default routes dengan IP gateway adalah 180.162.0.172/25 Bantu pliss


jawab

penjelasan ttg cra mengatur default routes dengan IP gateway adalah 180.162.0.172/25:

ip route 0.0.0.0 0.0.0.0 180.162.0.172

ip route : perintah untuk mengatur rute IP

0.0.0.0 0.0.0.0: nilai pertama dan ke-2 menunjukkan bahwa rute ini adlh default route artiny rute yg digunakan jika tidak ada rute spesifik yang cocok

180.162.0.172: IP address gateway yg digunakan sebagai next hop untuk mengirimkan paket yg tidak cocok dengan rute lainny.

dgn perintah diatas router akan mengirimkan paket menuju IP address 180.162.0.172 sebagai next hop apabila tidak ditemukan rute yg cocok untuk tujuan paket. hal ini memungkinkan jaringan terhubung ke Internet atau ke jaringan lainny diluar jaringan lokal.

30. langkah konfigurasi Access Point tp-link​


Telah terdapat langkah-langkah dalam konfigurasi Access Point TP-LINK yaitu ialah sebagai berikut:

Siapkanlah terlebih dahulu Access Point, kabel LAN, dan juga power dan setelah itu koneksikanlah kabel LAN pada komputer. Bukalah browser seperti Firefox, lalu ketikkanlah IP Address Access Point pada address bar lalu tekan enter.Masukkanlah username dan juga password dari Access Point. Kemudian, akan masuk ke halaman Access Point. Pada tempat ini telah digunakan untuk me-manage ataupun mengkonfigurasi segala macam pengaturan dan juga fungsi dari Access Point kemudian klik Next. Selanjutnya, pilihlah mode Access Point dan klik Next. Pada bagian selanjutnya yaitu ialah tempat untuk mengubah SSID dan juga mengisi password. Pada bagian network setting tidak perlu ada yang dirubah. Setelah itu, akan muncul halaman pemberitahuan maka dari itu pastikanlah semuanya telah benar dan pilih reboot. Kemudian, tunggulah hingga proses reboot telah selesai. Setelah proses reboot telah selesai maka dari itu kita dapat melihat laptop ataupun handphone untuk dapat mengetahui apakah wifinya telah berhasil dibuat. Apabila SSID yang telah diciptakan tersebut telah muncul maka dari itu proses konfigurasinya telah berhasil.Pembahasan

Pengertian dari TP-LINK yaitu ialah suatu perusahaan penyedia dunia untuk perangkat jaringan dan juga aksesoris yang dinilai terpercaya untuk dapat memenuhi segala macam aspek di dalam kehidupan kita sehari-hari.
