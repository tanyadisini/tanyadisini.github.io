31. Fungsi srtiap komponen dalam dardware komputer


ada banyak komponen dalam perangkat komputer, namun pada dasarnya diantaranya :

-Monitor, menampilkan gambar / Hasil output
-Keyboard, Menerima masukan berupa huruf. angka, perintah / Input
-Mouse, Menerima masukan perintah melalui kursor / Input
-CPU, Tempat Pemrosesan Informasi / Inputan
Power Supply, menyediakan sumber tenaga
Motherboard, adlah papan PCB yang menghubungkan masing" perangkat
VGA Card, berfungsi untuk menampilkan grafik / gambar
RAM, berfungsi sebagai tempat penyimapan data sementara
Prosesor, berfungsi sebagai otak komputer
Hardisk, berfungsi sebagai media penyimpanan file
CD Room, berfungsi sebagai pembaca CD

sekianproses device
-processor=melakukan proses program dalam komputer
-motherboard=menghubungkan seluruh komponen komputer
-ram=sebagai penyimpan data sementara
-vga=menampilkan grafis digital dari komputer ke monitor
-power supply=menghantarkan energi listrik ke seluruh bagian komputer
storage device
-memory=menyimpan data secara permanen namun bisa diolah oleh pengguna sendiri
input device
-keyboard=menginput huruf,angka,simbol,dll kedalam komputer
-mouse=memindahkan pointer/kursor dikomputer
output device
-monitor=menampilkan data teks/grafis
-printer=mencetak suatu data dikomputer kedalam bentuk kertas
