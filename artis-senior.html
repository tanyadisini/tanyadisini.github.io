29. kumohon senior senior yang baik cuma nomor 1 dan 3

[tex] log(2) = p[/tex]

[tex] log(3) = q[/tex]

[tex] log_{3}(12) = log_{3}(4 \times 3) [/tex]

[tex] log_{3}(12) = log_{3}(4) + log_{3}(3) [/tex]

[tex] log_{3}(12) = 1 + log_{3}(4) [/tex]

[tex] log_{a}(b) = \frac{ log_{c}(a) }{ log_{c}(b) } [/tex]

Sehingga

[tex] log_{3}(4) = \frac{ log(3) }{ log(4) } [/tex]

[tex] log_{3}(4) = \frac{ log(3) }{ log( {2}^{2} ) } [/tex]

[tex] log_{3}(4) = \frac{ log_{}(3) }{2 log(2) } [/tex]

[tex] log_{3}(4) = \frac{q}{2p} [/tex]

Maka hasil akhirnya

[tex] log_{3}(12) = 1 + log_{3}(4) [/tex]

[tex] log_{3}(12) = 1 + \frac{q}{2p} [/tex]

[tex] log_{3}(12) = \frac{1}{2p} (2p + q)[/tex]

[tex] log_{0.004}(0.125) = log_{ {250}^{ - 1} }( {2}^{ - 3} ) [/tex]

[tex] log_{0.004}(0.125) = ( - 1)( - 3)( log_{250}(2) )[/tex]

[tex] log_{0.004}(0.125) = 3( log_{250}(2) )[/tex]

[tex] log_{0.004}(0.125) = 3( \frac{1}{ log_{2}(250) } )[/tex]

[tex] log_{0.004}(0.125) = 3( \frac{1}{ log_{2}(25 \times 10) } )[/tex]

36. dibantu dong senior senior brainly​


Jawaban:

luas persegi:

=sisi × sisi

=14 × 14

=196 cm²

luas 3/4 lingkaran

=3/4 × πr²

=3/4 × 22/7 × 14 × 14

=462 cm²

luas keseluruhan

=196 cm² + 462 cm²

=658 cm²

s = 14 cm

r = 14 cm

L1 = luas persegi

maka,

L1 = s × s = 14 × 14 = 196 cm²

L2 = luas lingkaran

maka,

L2 = πr² = 22/7 . 14. 14 = 616 cm²

Luas gabungan :

L1 + L2

196 + 616 = 812 cm²
