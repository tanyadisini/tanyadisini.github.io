7. Quel âge a-t-il , Monsieur Charles Dupont ? Il a ... ans

( berapa umurnya, Charles Dupont? Dia berumur ... tahun )

A. cinquante ( 50 )

B. soixante ( 60 )

C. soixante deux ( 62 )

D. soixante trois ( 63 )

E. soixante –dix ✓ ( 70 )

8. Où est-ce que Charles Dupont s’habille ?

( di mana Charles Dupont berpakaian? )

A. devant la glace ✓ ( di depan cermin )

B. dans la salon ( di ruang tamu )

C. dans le chambre (di kamar tidur )

D. dans la salle de bain (di kamar mandi )

E. dans la salle à manger (di ruang makan )

Penjelasan:

Terjemahan-nya

Teks untuk pertanyaan nomor 7 - 8

　Charles Dupont berusia 70 tahun. Dia seorang penulis. Setiap hari, dia bangun jam 7 pagi, dia bangun, dia mandi dan berpakaian di depan cermin. Kemudian dia keluar, membeli koran dan pergi ke kafe untuk sarapan. Kemudian dia berjalan di sekitar taman kota sampai siang hari. Dia makan siang di Brasserie d'Alsace, lalu pulang dan menulis selama enam jam.

　Di malam hari, dia tinggal di rumah, menyiapkan sandwich di depan TV dan pergi tidur jam 11 pagi.
