25. There are 165 children and parents in the school. The number of children is twice the number of parents. How many parents are there in the school? Help Me​


Let’s use algebra to solve this problem. Let’s say the number of parents is “x”. Then the number of children is twice that, or 2x. The total number of children and parents is 165. So we can write an equation:

x + 2x = 165

Simplifying the left side:

3x = 165

Dividing both sides by 3:

x = 55

So there are 55 parents in the school.
