P dan R c. Q dan R b. P dan Q d. R dan S Tolong dibantu Kim Min Ju


P = {i,o,e,a}

Q = {5,7,11}

R = {1,3,9}

S = {2,3,4,5,6}

n(Q) = n(R)

n(R) ≠ n(S)

C. -111⁶⁵ ⟩ Angka satuannya selalu 1 {✓}

ket : ganjil = negatif dan genap = positif

A. -109⁹⁰ ⟩ 0 = genap = positif { ✓ }

B. -111⁷⁷ ⟩ 7 = ganjil = negatif { × }

C. -111⁶⁵ ⟩ 5 = ganjil = negatif { × }

{ Pilihan B dan C sudah bisa dibilang salah × }

____________________________

Kesimpulan :

-109⁹⁰ adalah bilangan ganjil positif
