10. Perbedaan secondary memory dengan main memory


Secondary dan main memory adalah dua jenis perangkat memori yang terdapat dalam sebuah sistem komputer dan berfungsi menyimpan data.

Pembahasan

Pada kesempatan ini, soal meminta kita untuk menyajikan perbedaan dari kedua memori tersebut. Berikut kakak akan mencoba menjawabnya.

MAIN MEMORI atau memori utama adalah perangkat penyimpanan data pada komputer dan umumnya berbentuk RAM.

Memori utama berfungsi sebagai media penyimpanan data sementara yang kemudian akan diolah oleh unit processor untuk kemudian disampaikan kepada pengunakan melalui media output.

Sementara itu, SECONDARY MEMORY atau memori tambahan adalah perangkat penyimmpanan data jangka panjang dan biasanya berbentuk ROM.

Dalam praktiknya, memori tambahan juga kerap difungsikan untuk menyediakan ruang penyimpanan data tambahan ketika tidak tersedia ruang cukup di memori utama.

23. Apa manfaat menganalisis data menggunakan in system in memory​


Jawaban:

Dengan menganalisis data menggunakan in system in memory, dapat memberikan manfaat sebagai berikut:

1. Kecepatan Analisis Data: In system in memory dapat menganalisis data secara real-time dan cepat. Hal ini membuat pengambilan keputusan menjadi lebih cepat dan tepat waktu.

2. Pengurangan biaya infrastruktur: In system in memory tidak memerlukan infrastruktur yang rumit seperti sistem penyimpanan atau jaringan, sehingga dapat mengurangi biaya infrastruktur.

3. Analisis Data di tempat: Dengan menggunakan in system in memory, analisis data dapat dilakukan di tempat tanpa harus mentransfer data ke server atau pusat data lainnya.

4. Pengukuran Subjektifitas Data: Dalam analisis data tradisional, ada tingkat subjektifitas dalam pengambilan keputusan. Namun, dengan menggunakan in system in memory, data dapat diukur lebih objektif.

5. Integrasi dengan Pengambilan Keputusan: In system in memory dapat diintegrasikan dengan sistem pendukung keputusan, sehingga keputusan dapat diambil dengan lebih tepat dan efisien.

Jawaban:

1. Memori adalah suatu alat atau medium yang mana informasi (data) atau instruksi (perintah) dapat disimpan dan dapat dikeluarkan kembali.

2. Fungsi memori untuk menyimpan informasi (data) sementara waktu atau untuk waktu yang lama, dimana informasi tersebut sewaktu-waktu dapat diambil kembali.

jenis jenis memori:

•DRAM (Dynamic Random Access Memory)

•SDRAM (Sychronous Dynamic Random Access Memory)

•RDRAM (Rambus Dynamic Random Access Memory)

•SRAM (Static Random Access Memory)

•EDORAM (Extended Data Out Random Access Memory)

•FPM DRAM (First Page Mode DRAM)

•Flash RAM.

manfaat memory:

Memori cache menyimpan semua data dan instruksi yang sering digunakan oleh CPU komputer.

Hal ini juga mengurangi waktu akses data dari memori utama.

Cache lebih cepat dari memori utama, dan kadang-kadang disebut juga memori CPU karena sangat dekat dengan chip CPU.
