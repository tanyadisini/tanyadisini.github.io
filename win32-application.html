7. 2. Jelaskan perbedaan antara web application dengan mobile application!


Penjelasan:

Perbedaan antara web application dan mobile application adalah sebagai berikut:

1. Platform: Web application dirancang untuk diakses melalui browser web di desktop atau laptop, sementara mobile application dirancang untuk diinstal dan diakses melalui smartphone atau tablet.

2. Aksesibilitas: Web application dapat diakses dari mana saja dengan koneksi internet, sedangkan mobile application memerlukan unduhan dan instalasi pada perangkat mobile yang dibutuhkan.

3. Fungsi: Web application lebih cenderung menggunakan fungsi-fungsi standar yang disediakan oleh browser seperti HTML, CSS dan JavaScript. Sedangkan, mobile application cenderung memanfaatkan fitur-fitur khusus pada perangkat mobile seperti kamera, GPS, atau sensor-sensor lainnya.

4. Pengalaman pengguna: Pengalaman pengguna pada web application dapat bervariasi tergantung pada jenis browser dan perangkat yang digunakan. Sedangkan, mobile application menawarkan pengalaman pengguna yang lebih konsisten dan berfungsi baik pada semua jenis perangkat mobile.

5. Perkembangan: Pengembangan web application lebih berfokus pada pengembangan back-end, sedangkan mobile application memiliki lebih banyak perhatian pada pengembangan front-end dan user interface.

22. protokol yang ada dalam application adalah?


Layer aplikasi atau application layer adalah sebuah layer abstraksi yang mengkhususkan pada protokol shared communication dan metode interface yang digunakan oleh host dalam sebuah jaringan komunikasi. Abstraksi pada layer aplikasi digunakan pada kedua model standar dalam jaringan komputer, TCP/IP atau Internet Protocol Suite dan model OSI.

Protokol yang ada dalam layer aplikasi adalah sebagai berikut

- TELNET.

Merupakan kependekan dari Telephone Network. Digunakan pada emulasi terminal yang memungkinkan klien Telnet untuk mengakses sumber dari server Telnet. Protokol ini digunakan untuk mengatur file dalam internet.  

Perintahnya adalah sebagai berikut

telnet [\\RemoteServer]

\\RemoteServer   : Nama server yang akan dikoneksikan

- FTP.

Merupakan kependekan dari File Transfer Protocol. Digunakan untuk melakukan transfer file. FTP bukan hanya merupakan sebuah protokol tapi juga merupakan sebuah program.  

Perintahnya adalah sebagai berikut

ftp machinename

- TFTP.  

Merupakan kependekan dari Trivial File Transfer Protocol yang merupakan versi dasar dari FTP.

Perintahnya adalah sebagai berikut

tftp [ options... ] [host [port]] [-c command]

- NFS.

Merupakan kependekan dari Network File System. Protokol ini memungkinkan host untuk berinteraksi dengan file sistem.

Perintahnya adalah sebagai berikut

service nfs start

- SMTP.

Merupakan kependekan dari Simple Mail Transfer Protocol. Protokol ini adalah bagian dari protokol TCP/IP yang memuat email sepanjang jaringan. Protokol ini bekerja sama dengan MTA (Mail Transfer Agent) untuk mengirimkan komunikasi ke komputer dan kotak masuk email yang benar.

Perintahnya adalah sebagai berikut

MAIL FROM:<mail@abc.com?

- LPD.

Merupakan kependekan dari Line Printer Daemon. Didesain untuk printer sharing. Daeman merujuk pada server atau agent

Perintahnya adalah sebagai berikut

lpd [ -d ] [ -l ] [ -D DebugOutputFile]

- X Window.

Merupakan protokol untuk menulis interface grafik aplikasi client/server.  

Perintahnya adalah sebagai berikut

Run xdm in runlevel 5

- SNMP.

Merupakan kependekan dari Simple Network Management Protocol. Digunakan untuk mengumpulkan data pada perangkat dalam jaringan dari stasiun manajemen pada interval acak atau tetap.  

Perintahnya adalah sebagai berikut

snmpget -mALL -v1 -cpublic snmp_agent_Ip_address sysName.0

- DNS.

Merupakan kependekan dari Domain Name System.  

Perintahnya adalah sebagai berikut

ipconfig /flushdns

- DHCP.

Merupakan kependekan dari Dynamic Host Configuration Protocol. Digunakan untuk memberi host pada alamat IP.  

Perintahnya adalah sebagai berikut

clear ip dhcp binding {address | * }

25. Apa Itu Application Letter?


Application letter sendiri disebut juga dengan surat lamaran kerja. Application letter biasanya merupakan syarat utama dalam melamar suatu pekerjaan. Dalam penulisan surat lamaran kerja, ada beberapa hal-hal yang harus diperhatikan karena menyangkut dengan yang disyaratkan oleh perusahaan, terkadang dalam pembuatan surat lamaran harus menggunakan bahasa inggris. Apabila dalam pembuatan surat lamaran tersebut dengan dirancang dengan baik dan benar serta memenuhi persyaratan yang diberikan oleh suatu perusahaan, maka biasanya application letter ini menjadi pertimbangan seseorang itu diterima atau dipanggil untuk mengikuti test selanjutnya.

Pembahasan

Tujuan dari application letter :

Memungkinkan untuk menjual kemampuan diri dengan menonjolkan keunggulan dari yang tertera di Curriculum Vitae atau CV. Memberikan kesempatan untuk menyertakan materi yang tidak termasuk dalam CV khususnya kualifikasi pribadi yang ada tuliskan ketika sedang mempersiapkan diri untuk melamar kerja. Memberikan gambaran bagaimana prospektifnya karir pelamar sebagai karyawan atau pegawai kedepannya melalui pertimbangan bahwa pelamar bisa menulis sebuah surat, walaupun agak kurang relevan di zaman elektronik seperti sekarang ini, namun bagi banyak karyawan dari segelintir perusahaan atau lembaga, dipertimbangkan sebagai salah satu bentuk keahlian mendasar yang mesti dipunyai dan sebagai uji coba kemampuan untuk berkomunikasi secara jelas dan efektif. 27. Yang termasuk application software yaitu … .


Yang termasuk application software yakni :

Microsoft Office atau software pengolah kata .  Adobe Photoshop .  Adobe Dreamweaver .  Adobe Acrobat .  Google Chrome .  WinRAR .  

Application Software dapat disebut sebagai jenis software yang dirancang untuk menyelesaikan tugas-tugas spesifik dalam operating system. Beberapa dari tugas ini adalah termasuk menangani dokumen, menangani perhitungan, mengedit gambar, mengedit video, dan yang lainnya.  

Pembahasan

Pemakaian dari Application software yakni melakukan tugas tertentu tergantung pada bagaimana ia dirancang, jadi tidak dapat melakukan tugas lain yang berada di luar cakupannya.

Application software diinstal hanya pada saat pengguna membutuhkannya. Contohnya, seseorang hanya akan menginstal software edit foto ketika ingin mengedit beberapa foto.

Waktu eksekusi Application software hanya akan berjalan jika diminta, Jadi seseorang akan memulai pengolah kata ketika perlu membuka atau menyiapkan beberapa dokumen.

Untuk application software, pengguna akan mempunyai interaksi langsung dengan software. Bahkan, pengguna dapat mengontrol cara software beroperasi.

Application software ini tidak dapat berjalan secara independen, Jadi mereka tergantung pada kondisi yang disediakan oleh operating system.

Pemrograman application software sendiri tidak rumit. Seorang programmer hanya perlu mempunyai pengetahuan tentang system software yang mendasarinya serta bahasa tingkat tinggi.

39. Contoh application letter​


Penjelasan:

Job application letter

atau surat lamaran pekerjaan merupakan surat yang dituliskan ketika kita ingin melamar pekerjaan untuk posisi/jabatan tertentu dalam suatu lowongan pekerjaan. Surat lamaran pekerjaan dalam bahasa Inggris terdiri atas 3 bagian besar, yaitu:

Salutation (pembukaan) adalah bagian pembukaan kepada siapa surat tersebut ditujukan (Umumnya pihak human resource department/HRD, kepala manajer, dll). Selalu gunakan format: Dear sir/madam (Jika nama serta jenis kelamin tidak diketahui) Dear Mr./Mrs./Ms. Last name (jika nama serta jenis kelamin diketahui)

Body of paragraph (isi)

Body atau isi adalah bagian inti dari cover letter. Pada bagian ini biasanya dituliskan latar belakang singkat tentang pendidikan dan pengalaman kerja yang menonjol, darimana informasi lowongan kerja diperoleh. Dan bisa juga mencantumkan kelengkapan data yang disertakan dalam amplop/e-mail, seperti curriculum vitae (CV) dan portofolio.

Jawaban:

perbedaannya ada pada penggunaannya.

Penjelasan:

yaitu web application digunakan bergantung kepada interaksi pengguna, apakah itu kontribusi content (Youtube, Twitter, Facebook) atau dengan mengumpulkan data dari sumber lain untuk ditampilkan kepada pengguna (Mint, Google Analytics, Klout) atau gabungan dari keduanya.

sedangkan form application digunakan untuk membuat aplikasi di mana user interface utama adalah browser. tentu, ini termasuk aplikasi yang dimaksudkan akan tersedia secara publik melalui world wide web, seperti e-commerce aplikasi. web form memiliki kegunaan lebih dari sekedar membuat situs web, tetapi aplikasi lain yang meminjamkan diri ke (thin front end) juga, seperti buku pedoman karyawan berbasis intranet atau aplikasi manfaat.
