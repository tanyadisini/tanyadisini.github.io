12. Jelaskan perbedaan dari media penyimpanan luar hard disk floppy disk dan flash disk.


Jelaskan perbedaan dari media penyimpanan luar hard disk, floppy disk, dan flash disk Jawaban

Pendahuluan

Perbedaan yang paling menonjol dari perbandingan media penyimpanan luar seperti hardisk, floppy disk dan flashdisk adalah kapasitas media penyimpanan itu sendiri. Semakin besar kapasitas suatu media penyimpanan, maka semakin banyak pula file yang bisa dipindahkan.

Pembahasan

Perbedaan media penyimpanan luar diatas adalah kapasitasnya. Untuk hardisk bisa dicek memiliki kapasitas penyimpanan paling besar diantara hardisk, floppy disk dan flashdisk. Sedangkan floppy disk merupakan media penyimpanan yang paling di kecil. Seiring dengan perkembangan jaman, media penyimpanan floopy disk sudah tidak berlaku kembali atau sudah sangat jarang digunakan. Contoh floppy disk adalah disket.

Kesimpulan

Demikian pembahasan mengenai media penyimpanan, dengan urutan pertama diberikan kepada hardisk sebagai dan urutan terakhir diberikan kepada floopy disk. Untuk mempelajari lebih lanjut atau mempelajari materi mengenai jaringan komputer atau ingin mempelajari kartu ATM dapat dibaca pada link berikut

13. direktory untuk floppy disk


Jawaban:

direktory untuk floppy disk yaitu A:

Penjelasan:

Pada komputer dengan sistem operasi Windows, directory floppy disk yaitu A: dan B:

Selanjutnya directory untuk harddisk yaitu C: dan seterusnya (sesuai jumlah partisi harddisk).

Pengaturan Drive A dan Drive B pada floppy disk dapat dijumpai dalam beberapa versi BIOS, seperti Phoenix BIOS. Perhatikan gambar berikut ini (ditunjukkan tanda panah kuning).

31. Apa perbedaan dari Floppy Disk dan USB disk dari segi kapasitas penyimpanan data?


Jawaban:

Perbedaan Floppy Disk dan USB Disk dalam segi kapasitas penyimpanan data adalah Floppy Disk memiliki kapasitas terbatas hanya sekitar 1,44 MB sedangkan USB Disk memiliki kapasitas sangat besar dan bisa mencapai 2TB.

Penjelasan:

Floppy Disk (Diket) adalah media penyimpanan data perangkat keras (hardware) yang banyak digunakan pada tahun 1990an sampai awal tahun 2000an pada jenis komputer lama. “Floppy” memiliki arti fleksibel. Sebab, memory disk tersebut memang sangat fleksibel walaupun saat itu fungsinya masih sangat terbatas.

USB Disk adalah media penyimpanan data perangkat keras (hardware) yang dihubungkan port USB yang mampu menyimpan berbagai format data dan memiliki kapasitas penyimpanan yang cukup besar.  Flashdisk dapat menyimpan data secara permanen walaupun aliran listrik pada rangkaian flashdisk diputuskan.
