5. Untuk menyimpan dalam GitHub Anda harus a. menghubungkan dengan akun GitHub b. menggunakan akun lain C. menambahkan akun lagi d. menggunakan save as e. menyimpan kode​


Jawaban:

Untuk menyimpan dalam GitHub, kamu perlu melakukan langkah-langkah berikut:

a. Menghubungkan dengan akun GitHub: Pertama, kamu perlu membuat akun GitHub jika belum punya. Setelah itu, kamu harus menghubungkan akun GitHub tersebut dengan akun Git di komputermu. Hal ini bisa dilakukan dengan mengatur konfigurasi pada Git, seperti menambahkan alamat email dan nama pengguna GitHub ke dalam Git.

Setelah terhubung, kamu bisa membuat repositori baru di GitHub atau menggunakan repositori yang sudah ada.

b. Menambahkan akun lagi: Tidak perlu menambahkan akun lain untuk menyimpan di GitHub. Yang perlu kamu lakukan adalah menggunakan akun GitHub yang sudah terhubung dengan Git di komputermu.

c. Menggunakan save as: Save as tidak terkait dengan menyimpan kode ke GitHub. Save as digunakan untuk menyimpan file dengan nama baru atau di lokasi yang berbeda di komputermu, bukan untuk menyimpan ke repositori GitHub.

e. Menyimpan kode: Setelah terhubung dengan akun GitHub, kamu bisa menggunakan perintah-perintah Git seperti "git add", "git commit", dan "git push" untuk menyimpan kode ke repositori GitHub. Dengan melakukan commit dan push, perubahan kode yang kamu lakukan akan tersimpan di repositori GitHub.

Jadi, jawabannya adalah A. menghubungkan dengan akun GitHub.

Makey Makey dapat digunakan sebagai alat edukasi dan kreativitas untuk mengajarkan konsep-konsep dasar elektronika dan pemrograman pada anak-anak maupun orang dewasa.

21. istilah pada github yang bisa digunakan untuk mengambil repostory dan remote adalah​


Jawaban:

JANGAN LUPA BINTANG 5 NYA YA KAK ^^

Penjelasan:

Istilah pada GitHub yang bisa digunakan untuk mengambil repository dan remote adalah:

1. Clone

Clone digunakan untuk mengambil (copy) repository Git menjadi local repository di komputer Anda. Caranya dengan mengetikkan perintah berikut:

`git clone url`

Dimana `url` adalah alamat URL lengkap repository Git yang ingin Anda clone.

Contoh:

`git clone https://github.com/namlnt/newrepo.git`

Git akan membuat folder `newrepo` lokal dan mengambil semua file dan versi yang ada di repository tersebut.

2. Fork

Fork digunakan untuk membuat salinan (copy) repository milik orang lain ke akun GitHub Anda. Caranya dengan menekan tombol "Fork" pada pojok kanan atas repository di GitHub.

Ini berguna jika Anda ingin mengontribusikan perubahan pada repository milik orang lain.

3. Remote

Git menggunakan istilah remote untuk merujuk ke repository Git yang disinkronkan dengan local repository Anda. Remote bisa berupa repository GitHub atau server Git lainnya.

Remote bisa ditambahkan menggunakan perintah `git remote add` dan disinkronkan dengan perintah `git push` dan `git pull`.

Semoga membantu! Jika ada istilah GitHub lainnya yang ingin Anda ketahui, silakan tanyakan. Saya akan senang membantu selengkap mungkin.

10. a. Readme Create

Jawaban yang benar adalah a. Readme Create.

Setiap repository umumnya memiliki file README yang berfungsi untuk memberikan penjelasan singkat, petunjuk penggunaan, dan informasi lainnya tentang proyek atau kode yang ada dalam repository tersebut. File README biasanya ditulis dalam format teks atau Markdown, dan diletakkan di direktori utama repository agar mudah diakses oleh pengguna atau kontributor.

11. Jawaban yang benar adalah a. Commit a new file.

Setelah selesai membuat file baru dalam repository, langkah selanjutnya adalah menyimpan perubahan tersebut dengan melakukan commit. Saat melakukan commit, Anda memberikan deskripsi singkat tentang perubahan yang dilakukan pada file tersebut. Setelah melakukan commit, perubahan akan dicatat dalam sejarah repository.

12. Guna melanjutkan proses instalasi Update Driver Software - USB IO Board, dilakukan dengan klik tombol d. Next.

Pilihan d. "Next" biasanya digunakan untuk melanjutkan proses instalasi atau konfigurasi setelah Anda menyelesaikan langkah sebelumnya. Dengan mengklik tombol "Next", Anda akan melanjutkan ke langkah berikutnya dalam proses instalasi Update Driver Software - USB IO Board.

13. Program aplikasi yang digunakan untuk melakukan instalasi kode pada sirkuit Makey Makey adalah Scratch.

Pilihan yang tepat adalah c. Scratch. Scratch adalah sebuah platform pemrograman visual yang digunakan untuk membuat animasi, permainan, dan interaksi dengan perangkat keras seperti Makey Makey. Dengan menggunakan Scratch, pengguna dapat membuat kode dengan mengatur blok-blok program secara grafis, tanpa perlu menulis kode secara langsung.

14. Istilah pada GitHub yang biasa digunakan untuk mengambil repository dari remote adalah d. Clone.

Ketika Anda ingin mendapatkan salinan lengkap dari repository yang ada di remote server GitHub ke komputer lokal Anda, Anda dapat menggunakan perintah "git clone" untuk melakukan proses cloning. Ini akan mengunduh semua file, sejarah commit, dan cabang (branch) yang ada dalam repository dan membuat salinan lokal dari repository tersebut di komputer Anda.

Pilihan yang benar adalah d. Clone.
