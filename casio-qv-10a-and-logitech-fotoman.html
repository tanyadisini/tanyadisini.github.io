18. -10a+(-10b+10aaku butu hasil di atas​


Hasil dari -10a + (-10b + 10a) adalah -10b.

Pembahasan

Penjumlahan dan pengurangan bilangan bulat dapat menggunakan sifat-sifat berikut :

Positif (+) ditambah positif (+) => kedua bilangan dijumlah, hasilnya positif. Contoh :

20 + 15 = 35

negatif (-) ditambah negatif (-) => kedua bilangan dijumlah, hasilnya negatif. Contoh :

-10 + -20 = - (10 + 20) = -30

Positif (+) ditambah negatif (-) dan sebaliknya => bilangan terbesar dikurang bilangan terkecil, hasilnya mengikuti bilangan terbesar. Contoh :

30 + -7 = +(30 - 7) = 23

22 - 50 = -(50-22) = -28

Positif (+) dikurang positif (+) => caranya sama seperti positif ditambah negatif.Negatif (-) dikurang negatif (-) => negatif bilangan pertama ditambah positif bilangan kedua. Karena jika ada dua simbol (-) berubah menjadi (+). Contoh :

-33 - -11 = -33 + 11 = -(33 - 11) = -22

Positif (+) dikurang negatif (-) => jumlah kedua bilangan, hasilnya positif. Karena ada dua simbol (-) berubah menjadi (+). Contoh :

30 - - 10 = 30 + 10 = 40

Notes :

Penjumlahan dan pengurangan bilangan bulat hanya bisa diselesaikan jika variabelnya sama. Misalnya a dengan a, b dengan b, dan seterusnya.

Penyelesaian soal :

-10a + (-10b + 10a)

= -10a -10b + 10a

= -10a + 10a -10b

= 0 - 10b

= -10b

PEMBAHASAN

Bentuk aljabar merupakan suatu bilangan yang memiliki besaran yang dinyatakan melalui penggunaan lambang-lambang angka dan huruf serta operasi hitung.

Koefisien merupakan suatu bilangan yang terdapat pada suku aljabar dimana dalam penulisannya diletakkan paling depan.

Variabel merupakan suatu simbol/lambang yang dipergunakan buat mewakili suatu bilangan secara umum.

Konstanta merupakan suatu bilangan yang tetap, atau merupakan suatu suku yang terdapat pada bentuk aljabar yang tidak memuat variabel.

Penguadratan Suku Dua

[tex]{\boxed{ \blue{(a + b)^{2} = a^{2} + 2ab + b^{2}}}}[/tex]

[tex]{\boxed { \blue{(a - b)^{2} = a^{2} - 2ab + b^{2}}}}[/tex]

Pemfaktoran Selisih Dua Kuadrat

[tex]{\boxed{ \blue{x^{2} - y^{2} = (x + y) (x - y)}}}[/tex]

Pemfaktoran Bentuk a² + bx + c

[tex]{\boxed{\blue{a^{2} + bx + c = (x + p) (x + q)}}}[/tex]
