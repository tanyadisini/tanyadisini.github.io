13. Pengertian dari Pangkat pecahan itu apa sih? kasih contoh soal dan pembahasannya juga ya... maaf kalo ngerepotin


Bilangan pangkat merupakan suatu bilangan yang didapatkan dari operasi perkalian terhadap suatu bilangan beberapa kali. Sedangkan bilangan pecahan merupakan suatu bilangan yang ditampilkan dalam bentuk:

 x/y

dimana:

x disebut pembilang dany disebut penyebut,x dan y merupakan bilangan bulat dany ≠ 0.

Perhitungan perpangkatan dalam pecahan memiliki prinsip yang sama dengan aturan perpangkatan pada bilangan bulat. Perpangkatan xn sama dengan perkalian bilangan x dengan bilangan x sebanyak n kali. Bagaimana dengan perpangkatan 91/2 ?

Sebagai contoh pada bilangan:

9n = 3

Pernyataan tersebut memiliki arti bahwa bilangan 9 jika dipangkatkan dengan n hasilnya sama dengan 3.

Lalu bagaimana caranya mencari nilai n?

Nilai n bisa dicari dengan cara berikut :

9n = 3(32)n = 3132n = 31 

Berarti nilai:

2n = 1

maka :

n = ½

sehingga:

91/2 = 3

Karena :

√9 = 3

maka :

 √9 = 9½ = 3

Dari uraian tersebut maka bilangan pangkat pecahan bisa didefinisikan dengan rumus berikut :

Bilangan a jika dipangkatkan dengan bilangan pecahan m/n sama dengan akar pangkat n dari bilangan apangkat m.

Dimana a ≥ 0 dan m, n merupakan bilangan bulat positif.

Sifat-sifat Operasi Bilangan Pangkat Pecahan

Pada operasi bilangan pangkat pecahan, berlaku juga sifat-sifat perpangkatan bilangan bulat seperti berikut ini :

ap x aq = ap+q

Apabila bilangan a pangkat p dikalikan dengan bilangan a pangkat q, maka hasilnya sama dengan bilangan adipangkatkan bilangan hasil dari p ditambah q.

ap : aq = ap-q, dimana a ≠ 0 dan p > q

Bilangan a pangkat p dibagi dengan bilangan a pangkat q, maka hasilnya sama dengan bilangan adipangkatkan bilangan hasil dari p dikurangi q.

Dimana nilai a tidak boleh nol dan p harus lebih besar dari q.

(ap)q = apq

Jika bilangan a pangkat p dipangkatkan dengan bilangan q, maka hasilnya sama dengan bilangan adipangkatkan bilangan p dikali q.

(ab)p = ap.bp

Jika bilangan a dikalikan b kemudian dipangkatkan dengan bilangan p, maka hasilnya sama dengan bilangan a pangkat p dikalikan bilangan b pangkat p.

 (a/b)p = ap/bp dimana nilai b ≠ 0

Bilangan pecahan a/b dipangkatkan dengan bilangan p, maka hasilnya sama dengan bilangan a pangkat pdibagi bilangan b pangkat p, dimana nilai b tidak boleh nol.

Cara Menyelesaikan Soal Operasi Bilangan Pangkat Pecahan

Untuk menyelesaikan soal bilangan berpangkat pecahan, bisa dilakukan dengan dua cara yakni :

Mengubah bilangan pokok menjadi bilangan berpangkat sama dengan bilangan penyebut pada pangkat pecahan.

Contoh :

42/3 = (22)3/2 = 22×3/2= 23 = 8

Pada contoh di atas, bilangan 4 diubah menjadi bilangan 2 pangkat 2, sehingga ketika pangkat 2 dikalikan dengan pangkat 3/2, langusng menghasilkan bilangan pangkat 3.

Dengan mengubah bilangan yang dipangkatkan menjadi bilangan berpangkat sama dengan bilangan penyebut pada pangkat pecahannya menghasilkan bentuk pangkat bukan pecahan, sehingga lebih mudah untuk dipangkatkan.

Mengubah bilangan pangkat pecahan menjadi akar

Bilangan pangkat pecahan bisa diubah menjadi akar dengan menggunakan rumus berikut :

am/n = a1/n x m = (a1/n)m

Contoh :

272/3 = 271/3×2 = (271/3)2 = (  )2  = 32 = 9

Pada contoh di atas, bilangan pangkat pecahan 1/n sama dengan akar pangkat n, sehingga pangkat m/ndipisahkan terlebih dahulu dengan perkalian 1/n x m.
