14. mr jones memperkirakan bahwa ada kemungkinan 30% bahwa perusahaannya akan mendirikan kantor Prancis di pheonix. jika ya, ia yakin 60% bahwa ia akan menjadi manajer operasi ini. berapa kemungkinannya bahwa mr jones akan menjadi manajer kantor cabang [phix]​


Jawab:

kemungkinannya bahwa mr jones akan menjadi manajer kantor cabang =

P(A n B) = 18%

Penjelasan dengan langkah-langkah:

P(A) = Peluang Kantor di Pheonix = 30% = 0,3

P(B) = Peluang menjadi manajer = 60% = 0,6

P(A n B) = P(A) x P(B) = 0,3 x 0,6 = 0,18 = 18%

Simple Present Tense:

(+) Subject + Verb-1 S/ES + Object

(-) Subject + Do/Does + Verb-1 + Object

(?) Do/Does + Subject + Verb-1 + Object + ?

Present Continuous Tense:

(+) Subject + Tobe (is/am/are) + Verb-ing + Object

(-) Subject + Tobe (is/am/are) + not + Verb-ing + Object

(?) Tobe (is/am/are) + Subject + Verb-ing + Object + ?
