3. kalau smoothing rambut, bilas nya dihitung 3 hari dr hari smoothing atau 3 hari setelah smoothing?


Setelah smoothing rambut, pembilasan atau keramas dapat dilakukan setelah 3 hari kemudian. Misalnya, Anda smoothing hari Senin, maka 3 hari kemudian Anda dapat membilas rambut yang telah diluruskan tersebut. Hal ini dilakukan agar krim dapat meresap sempurna dan bekerja maksimal pada rambut.

Pembahasan

Smoothing adalah proses perawatan rambut agar menjadi lembut, halus, lurus, dan mudah diatur. Perawatan rambut ini mengalami beberapa tahapan yang memerlukan waktu total sekitar 3 hingga 5 jam.

Berikut penjelasan singkat tahapan perawatan smoothing pada rambut.

1. Menganalisis rambut

Analisis jenis dan kondisi rambut diperlukan untuk menentukan penggunaan produk yang sesuai.

2. Menguji elastisitas rambut

Cara ini dilakukan dengan menarik sehelai rambut secara perlahan. Jika rambut tidak putus/kembali seperti semula setelah ditarik, maka elastisitas rambut baik. Jika rambut putus/patah dengan tarikan perlahan, maka elastisitas rambut buruk. Semakin buruk elastisitas rambut, risiko kerusakan (rambut patah) pada proses smoothing semakin besar.

3. Mencuci rambut

Sebagaimana hampir semua perawatan rambut, mencuci rambut atau keramas adalah hal penting yang perlu dilakukan sebelum melakukan perawatan. Tujuannya adalah untuk menghilangkan kotoran atau debu yang masih menempel di rambut. Dengan demikian, krim yang akan digunakan dapat bekerja secara maksimal.

4. Mengaplikasikan krim pelurus rambut

Krim pelurus rambut diaplikasikan secukupnya ke rambut secara merata. Aplikasi rambut dilakukan dari jarak 1 cm pangkal hingga ujung rambut. Upayakan pengaplikasian selesai dalam waktu 20 menit. Diamkan rambut 10 hingga 15 menit.

5. Menguji elastisitas rambut

Pada menit ke-10, uji elastisitas rambut untuk kesiapan pada tahap berikutnya. Jika dirasa belum siap, maka diamkan kembali selama 5 menit.

6. Membilas dan mengeringkan rambut

Rambut dibersihkan dengan air hangat hingga dingin tanpa shampoo dan conditioner, kemudian dikeringkan hingga 80% kering.

7. Mencatok rambut

Setelah rambut kering, semprotkan cairan pelindung rambut untuk melindungi rambut dari suhu panas. Lakukan pencatokan rambut sehingga seluruhnya menjadi lurus dan matang.

8. Mengaplikasikan krim penetral pada rambut

Aplikasikan krim penetral secukupnya pada rambut secara merata dan diamkan selama minimal 20 menit. Tujuannya adalah menghilangkan bahan kimia sisa dari pembilasan pertama.

9. Membilas rambut

Rambut dibilas sampai bersih dan gunakan conditioner untuk melindungi dan mengganti nutrisi rambut yang hilang saat proses pelurusan. Bilas kembali hingga bersih.

10. Mencatok ulang rambut

Pencatokan terakhir dilakukan dengan suhu sedang. Jika dirasa kondisi awal rambut sangat kering, maka pemberian vitamin akan sangat membantu hasil akhir yang maksimal.

Setelah proses akhir smoothing, rambut sebaiknya dibilas atau keramas seperti biasa minimal 3 hari setelah smoothing. Tujuannya adalah agar krim pelurus dapat bekerja secara maksimal pada rambut. Sangat direkomendasikan merawat rambut dengan masker pada pembilasan pertama setelah smoothing.

// AUTHOR : LordRimuruTempest

class node{

String namaSiswa;

node pointer;

public node(String namaSiswa){

this.namaSiswa = namaSiswa;

}

}

public class singleLinkedList {

node head, tail; int size = 0;

boolean isEmpty(){

return(size == 0);

}

void tambahData(node newData){

if(isEmpty()) head = tail = newData;

else{

tail.pointer = newData;

tail = newData;

}size++;

}

void hapusData(String nama){

node temp = head;

if(!isEmpty()){

if(head == tail && temp.namaSiswa.equals(nama)) head = tail = null;

else{

if(nama.equals(head.namaSiswa)){

head = temp.pointer;

temp = null;

}else{

while(!temp.pointer.namaSiswa.equals(nama)){

temp = temp.pointer;

}

if(temp.pointer.pointer != null) temp.pointer = temp.pointer.pointer;

else temp.pointer = null;

}

}size--;

}else System.out.println("List Kosong !!");

}

void display(){

node temp = head; int nomor = 1;

System.out.println("DATA SISWA : ");

if(size != 0){

do{

System.out.println(nomor++ + ". " + temp.namaSiswa);

temp = temp.pointer;

}while(temp != null);

}else System.out.println("List Kosong !!");

}

}

Pembahasan

Single linked list adalah salah satu jenis dari struktur data berupa deretan objek yang disebut objek node (mirip seperti kereta api yang terdiri atas beberapa gerbong, node dianggap sebagai satu gerbong dan linked list adalah keretanya). Program akan dibuat dengan bahasa java. Pada dasarnya, single linked list terdiri atas 5 unsur dasar sebagai berikut :

1. Objek node, untuk menyimpan data.

2. Node kepala(node head), yaitu node pada urutan pertama di dalam list.

3. Node ekor(node tail), yaitu node pada urutan terakhir dalam list.

4. Node link/pointer, untuk menandai objek node selanjutnya dari posisi node sekarang.

5. Method untuk menambah node, menghapus node, melihat ukuran list, dan sebagainya.

Setelah memahami strukturnya, maka gunakan struktur data ini untuk membuat program record data nilai siswa.

Cara kerja program :

1. Class node akan membentuk objek node dengan data berupa nama siswa.

2. Class linked list akan membentuk objek list yang terdiri atas beberapa node dan memiliki atribut head, tail, dan size serta memiliki beberapa method sebagai berikut :

Method isEmpty akan mengecek ukuran list. Jika size = 0 maka akan menghasilkan nilai true dan jika tidak maka bernilai false.Method tambahData berfungsi untuk menambah node pada bagian belakang list. Jika list masih kosong, maka node akan menjadi kepala(head) sekaligus ekor(tail) dan nilai size ditambah 1. Jika tidak, maka pointer tail akan menunjuk node tersebut, lalu node tersebut akan menjadi tail/ekor.Method hapusData akan menghapus node dengan nama siswa tertentu. Jika list kosong, maka akan dicetak "list kosong". Jika tidak, maka program akan mengecek apakah head = tail (hanya 1 data) dan apakah nilai temp.namaSiswa = nama. Jika benar maka list akan dikosongkan sehingga head = tail = null(kosong). Jika salah, maka program akan mengecek apakah nama = head.namaSiswa, jika benar maka posisi head akan digantikan oleh node berikutnya. Dan jika salah maka program akan melakukan looping sampai temp.pointer.namaSiswa = nama. Setelah berhenti maka temp.pointer yang memiliki data nama siswa yang dimaksud akan dihapus, dan pointer-nya akan diganti dengan node berikutnya. Jika node selanjutnya dari temp.pointer tidak ada, maka akan dikosongkan(null).

3. Class menu akan membentuk menu untuk mengoperasikan program record nama siswa. Program akan melakukan looping secara terus menerus untuk melakukan record data sampai user memilih keluar.

Kode program, pengoperasian dan output-nya dapat dilihat pada lampiran gambar.

Santan merupakan bahan alami yang paling 
familiar dalam urusan problema rambut.

paling populer yang difungsikan dalam 
meningkatkan pertumbuhan rambut.

Keberadaan kulit apel memiliki beberapa vitamin 
dasar yang dapat meningkatkan sirkulasi darah 
ke kulit kepala. Ketika sirkulasi berjalan secara 
lancar, maka akan berpengaruh baik pada 
pertumbuhan dalam memanjangkan rambut.

Semenjak dahulu kala, minyak zaitun telah 
dipercaya sebagai bahan alami untuk merawat 
rambut. Dulu nenek moyang kita selalu 
menggunakannya tidak cuma bermanfaat untuk 
menghaluskan rambut, akan tetapi juga bisa 
meningkatkan dalam proses pertumbuhan 
rambut secara cepat. 
Lalu bagaimana cara menggunakan minyak 
zaitun ini untuk memanjangkan rambut ? 
teteskan 4 hingga 5 tetes minyak zaitun setelah 
keramas, lalu pijatlah kulit kepala. Selanjutnya 
diamkan semalaman supaya kandungan minyak 
zaitun terserap oleh kulit kepala. Ketika pagi hari, 
jangan lupa untuk mencuci rambut hingga bersih 
betul.

Salah satu metode terbaik untuk memanjangkan 
rambut dapat Anda lakukan dengan 
memanfaatkan putih telur. langkahnya sediakan 
kurang lebih 3 butir telur, lalu ambil bagian 
putihnya saja, kemudian kocok hingga rata. 
Selanjutnya kita usapkan ke seluruh bagian 
rambut sambil dipijat, kemudian diamkan selama 
5-10 menit. Nutrisi di dalam telur akan meresap 
ke dalam rambut sehingga membuatnya lebih 
kuat juga sehat. Metode diatas mampu untuk 
membantu mempercepat proses perpanjangan 
rambut Anda.

Meminum air putih 8 gelas sehari tak hanya 
membuat tubuh lebih bugar, tapi juga 
mengaktifkan sel-sel dalam rambut yang 
mempercepat pertumbuhannya. Semakin kuat 
jaringan dalam rambut, kesuburannya juga 
semakin terjaga. 

6. Konsumsi Protein 
Protein bermanfaat guna memenuhi kebutuhan 
nutrisi penting bagi pertumbuhan rambut. 
Supaya rambut cepat tumbuh, sebaiknya penuhi 
kebutuhan protein Anda dengan bayak 
mengonsumsi daging merah, ayam, ikan, kacang- 
kacangan, produk olahan susu, dan juga telur. 

7. Vitamin B 
Agar tetap sehat dan subur, rambut Anda juga 
membutuhkan vitamin B. Tanpa asupan vitamin

pecah-pecah. Konsumsi multivitamin yang 
mengandung vitamin B, yang juga terdapat 
dalam pisang dan merica.

Beberapa salon kecantikan dan spa kini 
menyediakan perawatan massage kulit kepala 
dengan minyak khusus. Rasa hangat yang 
meresap ke dalam kulit kepala merangsang 
pertumbuhan dan kesehatan rambut. Rambut 
yang terjaga kesehatannya akan lebih mudah 
tumbuh. 

9. Kurangi Blow-Dry

17. sebutkan contoh dari kegiatan pemasaran single level marketting


sebutkan contoh dari kegiatan pemasaran single level marketing ?JawabanPendahuluan

Pemasaran adalah suatu sistem total dari kegiatan bisnis yang dirancang untuk merencanakan, menentukan harga, promosi dan mendistribusikan barang- barang yang dapat memuaskan keinginan dan mencapai pasar sasaran serta tujuan perusahaan. konsep inti pemasaran meliputi: kebutuhan, keinginan, permintaan, produksi, utilitas, nilai dan kepuasan; pertukaran, transaksi dan hubungan pasar, pemasaran dan pasar.

Pembahasan

Sistem pemasaran adalah kumpulan lembaga-lembaga yang melakukan tugas pemasaran barang, jasa, ide, orang, dan faktor-faktor lingkungan yang saling memberikan pengaruh dan membentuk serta mempengaruhi hubungan perusahaan dengan pasarnya. kegiatan pemasaran terdiri dari 2 jenis yaitu pemasaran langsung atau direct selling atau dikenal juga dengan istilah single level marketing dan pemasaran tidak langsung atau multi level marketing.

kegiatan pemasaran pada single level marketing yaitu kegiatan pemasaran yang dilakukan langsung oleh produsennya sendiri, jadi produsen bertindak sebagai distributor langsung untuk manawarkan produk-produknya. contoh dari single level marketing adalah penjual jajanan di pinggir jalan, warung-warung makan atau restoran cepat saji,

1. Forecasting adalah proses perkiraan atau prediksi mengenai suatu peristiwa atau kejadian di masa depan berdasarkan data historis dan faktor-faktor lain yang relevan. Tujuan dari forecasting adalah untuk memberikan perkiraan yang akurat mengenai apa yang mungkin terjadi di masa depan sehingga dapat digunakan untuk pengambilan keputusan dan perencanaan.

Lingkungan yang mempengaruhi proses forecasting meliputi:

a. Data Historis: Kualitas dan kuantitas data historis yang tersedia akan mempengaruhi akurasi forecasting. Data yang lengkap, akurat, dan relevan akan memberikan dasar yang lebih kuat untuk melakukan prediksi.

b. Perubahan Lingkungan: Perubahan lingkungan eksternal seperti tren pasar, kebijakan pemerintah, perubahan demografis, dan faktor-faktor lainnya dapat mempengaruhi pola permintaan dan perilaku konsumen. Oleh karena itu, perubahan lingkungan perlu diperhitungkan dalam proses forecasting.

c. Ketidakpastian: Beberapa faktor yang mempengaruhi forecasting mungkin tidak dapat diprediksi dengan pasti. Ketidakpastian seperti bencana alam, perubahan kebijakan, atau perubahan keadaan pasar dapat menyulitkan proses forecasting dan mempengaruhi akurasi perkiraan.

d. Teknologi dan Metode: Kemajuan teknologi dan perkembangan metode forecasting dapat mempengaruhi tingkat akurasi dan efisiensi dalam melakukan perkiraan. Penggunaan teknologi yang tepat dan penerapan metode forecasting yang sesuai akan meningkatkan hasil forecasting.

Perbedaan Singly Linked List, Double Linked List, Dan Circular Linked List

1. Singly Linked List merupakan suatu linked list yang hanya memiliki satu variabel pointer saja. Dimana pointer tersebut menunjuk ke node selanjutnya, biasanya field pada tail menunjuk ke NULL.

2. Doubly Linked List merupakan suatu linked list yang memiliki dua variabel pointer yaitu pointer yang menunjuk ke node selanjutnya dan pointer yang menunjuk ke node sebelumnya. Setiap head dan tailnya juga menunjuk ke NULL.

3. Circular Linked List merupakan suatu linked list dimana tail (node terakhir) menunjuk ke head (node pertama). Jadi tidak ada pointer yang menunjuk
