23. Diketahui hasil dari 23 43 63 83 ... (2n)3 = 2n2 (n 1)2. untuk membuktikan kebenaran rumus tersebut, untuk n=k 1 harus dibuktikan bahwa 23 43 63 83 ... (2k 2)3 sama dengan ...


Terdapat suatu pernyataan:

P(n): 23+43+63+83+...+(20n+3) = n(10n+13)

Untuk n = k+1, harus dibuktikan bahwa 23+43+63+83+...+(20(k+1)+3) sama dengan (k+1)(10(k+1)+13). Dengan induksi matematika, pernyataan di atas terbukti. Silakan simak penjelasan di bawah.

Penjelasan dengan langkah-langkah

Diketahui: P(n): 23+43+63+83+...+(20n+3) = n(10n+13)

Ditanya:

Untuk n = k+1, harus dibuktikan 23+43+63+83+...+(20(k+1)+3) = ?

Buktikan P(n)

Jawab:

Langkah awal

Untuk n = 1, maka:

Ruas kiri: 20·1+3 = 20+3 = 23

Ruas kanan: 1(10·1+13) = 10+13 = 23

Dengan demikian, 20·1+3 = 1(10·1+13). Akibatnya, P(n) terbukti untuk n = 1.

Langkah induksi

Asumsikan benar untuk n = k, yaitu:

23+43+63+83+...+(20k+3) = k(10k+13)

Pada langkah ini, harus dibuktikan untuk n = k+1, yaitu:

23+43+63+83+...+(20k+3)+(20(k+1)+3) = (k+1)(10(k+1)+13)

Akan dibuktikan dari ruas kiri.

23+43+63+83+...+(20k+3)+(20(k+1)+3)

= k(10k+13)+(20(k+1)+3)

= 10k²+13k+20k+20+3

= 10k²+20k+13k+23

= 10k²+20k+10-10+13k+23

= 10(k²+2k+1)+13k+23-10

= 10(k+1)²+13k+13

= 10(k+1)²+13(k+1)

= (k+1)(10(k+1)+13)

Dengan demikian, 23+43+63+83+...+(20k+3)+(20(k+1)+3) = (k+1)(10(k+1)+13). Akibatnya, P(n) terbukti untuk n = k+1.

Kesimpulan

Karena P(n) terbukti benar untuk n = 1 dan n = k+1, maka P(n) terbukti benar.

26. Kuis[+50]: Tentukan suku ke-24 jika suku ke-11 adalah 43 dan jika suku ke-21 adalah 83. [asal, report]


Jawaban:

U24 = 95

Pembahasan Materi:

Barisan aritmatika adalah suatu baris yang nilai setiap sukunya didapatkan dari suku sebelumnya melalui penjumlahan atau pengurangan dengan suatu bilangan.

Rumus barisan aritmatika:

Un = U1 + (n - 1) × b

     = a + (n - 1) × b

-------

Deret aritmatika adalah suatu penjumlahan suku-suku dari barisan aritmatika.

Rumus deret aritmatika:

Sn = ½ × n × (a + Un)

     = ½ × n × (2a + (n - 1) × b)

Keterangan:

Un = suku ke-n

U1 = a = suku pertama

b = beda

n = banyak suku pada barisan aritmatika

Penyelesaian:

Diketahui:

Suku ke-11 (U11) = 43

Suku ke-21 (U21) = 83

Ditanya: Suku ke-24 (U24) = ?

Dijawab:

- Eliminasikan nilai a.

U11 = a + 10b = 43

U21 = a + 20b = 83 -

---------------------------

-10b = -40

b = (-40) ÷ (-10)

= 4

- Subtitusikan nilai b.

a + 10b = 43

a + 10(4) = 43

a + 40 = 43

a = 43 - 40

= 3

- Menghitung nilai U24.

U24 = a + (n - 1) × b

= 3 + (24 - 1) × 4

= 3 + 23 × 4

= 3 + 92

= 95

Jadi, suku ke-24 dari barisan tersebut adalah 95.

Suku ke-24 apabila diketahui suku kesebelas bernilai 43 dan suku ke-21 nilainya 83 adalah 95

Pendahuluan:

Barisan Aritmatika merupakan suatu barisan bilangan dengan beda dua suku berurutan nilainya konstan dengan bentuk umum barisannya yaitu U₁, U₂, U₃, ..., Un

Rumus suku ke-n:

Un = a + (n - 1)b

Rumus menentukan selisih dua barisan berurutan:

b = Un - U(n - 1)Penyelesaian:

Diketahui:

U11 = 43

U21 = 83

Ditanyakan:

Suku ke-dua puluh empat?

Dijawab:

U11 = a + 10b

U21 = a + 20b

Eliminasi nilai a untuk memperoleh beda barisan aritmatika!

a + 10b = 43

a + 20b = 83

___________-

-10b = -40

b = -40/-10

b = 4

Setelah diperoleh beda/selisih antara dua suku yang berurutan, substitusikan kedalam salah 1 persamaan untuk mencari nilai suku pertamanya, kemudian jikalau sudah ketemu beda dan U1 atau nilai a, masukkan kedalam rumus umum

a + 10b = 43

a + 10(4) = 43

a + 40 = 43

a = 43 - 40

a = 3

Un = a + (n - 1)b

U24 = 3 + (24 - 1) . 4

U24 = 3 + 23 . 4

U24 = 92 + 3

U24 = 95

Kesimpulan:

Berdasarkan pembahasan di atas, nilai suku ke-24 apabila diketahui suku kesebelas bernilai 43 dan suku ke-21 nilainya 83 adalah 95
