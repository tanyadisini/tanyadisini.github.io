7. Dari Suatu deret arit matika diketahui U3 = 13 dan U7 =29. Jumlah 25 Suku Pertama Deret tersebutal. ​


Jawab:

Untuk menemukan jumlah 25 suku pertama dari suatu deret aritmatika, kita perlu tahu nilai suku pertama (U1) dan selisih antara setiap dua suku berturut-turut (d).

Dalam deret aritmatika, kita dapat menggunakan rumus umum untuk suku ke-n (Un):

Un = U1 + (n - 1)d

Kita telah diberikan informasi U3 = 13 dan U7 = 29. Mari kita gunakan informasi ini untuk menemukan nilai U1 dan d.

Untuk U3 (suku ke-3):

U3 = U1 + (3 - 1)d

13 = U1 + 2d   ...(1)

Untuk U7 (suku ke-7):

U7 = U1 + (7 - 1)d

29 = U1 + 6d   ...(2)

Sekarang, kita memiliki sistem dua persamaan dengan dua variabel U1 dan d. Kita dapat menyelesaikannya dengan cara mengurangkan persamaan (1) dari persamaan (2):

(2) - (1):

29 - 13 = (U1 + 6d) - (U1 + 2d)

16 = 4d

d = 4

Setelah menemukan nilai d (selisih antara suku-suku berturut-turut), kita bisa menggunakannya untuk mencari nilai U1 (suku pertama). Mari kita gunakan salah satu persamaan di atas, misalnya, persamaan (1):

13 = U1 + 2d

13 = U1 + 2(4)

13 = U1 + 8

Kurangkan 8 dari kedua sisi:

U1 = 13 - 8

U1 = 5

Sekarang kita telah menemukan U1 = 5 dan d = 4. Kita dapat menggunakan rumus untuk jumlah n suku pertama dari deret aritmatika (Sn):

Sn = (n/2)[2U1 + (n-1)d]

Dalam hal ini, n = 25, U1 = 5, dan d = 4. Mari kita substitusi:

S25 = (25/2)[2(5) + (25-1)(4)]

S25 = (25/2)[10 + 24(4)]

S25 = (25/2)[10 + 96]

S25 = (25/2)[106]

S25 = 53 * 106
