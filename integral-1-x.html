Integral

Integral merupakan invers atau kebalikan dari pen diferensialan. Atau bisa kita katakan juga, integral adalah lawan dari turunan.

[tex] \\ [/tex]

Jika f(x) adalah turunan pertama dari fungsi F(x). Maka dalam integral dituliskan dalam bentuk [tex] \tt \int f(x) dx = F(x) + C [/tex]

Jenis integral ada dua, yaitu integral tak tentu & integral tertentu. Bentuk umum nya:

Integral tak tentu : [tex] \boxed{\tt \int f(x) dx = F(x) + C} [/tex]Integral tertentu : [tex] \boxed{\tt \int \limits_{b}^{a} f(x)\: dx = F(x) = F(b) - F(a)} [/tex]

Sifat-sifat integral tak tentu

[tex] \boxed{\tt \int c . f(x) dx = c \int f(x) dx} [/tex]

[tex] \boxed{\tt \int [f(x) + g(x)] dx = \int f(x) dx + \int g(x) dx } [/tex]

[tex] \boxed{\tt \int [f(x) - g(x)] dx = \int f(x) dx - \int g(x) dx} [/tex]

Aturan integral tak tentu dari fungsi aljabar (rumus dasar)

[tex] \tt \int dx = x + C [/tex]

[tex] \tt \int a \: dx = ax + C [/tex]

[tex] \tt \int ax^{n} dx = \frac{a}{n + 1}x^{n + 1} + C [/tex]

[tex] \tt \int x^{n} dx = \frac{1}{n + 1}x^{n + 1} + C [/tex]

[tex] \tt \int \frac{1}{x} dx = In |x| + C [/tex]

[tex] \tt \int e^{x} dx = e^{x} + C [/tex]

[tex] \tt \int p^{x}dx = \frac{p^{x}}{In \: p} + C [/tex]

[tex] \\ \\ [/tex]

Penyelesaian

Gunakan rumus : [tex] \tt \int ax^{n} dx = \frac{a}{n + 1}x^{n + 1} + C [/tex]

Penjelasan dengan langkah-langkah:

Soal a

[tex]\displaystyle \int \frac{x+1}{x^2-4x-12}~dx=\int \frac{x+1}{(x-6)(x+2)}~dx[/tex]

Faktor linear berbeda. Diselesaikan dengan pecahan parsial dekomposisi

[tex]\begin{aligned}\frac{x+1}{(x-6)(x+2)}&\:=\frac{A}{x-6}+\frac{B}{x+2}\\x+1\:&=A(x+2)+B(x-6)\\x+1\:&=(A+B)x+2A-6B\end{aligned}[/tex]

Kesamaan polinomal

A + B = 1 → A = 1 - B

substitusi

2A - 6B = 1

2(1 - B) - 6B = 1 → B = ⅛

maka:

A = 1 - ⅛ = ⅞

sehingga:

[tex]\begin{aligned}\int \frac{x+1}{x^2-4x-12}~dx&\:=\int \frac{x+1}{(x-6)(x+2)}~dx\\\:&=\int \left ( \frac{7}{8(x-6)} +\frac{1}{8(x+2)}\right )~dx\\\:&=\frac{7}{8}\ln|x-6|+\frac{1}{8}\ln|x+2|+C\\\:&=\frac{7\ln|x-6|+\ln|x+2|}{8}+C\end{aligned}[/tex]

Soal b.

[tex]\displaystyle \int \frac{2x+1}{x^2-7x+6}~dx=\int \frac{2x+1}{(x-6)(x-1)}~dx[/tex]

Faktor linear berbeda.

[tex]\begin{aligned}\frac{2x+1}{(x-6)(x-1)}&\:=\frac{A}{x-6}+\frac{B}{x-1}\\2x+1\:&=A(x-1)+B(x-6)\\2x-1\:&=(A+B)x-(A+6B)\end{aligned}[/tex]

Kesamaan polinomial

A + B = 2

A + 6B = -1

________

-5B = 3 → B = -⅗

substitusi

A - ⅗ = 2 → A = 13/5

diperoleh:

[tex]\begin{aligned}\int \frac{2x+1}{x^2-7x-6}~dx&\:=\int \frac{2x+1}{(x-6)(x-1)}~dx\\\:&=\int \left ( \frac{13}{5(x-6)} -\frac{3}{5(x-1)}\right )~dx\\\:&=\frac{13}{5}\ln|x-6|-\frac{3}{5}\ln|x-1|+C\\\:&=\frac{13\ln|x-6|-3\ln|x-1|}{5}+C\end{aligned}[/tex]

Soal c.

[tex]\displaystyle \int \frac{dx}{x^2-1}=\int \frac{dx}{(x-1)(x+1)}[/tex]

Faktor linear berbeda.

[tex]\begin{aligned}\frac{1}{(x-1)(x+1)}&\:=\frac{A}{x-6}+\frac{B}{x-1}\\1\:&=A(x-1)+B(x+1)\\1\:&=(A+B)x-(A-B)\end{aligned}[/tex]

Kesamaan polinomial

A + B = 0 → A = -B

Substitusi

A - B = -1

-B - B = -1 → B = ½ → A = -½

sehingga:

[tex]\begin{aligned}\int \frac{dx}{x^2-1}&\:=\int \frac{dx}{(x-1)(x+1)}\\\:&=\int \left ( -\frac{1}{2(x-1)} +\frac{1}{2(x+1)}\right )~dx\\\:&=\frac{1}{2}\ln|x+1|-\frac{1}{2}\ln|x-1|+C\\\:&=\frac{\ln|x+1|-\ln|x-1|}{2}+C\\\:&=\frac{1}{2}\ln\left | \frac{x+1}{x-1} \right |+C\\\:&=\coth^{-1}+C\end{aligned}[/tex]

Hasil dalam fungsi invers hiperbolik.

33. tentukan nilai integral .. integral x(x+4)(x-1)dx


Hasil integral dari ∫ x (x+4) (x-1) dx adalah ¹/₄ x⁴ + x³ - 2x² + C. Nilai dari "C" dapat diperoleh setelah ditentukan batas atas dan bawah dari integral tak tentu pada soal.

Penjelasan dengan langkah-langkah

Integral merupakan konsep matematika yang memberikan gambaran fungsi luas dan volume melalui penjumlahan data yang bernilai kecil dan tidak dapat diukur secara individual. Berikut rumus umum integral:

[tex]\bf \int {x^n} \, dx = \dfrac{1}{n+1}x^{n+1}+C[/tex]

Keterangan:

x = variabel.n = angka yang menunjukkan pangkat awal.C = konstanta.

Diketahui:

[tex]\sf \int {(x)(x+4)(x-1)} \, dx[/tex]

Ditanyakan:

Nilai integral tak tentu di atas =?

Penyelesaian:

[tex]\begin{array}{ll}\sf \int {(x)(x+4)(x-1)} \, dx &\sf = \int (x^2+4x)(x-1)~dx\\\\&\sf = \int (x^3-x^2+4x^2-4x)~dx\\\\&\sf = \int (x^3+3x^2-4x)~dx\\\\&\sf = \dfrac{1}{3+1}x^{3+1}+\dfrac{3}{2+1}x^{2+1}-\dfrac{4}{1+1}x^{1+1}+C\\\\&\sf = \dfrac{1}{4}x^4+x^3-2x^2+C\end{array}[/tex]
