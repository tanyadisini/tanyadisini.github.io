6. gustave verspijk adalah orang belanda yang melawan...​


Jawaban:

Gustave verspijk adalah orang belanda yang melawan pangeran hidayatullah dan demang leman

Penjelasan:

Hindia Timur atau Indonesia telah lama dikenal sebagai daerah penghasil rempah-rempah seperti vanili, lada, dan cengkeh. Rempah-rempah ini digunakan untuk mengawetkan makanan, bumbu masakan, bahkan obat. Karena kegunaannya, rempah-rempah ini sangat laku di pasaran dan harganya pun mahal. Hal ini mendorong para pedagang Asia Barat datang dan memonopoli perdagangan rempah-rempah. Mereka membeli bahan-bahan ini dari para petani di Indonesia dan menjualnya kepada para pedagang Eropa.  

Secara umum, kedatangan bangsa Eropa ke Asia termasuk ke Indonesia dilandasi keinginan mereka untuk berdagang, menyalurkan jiwa penjelajah, dan menyebarkan agama. Adapun sebab dan tujuan bangsa Eropa ke dunia Timur adalah sebagai berikut :

1.      Mencari kekayaan termasuk berdagang (Gold)

2.      Mencari kemuliaan bangsa (Glory)

3.      Menyebarkan agama (Gospel)

Sejak abad ke-3, rempah-rempah memang merupakan bahan dagang yang sangat menguntungkan. Hal ini mendorong orang-orang Eropa berusaha mencari harta kekayaan ini sekalipun menjelajah semudera. ada awalnya, tujuan kedatangan bangsa Eropa ke Indonesia hanya untuk membeli rempah-rempah dari para petani Indonesia. Namun, dengan semakin meningkatnya kebutuhan industri di Eropa akan rempah-rempah, mereka kemudian mengklaim daerah-daerah yang mereka kunjungi sebagai daerah kekuasaannya. Di tempat-tempat ini, bangsa Eropa memonopoli perdagangan rempah-rempah dan mengeruk kekayaan alam sebanyak mungkin. Dengan memonopoli perdagangan rempah-rempah, bangsa Eropa menjadi satu-satunya pembeli bahan-bahan ini. Akibatnya, harga bahan-bahan ini pun sangat ditentukan oleh mereka. Untuk memperoleh hak monopoli perdagangan ini, bangsa Eropa tidak jarang melakukan pemaksaan. Penguasaan sering dilakukan terhadap para penguasa setempat melalui suatu perjanjian yang umumnya menguntungkan bangsa Eropa.  Adapun bangsa eropa yang datang ke Indonesia antara lain Bangsa Portugis, Bangsa Spanyol,  Bangsa Inggris,  dan yang terakhir adalah Bangsa Belanda.  

Lahirnya VOC

Untuk mengatasi persaingan diantara pedagang-pedagang Belanda sendiri, pada tanggal 20 Maret 1682 Belanda membentuk VOC (Vereenigde OostIndische Compagnie) atau persekutuan Dagang Hindia Timur atas usulan Johan Van Oldenbarneveld. Tujuan pembentukan VOC tidak lain adalah menghindari persaingan antar pengusaha Belanda (intern) serta mampu menghadapi persaingan dengan bangsa lain terutama Spanyol dan Portugis sebagai musuhnya (ekstern). VOC dipimpin oleh De Heren Zuventien (Dewan Tujuh Belas) yang berkedudukan di Amsterdam. Oleh Pemerintahan Belanda, VOC diberi oktroii (hak-hak istimewa). Artinya dengan hak-hak tersebut berarti VOC memiliki kekuasaan seperti suatu negara. Mereka dapat bertindak bebas tanpa harus konsultasi terlebih dulu dengan pemerintah Belanda di negeri induk. Hak-hak istimewa tersebut adalah sebagai berikut:

Dianggap sebagai wakil pemerintah Belanda di Asia

Memonopoli perdagangan

Mencetak dan mengedarkan uang sendiri

Mengadakan perjanjian

Menaklukkan perang dengan negara lain

Menjalankan kekuasaan kehakiman

Pemungutan pajak

Memiliki angkatan perang sendiri

Mengadakan pemerintahan sendiri

カレンダわ どれです か? [ Karendaa wa dore desu ka? ]

Artinya : "Kalender yang mana?"

•> カレンダ [Karendaa] artinya kalender.

•> わ [wa] digunakan sebagai penjelas dari suatu kalimat.

•> どれ [Dore] artinya yang mana. Digunakan untuk menanyakan sesuatu yang memiliki rupa yang sama namun memiliki perbedaan yang mencolok. Misalnya pita biru dan merah.

•> です [Desu] digunakan sebagai bentuk kalimat itu positif.

•> か [Ka] digunakan sebagai bentuk kalimat tersebut adalah tanda tanya.

18. bagaimana membuat kalimat dengan menggunakan will,can,may,should,shall,would,need dan dore


1. Will digunakan saat membicarakan masa yg akan datang ( future tense )

Contoh : i will get merried next month ( aku akan nikah bulan depan )

Atau bisa berupa kalimat memerintah

Contoh : will you take me?? ( mau gk kamu antar aku )

2. Can digunakan sebagai ability ( kemampuan )
Contoh : i can speak english ( aku bisa bicara inggris )

Atau bisa juga kalimat perintah informal tapi penggunaan can untuk sesama teman sehari2..

Can you help me?? ( bisa bantuin aku )

3. May digunakan untuk ijin

May i say something?? ( bolehkah aku mengatakan sesuatu )

Atau bisa juga digunakan saat meminjam barang dan permintaan halus..
Contoh : may i borrow your pen ( boleh aku minjam pulpen )

4. Should digunakan keharusan atau seharusnya ( saran ) anjuran..

Contoh : you should take a rest, now ( kamu harusnya tidur sekarang )

5. Shall berupa kata ajakan atau tawaran, dan keharus tapi secara pasif

Contoh : then, shall we do, now?? ( lalu, apa yg kita harus lakukan sekarang??
Shall we bring the stuff?? ( haruskah kita bawa barangnya??

6. Would bisa dijadikan kalimat permintaan sopan

Contoh : would you take the pen ( maukah kamu ambil pulpen itu )

Atau bisa digunakan kata lampaunya will

Contoh : i would get merried

7. Need situasi digunakan saat kamu perlu sesuatu
Contoh : i need some sugar ( aku butuh beberapa gula )

8. Done ( kata lampau dari do ) yg artinya telah mengerjakan

Contoh : he was done his homework ( dia sudah menyelesaikan prnya )

19. Digunakan untuk menujukan benda yang dekat dengan pembicara dare dore sore kore kareDigunakan untuk menujukan benda yang dekat dengan pembicara dare dore sore kore kare


Kata yang digunakan untuk menunjukkan benda yang dekat dengan pembicara adalah Kore yang artinya adalah "Ini".

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

Kata tunjuk dalam Bahasa Jepang disebut dengan Shiji-shi. Kegunaan nya yaitu untuk menunjukkan sesuatu. Kata tunjuk dalam Bahasa Jepang dibagi menjadi 4 berdasarkan awalan bunyi tiap kata nya. Awalan bunyi kata tunjuk dalam Bahasa Jepang adalah Ko-So-A-Do.

Kata tunjuk berawalan Ko : Digunakan untuk menunjukkan sesuatu yang dekat dengan pembicara. Kata tunjuk dengan awalan bunyi ini umumnya diartikan dengan "Ini", "Sini", dan sebagainya.

Kata tunjuk berawalan So : Digunakan untuk menunjukkan sesuatu yang jauh dengan pembicara, tapi dekat dengan lawan bicara. Kata tunjuk dengan awalan bunyi ini umumnya diartikan dengan "Itu", "Situ", dan sebagainya.

Kata tunjuk berawalan A : Digunakan untuk menunjukkan sesuatu yang jauh dengan pembicara juga lawan bicara. Kata tunjuk dengan awalan bunyi ini kalau diartikan kurang lebih mirip seperti yang berawalan So, yaitu dengan "Itu", "Sana", dan sebagainya.

Kata tunjuk berawalan Do : Digunakan untuk menanyakan sesuatu yang ditunjukkan. Kata tunjuk dengan awalan bunyi ini umumnya diartikan dengan "Mana", "Yang mana", dan sebagainya.

- - - - - - - - -

Kore / Sore / Are / Dore

Keempat kata tersebut merupakan kata tunjuk benda, yang di mana Kore berarti "Ini", Sore dan Are berarti "Itu", dan Dore berarti "Yang mana". Penggunaan kata-kata nya seperti yang sudah saya jelaskan di atas, yaitu dibedakan dari awalan bunyi nya. Perbedaan penggunaannya tidak saya jelaskan lagi masing-masing, namun intinya keempat kata tunjuk tersebut digunakan untuk menunjuk benda.

Dare

Kata ini tidak termasuk ke kata tunjuk, melainkan merupakan kata tanya yang memiliki arti "Siapa".

Kare

Kata ini juga tidak termasuk ke kata tunjuk, melainkan merupakan kata ganti orang ketiga yang memiliki arti "Dia". Yang dimaksud dia di sini hanya untuk laki-laki saja. Jika untuk perempuan menggunakan Kanojo.

37. 1.(Active) Gustave Eiffel built the most iconic tower of Paris in 1889 2.(Active) Somebody delivers the daily newspaper to our door every morning 3.(ACTIVE) Water covers most of the earth surface


PASSIVE VOICE

1. The most iconic tower of Paris was built by Gustave Eiffel.

2. The daily newspaper is delivered to our door every morning.

3. Most of earth' s surface is covered by water.

EXPLANATION

Passive voice is used when we want to emphasize the recipient of the verb and the action. We can omit the agent (the doer of action) if the agent isn't obvious or unknown.

The main construction of passive voice is the combination of auxiliary BE and past participle verb.

BASED ON YOUR QUESTION

1) Simple past

➡ Active : S+Verb2+O

➡ Passive : S(O)+to be(was/were)+Verb3+by agent.

Subject (Gustave Eiffel)+Verb(built)+ Object (the most iconic tower of Paris)+Adverb (in 1889)

changes to :

New subject (The most iconic tower of Paris )+Verb (was built)+by+Agent (Gustave Eiffel)+Adverb (in 1889).

2.Simple present

➡ Active : S+Verb1(s/es)+O

➡ Passive : S(O)+to be(is/are/am)+Verb3+by agent.

Subject (somebody)+Verb(delivers)+Object (the daily newspaper)+PP( to our door)+Adverb (every morning)

changes to :

New subject(the daily newspaper)+Verb (is delivered)+PP (to our door)+Adverb (every morning).

The word "someone" is omitted because it doesn't refer to the spesific person.

3. Simple present

➡ Active : S+Verb1(s/es)+O

➡ Passive : S(O)+to be(is/are/am)+Verb3+by agent.

Subject (Water)+Verb(covers)+Object (most of the earth's surface)

changes to :

New subject(most of the earth's surface)+Verb (is covered)+by+ Agent(water).

1. Formal Audiency, merupakan pendengar formal. Dalam formal audiency, orang-orang mendengarkan informasi yang disampaikan orang lain secara benar serta informasi tersebut memliki faedah yang banyak. Contohnya, seperti jamaah mendengarkan khutbah Jumat. Khutbah Jumat tersebut termasuk ceramah yang memiliki faedah yang amat besar. Maka dari itu, jamaah adalah pendengar formal.

2. Spectator Causal Crowds, merupakan suatu kerumunan orang-orang atau penonton yang ingin melihat suatu peristiwa tertentu. Dalam spectator causal crowds, orang-orang atau penonton beramai-ramai untuk saling menyaksikan apa yang dilihatnya. Baik itu peristiwa/kejadian yang sedang berlangsung, atau suatu pertunjukkan serta pertandingan yang layak untuk di tonton oleh banyak orang. Sehingga menyebabkan banyaknya kerumunan dari orang-orang tersebut dikarenakan adanya rasa penasaran atau rasa skeptis terhadap peristiwa yang ingin dilihatnya. Contohnya, seperti menyaksikan pertandingan sepak bola secara langsung di stadion.

3. Immoral Crowds, merupakan suatu kerumunan yang tidak adanya rasa moral atau tidak bermoral. Dalam immoral crowds, orang-orang yang sedang berkumpul dianggap tidak bermoral dikarenakan perbuatan yang dilakukan dapat merusak moral bangsa atau dianggap masyarakat sebagai suatu perbuatan yang tidak ada faedahnya sama sekali baik masa sekarang maupun masa mendatang. Contohnya, yaitu pesta miras. Pesta miras dianggap tidak bermoral karena hal itu tidak baik atau tidak mencerminkan rasa etis terhadap masyarakat atau bangsa. Dalam pesta miras, orang-orang yang melakukan perbuatan tersebut mencerminkan rasa gaya hidup yang prestise, serta lama kelamaan dapat menimbulkan sifat hedonisme yang berlebihan atau bahkan lebih kacau lagi mungkin sifat sekularisme pun dapat terjadi.

4. Panic Causal Crowds, merupakan suatu kerumunan orang-orang yang panik, dan berusaha untuk menyelamatkan diri dari bahaya. Dalam panic causal crowds, orang-orang bersifat tegang dan amat panik untuk menyelamatkan dirinya dari musibah yang menimpanya. Contohnya, adalah saat terjadi tsunami banyak orang yang berkeliaran dan berlari-larian untuk menyelamatkan dirinya dari terjangan air tsunami tersebut. Serta, mencari tempat yang lebih aman untuk dirinya dalam waktu sementara. Dalam keadaan seperti itu, tidak hanya rasa panik yang timbul, rasa sedih pun dapat terjadi dikarenakan adanya bencana tsunami.
