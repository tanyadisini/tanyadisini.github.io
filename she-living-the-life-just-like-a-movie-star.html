Explanation

✓ Penggunaan SO

So digunakan dalam kalimat positif. Kata so muncul setelah and dan diikuti oleh auxiliary verb dan subject.

Rumus: Sentence + and + so + aux + S.

✓ Penggunaan NEITHER

Neither digunakan pada kalimat negatif.

Rumus: Sentence + and + neither + aux + S

PENJELASAN

Kalimat nomor 1-5 di jawaban dibuat menggunakan rumus kalimat verba Simple Past Tense. Rumusnya:

(+ / Positif) S + V2 + O

(- / Negatif) S + didn't + V1 + O

(? / Interogatif) Did + S + V1 + O?

Untuk kalimat nomor 6-10 di jawaban, rumus yang digunakan adalah rumus kalimat nominal Simple Past Tense. Rumusnya:

(+ / Positif) S + was/were + C

(- / Negatif) S + wasn't/weren't + C

(? / Interogatif) Was/Were + S + C?

*S = subject; V = verb; O = object; C = complement
