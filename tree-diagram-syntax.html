5. tolong tulis kan tree diagram syntax dari kalimat " I have two brother "​


Phrase structure tree

Explanation

Syntax merupakan salah satu cabang dari ilmu kebahasaan yang mempelajari tentang struktur pembentukan kata-kata menjadi satu kesatuan kalimat terstruktur dan bermakna jelas.

Penggunaan pohon diagram syntax merupakan penggambaran bagaimana adanya hubungan hirarki antara komponen-komponen/kata-kata yang terkandung dalam suatu kalimat.  

Penyusunan pohon diagram syntax/pohon struktur frasa berdasarkan karakteristiknya { syntax category }  :  

1. Kelas kata { Parts of speech }

Parts of speech  meliputi determiner (det), Adjective (Adj), Noun (N), Pronoun (Pro), Preposition (P), Adverb ( Adv), Auxiliary ( Aux), Verb ( V).

2.  Tatanan penyusunan frasa ( Phrase Structure Grammar )

Pola penyusunan meliputi Noun phrase (NP), Adjective phrase ( AdjP), Verb phrase (VP), Adverb phrase(AdvP), Preposition phrase(PP) .

5 frasa utama gramatikal dala analisis pohon diagram syntax :

1. Noun phrase  

Sekumpulan kata/frasa  yang memodifikasi kata benda.

Exp :  

The old woman crossed the street.

2. Adjective phrase

Frasa yang memberikan penjelasan detail dari kata benda. Frasa terdiri dari keterangan dan kata sifat.

Exp :

She is very clever.

3. Verb phrase  

Frasa yang terbentuk paling tidak terdapat satu kata kerja utama dengan didukung kata kerja pengikat/ pelengkap kata kerja.

Exp  :

The gold miner found the diamond.

4. Adverb phrase

Frasa yang berperan memberikan penjelas keterangan waktu (when), alasan (why), keterangan tempat (where) dan bagaimana sesuatu itu terjadi (How).

Exp :

They are going to go to Bali next week.

5. Preposition phrase  

Frasa yang berperan sebagai kata depan maupun imbuhan.

Exp :

He went to Surabaya by a train.

Konsep Penyusunan frasa dalam pohon diagram syntax

1. S=> NP  VP

2. NP=> {det}  {Adj}  N  {PP}  

3. VP=> V { NP } { PP} { Adv}

4. PP => P  NP

5. AP => A { PP }  

Contoh struktur frasa :

1. The old woman => det Adj N

2. The mouse in the cage => det N PP

3. Catch that mouse => V NP

4. Enter the house quietly=>V NP Adv

5. Eat an apple in the kitchen=> V NP PP

6. Run into the hole quickly => V PP Adv

Maka, penyusunan pohon diagram syntax dari kalimat “I have two brothers ”  :

I => N

have => V

two brothers => det N

19. tolong tulis kan tree diagram syntax dari kalimat " I am the first child "​


PHRASE STRUCTURE TREE

Explanation

Syntax merupakan salah satu cabang dari ilmu kebahasaan yang mempelajari tentang struktur pembentukan kata-kata menjadi satu kesatuan kalimat terstruktur dan bermakna jelas.

Penggunaan pohon diagram syntax merupakan penggambaran bagaimana adanya hubungan hirarki antara komponen-komponen/kata-kata yang terkandung dalam suatu kalimat.  

Penyusunan pohon diagram syntax/pohon struktur frasa berdasarkan karakteristiknya { syntax category }  :  

1. Kelas kata { Parts of speech }

Parts of speech  meliputi determiner (det), Adjective (Adj), Noun (N), Pronoun (Pro), Preposition (P), Adverb ( Adv), Auxiliary ( Aux), Verb ( V).

2. Tatanan penyusunan frasa ( Phrase Structure Grammar )

Pola penyusunan meliputi Noun phrase (NP), Adjective phrase ( AdjP), Verb phrase (VP), Adverb phrase(AdvP), Preposition phrase(PP) .

5 frasa utama gramatikal dalam  pohon diagram syntax :

1. Noun phrase  

Sekumpulan kata/frasa  yang memodifikasi kata benda.

Exp :  

The old woman crossed the street.

2. Adjective phrase

Frasa yang memberikan penjelasan detail dari kata benda. Frasa terdiri dari keterangan dan kata sifat.

Exp :

She is very clever.

3. Verb phrase  

Frasa yang terbentuk paling tidak terdapat satu kata kerja utama dengan didukung kata kerja pengikat/ pelengkap kata kerja.

Exp :

The gold miner found the diamond.

4. Adverb phrase

Frasa yang berperan sebagai penjelas kata kerja, memberikan informasi detail keterangan waktu (when), alasan (why), keterangan tempat (where) dan bagaimana sesuatu itu terjadi (How).

Exp :

They are going to go to Bali next week.

5. Preposition phrase  

Frasa yang berperan sebagai kata depan maupun imbuhan.

Exp :

He went to Surabaya by a train.

Konsep Penyusunan frasa dalam pohon diagram syntax

1. S=> NP  VP

2. NP=> {det }  {Adj}  N  {PP}  

3. VP=> V { NP } { PP} { Adv}

4. PP => P  NP

5. AP => A { PP }  

Contoh struktur frasa :

1. The old woman => det Adj N

2. The mouse in the cage => det N PP

3. Catch that mouse => V NP

4. Enter the house quietly=>V NP Adv

5. Eat an apple in the kitchen=> V NP PP

6. Run into the hole quickly => V PP Adv

Conclusion

Maka, penyusunan pohon diagram syntax dari kalimat “I am the first child”  :

I => N  

am => Aux

The first child => det Adj N

33. tolong tuliskan tree diagram syntax dari kalimat " recently, the old men and women left"​


TREE DIAGRAM SYNTAX

AdvP (NP) {det} {NP} V

Jawaban rinci bisa dilihat di lampiran gambar

EXPLANATION

Tree diagram syntax adalah diagram bersusun yg menunjukkan struktur  kehirarkian komponen kalimat.

Struktur paling atas dari pohon diagram Syntax, ditandai dengan  simbol "S" di mana singkatan dari "Sentence" (Kalimat).

Kemudian dari sini, dibagi lagi ke dalam 2 konstituen (grup kecil dlm kalimat); NP & VP.  

Urutan paling kiri dimulai dengan simbol NP (Noun Phrase) di mana diisikan oleh  awalan (determiner), kata sifat (adjective), kata benda (Noun) dan sebagainya.

Sedangkan, yg paling kanan diisi oleh VP (Verb Phrase) di mana dicabangkan dengan frasa nomina (NP), frasa preposisi (PP), kata kerja (V) dan sebagainya.

Simbol² dlm analisis sintaksis linguistik

S : Sentence

NP : Noun Phrase

VP  : Verb Phrase

V    : Verb

N    : Noun

Det  : Determiner

PP   : Prepositional Phrase

Pro   : Pronoun

PN     : Proper Noun

AdvP : Adverb Phrase

AP : Adjective Phrase

Frasa utama dalam pohon diagram syntax

1)  S ➡ NP VP

             S

           /    \

         NP  VP

2) NP ➡ Det N

             S

           /   \

        NP    VP

        /  \

     Det  N

4) VP ➡ V PP

                S

             /     \

           NP     VP

           / \       /  \

       Det N    V  PP

5) PP ➡ Prep NP

                S

             /     \

           NP     VP

           / \      /   \

        Det  N  V    PP

                           /   \

                       Prep  NP

Contoh struktur frasa & pohon diagramnya :

1. The little cat  : det Adj N

                       NP

                    I        I       I       /   \

                    I        I       I     det   N

                 The police in   the  office

3. Teach the students :  V NP

                       VP

                    /        \

                  V          NP
