19. Hasil kali usia danang dan ria adalah 180 tahun.ria 3 tahun lebih muda daripada danang. Usia danang adalah


Misalkan :
⇒ Danang = a
⇒ Ria = b

a x b = 180

a = b + 3

Maka,

a x b = 180

(b + 3) x b = 180

b² + 3b = 180

b² + 3b - 180 = 0

(b + 15)(b - 12) = 0

b + 15 = 0
b = -15 → tanda minus untuk Umur tidak memenuhi


b - 12 = 0
b = 12 Tahun ⇒ Umur Ria

Maka, Umur danang

= b + 3
= 12 + 3
= 15 Tahun ✔
