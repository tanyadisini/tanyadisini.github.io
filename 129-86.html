9. bentuk sederhana dari 86/129


Bentuk sederhana dari [tex]\frac{86}{129}[/tex] adalah [tex]\frac{2}{3}[/tex].

Penyelesaian Soal :

Untuk menyederhanakan pecahan dapat dilakukan dengan cara sebagai berikut :

LANGKAH PERTAMA (I)

Tentukan FPB dari 86 dan 129.

FPB merupakan faktor persekutuan terbesar, dan cara hitungnya yaitu :

   86               129

    ∧                  ∧

 2   43          3   43

Nilai faktorisasi

86 = 2 × 43

129 = 3 × 43

Jadi FPB dari 86 dan 129 adalah 43.

LANGKAH KEDUA (II)

Kemudian membagi pembilang dan penyebut dengan FPB. Dengan cara sebagai berikut :

[tex]\frac{86}{129}[/tex] ÷ [tex]\frac{43}{43}[/tex] = [tex]\frac{2}{3}[/tex]

∴ Kesimpulan Bentuk sederhana dari [tex]\frac{86}{129}[/tex] adalah [tex]\frac{2}{3}[/tex].

34. OK Google diketahui tiga bilangan ganjil berurutan jumlahnya 129 jumlah bilangan terbesar dan terkecil bilangan tersebut adalah A 96 B 94 C 86 D 84


Mapel : Matematika
Bab : Bilangan 

Diketahui :
Tiga bilangan ganjil berurutan jumlahnya = 129
Bilangan pertama = a
Bilangan kedua = a + 2
Bilangan ketiga = a + 4

Ditanyakan :
Jumlah bilangan terbesar dan terkecil = ...?

Penyelesaian :
129 = a + a + 2 + a + 4 
129 = 3a + 6
  3a = 129 - 6
  3a = 123
    a = 123/3
    a = 41

Bilangan pertama = a = 41
Bilangan kedua = a + 2 = 41 + 2 = 43
Bilangan ketiga = a + 4 = 41 + 4 = 45

Jumlah bilangan terbesar dan terkecil
= 45 + 41
= 86Tiga bilangan ganjil berurutan
misal bilangan ganjil terkecil = x

Barisannya berarti : x, x+2, x+4 

**Kenapa x + 2 dan x+4? misal x = 1, berarti bilangan ganjil setelah 1 itu 3 yaitu x+2 dan bilangan ganjil setelah 3 itu 5 yaitu x+4

Jumlah bilangan adalah
x + x + 2 + x + 4 = 129
3x + 6 = 129
3x = 123
x = 41

Jumlah bilangan terbesar dan terkecil
Bilangan terkecil = x = 41
Bilangan terbesar = x + 4 = 45

Jumlah = 41+45 = 86 (C)
