2. Turunan dy/dx dari fungsi


soal:

[tex]\textcolor{#8D44AD}{2x^3y^2-5y^3+x^2=4x-3y\sqrt{y}+2x\sqrt{x}}=[/tex]

jawab :

[tex]=2x^3y^2-5y^3+x^2-4x+3y\sqrt{y}-2x\sqrt{x}=0[/tex]

[tex]=f\left(x,y\right)=2x^3y^2-5y^3+x^2-4x+3y\sqrt{y}-2x\sqrt{x}[/tex]

[tex]=f\left(x,y\right)=2x^3y^2-5y^3+x^2-4x+3y\sqrt{y}-2x\sqrt{x},\ f_x=?[/tex]

[tex]=f\left(x,y\right)=2x^3y^2-5y^3+x^2-4x+3y\sqrt{y}-2x\sqrt{x},\ f_y=?[/tex]

[tex]=f_x=6x^2y^2+2x-4-3\sqrt{x}[/tex]

[tex]=f_y=4x^3y-15y^2+\frac{9}{2}\cdot\sqrt{y}[/tex]

[tex]=\frac{dx}{dy}=-\frac{6x^2y^2+2x-4-3\sqrt{x}}{4x^3y-15y^2+\frac{9}{2}\cdot\sqrt{y}}[/tex]

[tex]=\frac{dx}{dy}=-\frac{6x^2y^2+2x-4-3\sqrt{x}}{\frac{8x^3y-30y^2+9\sqrt{y}}{2}}[/tex]

[tex]=\frac{dx}{dy}=-\frac{6x^2y^2+2x-4-3\sqrt{x}\cdot2}{8x^3y-30y^2+9\sqrt{y}}[/tex]

[tex]=\mathbf{nilai}\ \mathbf{\frac{dx}{dy}}\mathbf{=-\frac{12x^2y^2+4x-8-6\sqrt{x}}{8x^3y-30y^2+9\sqrt{y}}}[/tex]

4. jika y : 3x²/2x+3 turunan adalah dy/dx

[tex]\displaystyle y=\frac{3x^2}{2x+3}\\\\\\\frac{dy}{dx}=\frac{6x(2x+3)-3x^2(2)}{(2x+3)^2}\\\frac{dy}{dx}=\frac{12x^2+6x-6x^2}{(2x+3)^2}\\\frac{dy}{dx}=\frac{6x^2+6x}{(2x+3)^2}[/tex]

Beberapa konsep yang dipakai:

[tex]\displaystyle\triangleright~\frac{d}{dx}(y_1+y_2+\cdots_y_n)=y'_1+y'_2+\cdots+y'_n\\\triangleright~\frac{d}{dx}(ax^n)=anx^{n-1}\\\triangleright~\frac{d}{dx}\left(\frac uv\right)=\frac{u'v-uv'}{v^2}[/tex]

15. × pangkat 2 - XY + Y = 2 bantu jawab dong soal dy/dx turunan fungsi berantai


Untuk fungsi implisit, dalam menentukan dy/dx, turunkan kedua ruas terhadap x, hinga menjadi:

d/dx (x² - xy + y) = d/dx (2)

Rangkaian proses turunan menjadikan:
2x - (1.y + x.dy/dx) + dy/dx = 0

Catatan: turunan dari y terhadap x adalah dy/dx, bukan 0.

Kelompokkan yang mengandung dy/dx atau tidak,
(2x - y) + (1-x) dy/dx = 0

Dengan menempatkan dy/dx di ruas kiri:
dy/dx = (2x - y)/(1 - x)
dy/dx = (y - 2x)/(x - 1), dengan x ≠ 1

28. Tentukan turunan pertama dy/dx dari​


Jawab:

[tex]\frac{dy}{dx}=\frac{t^2+1}{t^2-1}[/tex]

Penjelasan dengan langkah-langkah:

TURUNAN FUNGSI PARAMETRIK

Diketahui :

[tex]x=t+\frac{1}{t}\\\\y=t-\frac{1}{t}[/tex]

Ditanya :

tentukan [tex]\frac{dy}{dx}[/tex]

Penyelesaian :

rumus turunan untuk fungsi pangkat adalah :

[tex]y=ax^k~~\to~~\frac{dy}{dx}=kax^{k-1}[/tex]

> cari [tex]\frac{dy}{dt}[/tex]

[tex]y=t-\frac{1}{t}\\\\y=t-t^{-1}\\\\\frac{dy}{dt}=1-(-1)t^{-2}\\\\\frac{dy}{dt}=1+\frac{1}{t^2}[/tex]

> cari [tex]\frac{dx}{dt}[/tex]

[tex]x=t+\frac{1}{t}\\\\x=t+t^{-1}\\\\\frac{dx}{dt}=1+(-1)t^{-2}\\\\\frac{dx}{dt}=1-\frac{1}{t^2}[/tex]

> cari [tex]\frac{dy}{dx}[/tex]

[tex]\frac{dy}{dx}=\frac{\frac{dy}{dt}}{\frac{dx}{dt}}\\\\\frac{dy}{dx}=\frac{1+\frac{1}{t^2}}{1-\frac{1}{t^2}}\\\\\frac{dy}{dx}=\frac{\frac{t^2+1}{t^2}}{\frac{t^2-1}{t^2}}\\\\\frac{dy}{dx}=\frac{t^2+1}{t^2-1}[/tex]
