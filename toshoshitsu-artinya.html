1. Toshoshitsu wa doko desuka



sebagai contoh=

pertanyaan = としょしつ わ どこ ですか？( kantin dimana)

jawaban = "あそこ です。“
disana


catatan = anda dapat mengubah jawaban anda menurut kondisi tempatnya

pola kalimat :
KB(nama tempat) wa KB(letak) desu.

ここ= disini ( jika tempat itu dekat dengan anda)
せこ= disitu ( jika tempat itu jauh dengan keberadaan anda tetapi dekat dengan keberadaan teman)
あそこ= disana ( jika tempat itu jauh dari keberadaan anda dan teman anda)

anda juga dapat mengubahnya dengan menggunakan kata / partikel berikut ini

となり=disamping
まえ= depan
うしろ= belakang
うえ=atas

rumusnya = tempat 1) は (tempat2) の (posisi) です

sebagai contoh =

pertanyaan = としょしつ わ どこ ですか？( kantin dimana)

jawaban = としょしつ は じむしつ の まえ です( perpustakaan ada didepan ruang tata usaha)

Toshoshitsu wa doko desuka

Arti Kalimat :
» Kantin ada dimana ? «

Bahasa Jepangnya :
» 図書室 は どこ ですか «

Arti Per Kalimat :
Toshoshitsu = 図書室 = Kantin
Wa = は (Kata Bantu)
Doko = どこ = Dimana
Desuka = ですか (Kalimat tanya)
