3. TOLONG BANTU JAWABSOAL : INFORMATIKAKELAS : 7TENTANG : SEJARAH KOMPUTER​


Jawaban:

Sejarah komputer berawal dari abad ke-19, ketika Charles Babbage mencoba untuk mendesain komputer mekanik yang bernama Mesin Analitikal. Desain ini menjadi basis kerangka desain komputer sekarang.

Secara garis besar, pengembangan komputer terbagi dalam empat generasi

Generasi pertama dimulai ketika komputer digunakan dalam akademi dan militer, seperti Komputer Atanasoff-Berry dibuat pada tahun 1937 untuk menyelesaikan sistem persamaan linear dan komputer Colossus untuk memecahkan kode rahasia Jerman Nazi. Kemudian, ENIAC yang dibangun pada tahun 1946 menjadi komputer pertama untuk tujuan umum. Ketika komputernya dinyalakan untuk pertama kali, Philadelphia mengalami mati listrik. Komputer generasi ini menggunakan tabung vakum untuk menyimpan data dan ukurannya memakan satu ruangan.

Generasi kedua dimulai ketika transistor menggatikan tabung vakum. UNIVAC diperkenalkan ke publik tahun 1951 untuk penggunaan komersial. Kemudian, pada tahun 1953, IBM memulai bisnis komputernya dengan merilis IBM 650 dan IBM 700. Berbagai bahasa pemrograman mulai dikembangkan dan komputer mulai memiliki memori dan sistem operasi.

Generasi ketiga dimulai ketika teknologi transistor meningkat menjadi sirkuit terpadu. Komputer mini adalah inovasi yang signifikan dalam generasi ini dan mempengaruhi generasi komputer selanjutnya. NASA menggunakan komputer generasi ini untuk melancarkan Program Apollo, seperti Komputer Bimbingan Apollo untuk mempermudah kendali Apollo Command/Service Module. Digital Equipment Corporation menjadi perusahaan komputer nomor dua di belakang IBM dengan komputer PDP dan VAX-nya. Komputer ini membawa ke pengembangan sistem operasi yang berpengaruh, Unix.

Generasi keempat dimulai pada dasawarsa 1970-an ketika penemuan MOSFET dan integrasi berskala besar selanjutnya membawa ke pengembangan mikroprosesor di awal 1970-an. komputer pribadi yang semakin kecil berkat mikroprosesor mulai berkembang, dimulai dari komputer rumahan dan komputer meja. Teknologi selanjutnya adalah laptop dan ponsel cerdas yang sangat fenomenal, membawa berbagai perusahaan teknologi ke dalam perang paten atas ponsel cerdas.

Pembahasan

Pohon Biner adalah struktur data seperti daftar tertaut untuk menyimpan koleksi data. Dalam ilmu komputer, Binary Tree adalah struktur data pohon di mana setiap node memiliki paling banyak dua anak. Secara khusus, anak-anak diberi nama kiri dan kanan. Salah satu penggunaan pohon biner yang umum adalah pohon biner terurut dan yang lainnya adalah tumpukan biner. Dalam algoritma diperkenalkanlah sebuah pohon biner himpunan terpisah sub pohon kiri dan kanan. Pohon biner ini biasanya digunakan untuk mempresentasikan suatu algoritma dalam menentukan penyelesaian.

10. informatika kelas 7 halaman 187 kurikulum merdeka​​


Dalam soal diatas adalah soal latihan tentang bagaimana cara menggunakan dan mengaplikasikan sticky notes dalam lembar kerja di Microsoft Excel.

Sticky notes pada Excel adalah fitur yang memungkinkan pengguna untuk menambahkan catatan atau komentar pada sel tertentu dalam lembar kerja. Sticky notes pada Excel dikenal juga dengan sebutan "comments" atau "notes".

Dengan menggunakan sticky notes, pengguna dapat menambahkan catatan atau penjelasan tambahan tentang suatu sel atau rangkaian sel tertentu, baik itu tentang rumus, instruksi penggunaan, atau catatan lain yang dianggap penting. Sticky notes pada Excel juga dapat membantu dalam berkolaborasi dengan pengguna lain dalam proyek yang sama, karena catatan atau komentar dapat dibagikan dan dilihat bersama.

Untuk menambahkan sticky notes pada Excel, pengguna dapat memilih sel tertentu dan klik kanan, lalu pilih "Add Note" atau "Insert Comment" dari menu yang muncul. Setelah itu, pengguna dapat mengetikkan catatan atau komentar pada sticky note yang muncul dan menyimpannya. Sticky notes akan muncul sebagai tanda kecil di sebelah kanan sel yang dipilih, dan dapat dilihat atau diedit dengan mengarahkan kursor ke sticky note tersebut.

Pembahasan:

Cara menggunakan sticky notes dalam lembar kerja seperti Excel.

Sticky notes (atau post-it) dapat digunakan sebagai catatan atau penanda sementara dalam lembar kerja Excel. Untuk menggunakan sticky notes, Anda dapat mengikuti langkah-langkah berikut:

Buka lembar kerja Excel dan pilih sel yang ingin Anda tambahkan sticky note-nya.Klik kanan pada sel tersebut dan pilih "Add Note" atau "Insert Comment" dari menu yang muncul.Ketikkan formula atau catatan yang ingin Anda tulis di bagian bawah sticky note.Klik di luar sticky note untuk menyimpan catatan tersebut. Sticky note akan muncul sebagai tanda kecil di sebelah kanan sel yang dipilih.Untuk menampilkan kembali catatan atau formula di sticky note, arahkan kursor ke sticky note tersebut dan tekan tombol kiri mouse.

Anda dapat mengulangi langkah-langkah tersebut untuk menambahkan sticky notes pada sel lain dalam lembar kerja Excel. Pastikan untuk menyimpan lembar kerja Anda secara teratur agar sticky notes tidak hilang saat Anda menutup program Excel.

= (1 * 2) + (1 * 2) + (1 * 2) + (1 * 2) + (1 * 2)

Lalu, karena ada 5 digit, kita buat 0 nya dimulai dari posisi terakhir yang ada tanda kurungnya:

1111 (1)

Kalau di urutkan dimulai dari 0 bukan 1. Jadi si yang terakhir ini pangkatnya nol, atau secara matematisnya begini:

= (1 * 2) + (1 * 2) + (1 * 2) + (1 * 2) + (1 * 2⁰)
Terus diikuti dari kanan 0 ke kiri "tambah satu" seperti ini:

= (1 * 2⁴) + (1 * 2³) + (1 * 2²) + (1 * 2¹) + (1 * 2⁰)

Hitung seperti biasa.

= (1 * 16) + (1 * 8) + (1 * 4) + (1 * 2) + (1 * 1)

= 16 + 8 + 4 + 2 + 1

= 31

Jadi hasil bilangan desimal dari bilangan biner 11111 tadi yaitu 31.

40. rangkuman informatika kelas 7 tentang program komputer​


Jawaban:

A. Pengenalan Antarmuka Pengguna

Teknologi saat ini mampu menangani interaksi perangkat dengan pengguna melalui tombol nyata, papan kunci (keyboard), sentuhan ke antarmuka grafis yang biasa disebut Graphical User Interface (GUI), maupun suara atau Voice User Interface (VUI). Untuk perangkat komputer, ponsel/smartphone, perangkat game, pemutar MP3, dan sebagainya, interaksi yang digunakan umumnya adalah GUI.

GUI adalah bentuk antarmuka pengguna dengan menggunakan ikon dan elemen grafis sebagai pengganti perintah berbasis teks yang ditikkan.

GUI menjadi populer karena lebih intuitif, mudah digunakan dan dipelajari daripada model interaksi Command Line Interface (CLI) yang memerlukan perintah yang harus ditik. Dengan CLI, jika pengguna tidak tahu perintahnya atau salah ketik, perintah tidak dapat dimengerti oleh komputer. Dengan GUI, pengguna tinggal memilih perintah yang disediakan.

B. Folder dan File

Folder/directory adalah tempat penyimpanan elektronis yang menyimpan file elektronis atau folder lain. Dalam dunia nyata, folder bisa dianalogikan sebagai map yang digunakan untuk menyimpan dokumen (surat, invoice, nota, dan lainnya), atau berisi map yang lebih kecil. Folder/directory dikelola oleh program komputer yang disebut sebagai File Manager yang merupakan bagian dari sistem operasi.

File elektronis sendiri berarti file yang diciptakan oleh komputer/ponsel untuk menyimpan Data secara deskret pada alat penyimpan elektronis. File adalah unit penyimpanan di komputer, yang akan dibaca atau ditulis. File diciptakan oleh program/aplikasi tertentu, dan file dapat dibaca oleh program tertentu pula.
