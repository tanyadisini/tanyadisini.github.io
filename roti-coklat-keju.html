6. Toko roti bu indah menyediakan roti pisang, roti coklat, dan roti keju. harga 2 roti pisang, 1roti coklat, dan 1 roti keju Rp15.300,00. harga 3 roti pisang, 2 roti coklat, dan 1 roti keju Rp22.800,00. harga 1 roti pisang, 3 roti coklat, dan 2 roti keju Rp22.100,00. Jika nia membeli 1 roti pisang dan 5 roti coklat, ia harus membayar sebesar? Pake jalannya yaa kak.


Kelas         : 10
Mapel        : Matematika
Kategori    : Bab 3  Persamaan Linear Tiga Variabel
Kata kunci : SPLTV, soal cerita, roti toko Bu Indah

Kode : 10.2. 3 [Kelas 8 Matematika Bab Persamaan Linear Tiga Variabel]

Penjelasan : 

Metode penyelesaian SPLTV

1.  Metode subtitusi yaitu menyatakan suatu variabel dalam variabel lainnya yang selanjutnya digunakan untuk mengganti variabel yg sama dalam persamaan lainnya.
2.  Metode eliminasi yaitu mengeliminasi atau menghilangkan salah satu variabel untuk mendapatkan nilai variabel lainnya.
3.  Metode gabungan eliminasi dan subtitusi yaitu menentukan nilai salah satu variabel dengan metode eliminasi, selanjutnya nilai variabelnya.
-----------------------------------------------------

Soal : 

Toko roti bu indah menyediakan roti pisang, roti coklat, dan roti keju. harga 2 roti pisang, 1roti coklat, dan 1 roti keju Rp15.300,00. harga 3 roti pisang, 2 roti coklat, dan 1 roti keju Rp22.800,00. harga 1 roti pisang, 3 roti coklat, dan 2 roti keju Rp22.100,00. Jika nia membeli 1 roti pisang dan 5 roti coklat, ia harus membayar sebesar?
a. Rp7.500,00
b. Rp11.500,00
c. Rp15.000,00
d. Rp21.500,00
e. Rp23.500,00

Pembahasan : 

misalkan
roti pisang = p
roti coklat = c
roti keju = k

Penulisan persamaan dari pernyataan soal

2p + c + k = 15.300  ... pers I
3p + 2c + k = 22.800  .... pers II
p + 3c + 2k = 22.100  .... pers III

eliminasikan k dari persamaan I dan II

2p +   c + k = 15.300
3p + 2c + k = 22.800
--------------------------- -
-p - c           = -7500     (dikali -1 kesemua ruas)
p + c = 7500    .... pers IV

eliminasikan k dari persamaan I dan III

2p +   c + k = 15.300  |×2|    4p + 2c + 2k = 30.600
p + 3c + 2k = 22.100  |×1|      p + 3c + 2k = 22.100
                                            ------------------------------ -
                                             3p - c           = 8500    .... pers V

eliminasikan c dari persamaan IV dan V

p  + c = 7500 
3p - c = 8500
------------------ + 
4p      = 16.000
       p = 16.000 / 4
       p = 4.000

subtitusikan p = 4.000 ke dalam pers IV

p  + c = 7500
4.000 + c = 7.500
             c = 7.500 - 4.000
             c = 3.500

subtitusikan p = 4.000 dan c = 3.500 ke dalam pers I

2p + c + k = 15.300
2 (4.000) + 3.500 + k = 15.300
8.000 + 3.500 + k = 15.300
11.500 + k = 15.300
               k = 15.300 - 11.500
               k = 3.800

Harga : 1 roti pisang = Rp 4.000
             1 roti coklat = Rp 3.500
             1 roti keju = Rp 3.800

Nia membeli 1 roti pisang dan 5 roti coklat
p + 5c = 4.000 + 5 (3.500)
           = 4.000 + 17.500
           = 21.500

Jadi Nia harus membayar 1 roti pisang dan 5 roti coklat adalah Rp 21.500

Pembahasan

Sistem persamaan linear dua variabel adalah sistem persamaan yang terdiri atas dua atau lebih persamaan linear dua variabel. Bentuk umum sistem persamaan linear dua variabel sebagai berikut.

ax + by = p

cx + dy = q

Penyelesaian dari sistem persamaan dua variabel adalah pasangan x dan y yang memenuhi kedua persamaan tersebut. Dalam mencari penyelesaian persamaan dua variabel dapat digunakan cara substitusi, eliminasi, atau campuran (eliminasi dan subtitusi).

Penyelesaian

diket:

misal roti coklat = x

         roti keju = y, maka

2x + 3y = Rp23.000,00  ... persamaan (1)

x + 2y = Rp14.000,00     ... persamaan (2)

ditanya:

harga 1 roti coklat dan 1 roti keju....?

jawab:

- menentukan nilai x dan y lebih dulu

 Eliminasi persamaan (1) dan (2)

 2x + 3y = Rp23.000,00    | × 1

  x + 2y = Rp14.000,00      | × 2

  sehingga

  2x + 3y = Rp23.000,00

  2x + 4y = Rp28.000,00

  __________________ -

         -y = -Rp5.000,00

          y = Rp5.000,00

 Substitusi y = Rp5.000,00 ke persamaan (2)

 x + 2y = Rp14.000,00

 x + 2(Rp5.000,00) = Rp14.000,00

 x + Rp10.000,00 = Rp14.000,00

 x = Rp14.000,00 - Rp10.000,00

 x = Rp4.000,00

- menentukan harga 1 roti coklat dan 1 roti keju

 x + y = Rp4.000,00 + Rp5.000,00

          = Rp9.000,00

Kesimpulan

Jadi, jumlah uang yang harus dikeluarkan Maher adalah Rp9.000,00.

40. si membeli 3 roti coklat dan 5 roti keju dengan harga RP39.000,00, sedangkan barkah juga membeli 1 roti coklat dan 1 roti keju dengan harga RP11.000,00, jika santi ingin membeli 4 roti coklat dan 2 roti keju, berapa santi harus membayar roti tersebut​


Jawab:

Harga yg harus dibayar santi adalah Rp 38.000,00

Penjelasan dengan langkah-langkah:

Misalkan roti coklat = C dan roti keju = K, maka:

3C + 5K = 39.000 .... (1)

1C + 1K = 11.000 .... (2)

-Pakai metode eliminasi: (1) dikalikan 1 dan (2) dikalikan 3

-Hasil perkaliannya dikurangkan (amati lampiran)

-Hasil akhirnya ditemukan nilai K = 3.000

Masukkan nilai K dalam perhitungan (2):

1C + 1K = 11.000

1C + 3.000 = 11.000

1C = 11.000 - 3.000

C = 8.000

Harga yg harus dibayar santi = 4C + 2K

Harga yg harus dibayar santi = (4 x 8.000) + (2 x 3.000)

Harga yg harus dibayar santi = 32.000 + 6.000

Harga yg harus dibayar santi = 38.000

Jadi harga yg harus dibayar santi adalah Rp 38.000,00
