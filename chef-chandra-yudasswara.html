Jawaban:

Passive voice

Tukar tempat objek dan subjeknya

Karena passive voice pola nya o+was/were+by+v+s

Kalimat nya past simple

Jawaban:

﻿Passive Voice:

﻿The cakes were baked by the chef.

Penjelasan:

﻿Jawaban dicetak tebal dan merupakan materi pembelajaran Passive Voice dari Siimple Past Tense karena mejelaskan aktivitas kegiatan di masa lampau dalam Bahasa Inggris.

﻿Kalimat aktif subjeknya aktif melakukan aksi dalam kalimat.

﻿Kalimat pasif subjeknya pasif dikenai aksi dalam kalimat.

﻿Simple Past Tense:

Active Voice: ﻿(+) Subject + Verb-2 + Object

﻿Passive Voice: (+) Subject + Tobe (was/were) + Verb-3 + Object

37. Uang Andri 2/3 uang Chandra dan uang Chandra 4/5 uang Burhan. Jika jumlah uang Burhan dan Chandra Rp540.000,00 maka selisih uang Andri dan Chandra adalah …​


A = Andri

B = Burhan

C = Chandra

A = 2/3 C

C = 4/5 B --> B = 5/4 C

B + C = 540.000

5/4 C + C = 540.000

9/4 C = 540.000

9 C = 2.160.000

C = 240.000 (uang Chandra)

A = 2/3 C = 2/3 x 240.000 = 160.000 (uang Andri)

Selisih uang Andri dan Chandra = 240.000 - 160.000 = 80.000
