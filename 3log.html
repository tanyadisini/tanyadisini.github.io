24. 3log 16- 3log 4- (3log 18 + 3log 6)=.....


Logaritma merupakan kebalikan dari perpangkatan

ᵃlog b = n dalam bentuk perpangkatan → aⁿ = b

Sifat" logaritma

1) ᵃlog (b.c) = ᵃlog b + ᵃlog c

2) ᵃlog b x ᵇlog c = ᵃlog c

3) ᵃlog (ᵇ/ₓ) = ᵃlog b - ᵃlog x

4) ᵃlog b = 1/ᵇlog a

5) ᵃlog (b/c) = - ᵃlog (c/b)

6) ᵃlog bⁿ = n. ᵃlog b

7) ᵃ²log b = ¹/₂. ᵃlog b

8) ᵃlog aⁿ = n

9) ᵃlog a = 1

Pembahasan

³log 16 - ³log 4 - (³log 18 + ³log 6)

= ³log (¹⁶/₄) - (³log(18 x 6)

= ³log 4 - ³log 108

= ³log (⁴/₁₀₈)

= ³log (¹/₂₇)

= ³log 3⁻³

= -3 ³log 3

= -3 . 1

= -3
