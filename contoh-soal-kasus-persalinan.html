13. Salin dan lengkapilah soal soal berikut! a. 43° = ...


Salin dan lengkapi soal-soal berikut

a. 43° = 2.580'

b. 8° = 28.800"

c. 3/4° =45'

d. 8,3° = 498'

e. 72° 86' 36" = 73° 16' 36"

f. 15° 56' 75" = 15° 57' 15"

g. 9° 36' = 9,6°

h. 40,6° =  40° 36'

i. 36° 65' = 37° 5'

j. 17' 76" = 18' 16"

Penjelasan dengan langkah-langkah:

Untuk menyelesaikan soal di atas kita harus mengetahui 1 derajat berapa menit dan 1 menit berapa detik.

1° = 60 menit = 60'

1° = 3.600 detik = 3.600"

1' = 60 detik = 60"

Soal lebih lengkap dari soal di atas dapat dilihat pada gambar terlampir.

a. 43° = 43 x 60'

43° = 2.580'

b. 8° = 8 x 3.600"

8° = 28.800"

c. 3/4° = 3/4 x 60'

3/4° = 45'

d. 8,3° = 8,3 x 60'

8,3° = 498'

e. 72° 86' 36" = 72° + 1° 16' 36"

72° 86' 36" = 73° 16' 36"

f. 15° 56' 75" = 15° 56' + 1' 15"

15° 56' 75" = 15° 57' 15"

g. 9° 36' = 9° + 36/60°

9° 36' = 9° + 3/5°

9° 36' = 9° + 0,6°

9° 36' = 9,6°

h. 40,6° = 40° + 0,6°

40,6° = 40° + 0,6 x 60'

40,6° = 40° + 36'

40,6° = 40° 36'

i. 36° 65' = 36° + 1° 5'

36° 65' = 37° 5'

j. 17' 76" = 17' + 1' 16"

17' 76" = 18' 16"
