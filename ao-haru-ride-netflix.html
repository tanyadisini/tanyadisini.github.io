6. apa arti aoi aoi ano sora


Arti dari "aoi aoi ano sora" adalah "birunya langit itu". ✔

PENJELASAN singkat

• 青い (aoi) = biru

• あの (ano) = kata tunjuk itu (jauh)

• 空 (sora) = langit.

Pembahasan.

Bahasa Jepang tidak bisa diartikan secara literatur. Oleh karena itu pengulangan kata "aoi" ini bisa dibilang makna dari langit.

Langit merupakan atap dari dunia, maka dari pengulangan aoi di sini bermakna "biru langit tidak hanya berada di satu tempat"

Kalimat ini juga bermakna kondisi langit di kota / tempat tersebut tengah Bagus (ii no tenki)

• 青い (aoi)

Aoi merupakan salah satu nama dalam warna. Aoi mengartikan "biru"

Perlu diketahui, aoi merupakan bentuk kata sifat berakhiran "-i" . Maknanya, jika kita mengucapkan aoi tanpa i pun diperbolehkan ✔

Aoi juga kerap kali dijadikan nama orang. AKAN TETAPI, ketika sudah diterapkan menjadi nama orang, kanji yang dipakai adalah "葵 (aoi)" ✔

Kalaupun ingin menggunakan kanji 青 (ao), seseringnya orang-orang tidak akan menambahkan "-i" untuk nama. Maka dari itu jika seseorang dipanggil Aoi, dia menggunakan kanji 葵 sebagai nama nya. Jika Ao, dia memakai kanji 青

-----------

• あの (ano)

Ano merupakan kata tunjuk benda mati maupjn hidup. Ano biasanya digunakan untuk menujuk benda yang jauh dan tidak dapat disentuh.

Perbedaan ano dan are;

• ano : itu. (Benda nya jauh untuk kita jangkau, dan tidak bisa kita pegang tetapi masih dalam satu ruangan. Atau bisa juga untuk perandaian)

• are: itu (bendanya cukup jauh untuk kita jangkau, tetapi masih berada cukup dekat dengan kita)

Contoh:

• あれはどなた？(Are wa donata?) : orang itu siapa?

• あのニュースは本当け? (Ano nyuusu wa honto ke?) : jadi berita itu benar, ya?

• 空 (sora)

Sora merupakan bentuk kata sifat berakhiran na (-na). Seharusnya kita membacanya sora yakni 空な. Tetapi karena kata sifat na akhirnya "na" nya tidak pernah tibaca, maka dari itu selalu dikurung.

Sora mengartikan langit bukan awan. Beberapa orang menganggap bahwa langit dan awan sama.

• 空 (sora) : langit

• 雲 (kumo) : awan.

Contoh kalimat:

• 空が晴れるだ。いい天気が良かった！ (sora ga hareru da. Ii tennki ga yokatta!) = langitnya cerah. Syukurlah, cuacanya Bagus.

• ヤバイ、黒の雲。雨が降ってかな (yabai, kuro no kumo. Ame ga futte ka na) : gawat, awannya mendung. Kayaknya akan turun hujan, kah...

24. Tulisan hiragana 'aoi'


Hiragana kata 'aoi' adalah あおい

Pembahasan

'Aoi' umumnya ditulis dengan bentuk kanji, dalam bentuk kanji ditulis 青い (aoi). Dalam bahasa Indonesia diartikan biru (yang merupakan nama dari salah satu warna), atau bisa berarti hijau kebiruan (artinya tergantung konteks kalimatnya).

Kata 青い (aoi) merupakan salah satu golongan kata sifat i-keiyoushi yang berarti yaitu kata sifat dengan akhiran -i.

Selain itu, kata 青い (aoi) bisa disingkat menjadi 青 (ao), artinya pun tetap sama.

Contoh penggunaan kata tersebut dalam kalimat :

Sora wa aoi desu : langitnya (berwarna) biru.Watashi wa aoi ga suki desu : aku suka warna biru.Umi wa aoi desu : lautnya (berwarna) biru.

__________

Definisi kanji 青

Kanji 青 -›

Arti ke bahasa Indonesia : biru.Kun'yomi (cara baca Jepang) : あお (ao). On'yomi (cara baca China) : セイ (sei), ショウ (shou).Level kanji : N4 JLPT.Memiliki sebanyak 8 guratan.

Kata 'Aoi' bila ditulis dalam Hiragana menjadi あおい (Aoi), yang artinya adalah warna "Biru". Namun tergantung dari Kanji nya, kata ini bisa juga berati warna "Hijau yang agak kebiruan".

あ = Aお = Oい = I

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pendahuluan :

Huruf Jepang dibagi menjadi Hiragana, Katakana, dan Kanji. Hiragana dan Katakana disebut juga Kana, merupakan huruf dasar yang masing-masing hurufnya hanya terdiri dari satu bunyi dan tidak memiliki arti. Sedangkan Kanji masing-masing hurfunya bisa memiliki lebih dari satu bunyi dan kebanyakan memiliki arti.

Kata sifat dalam Bahasa Jepang disebut dengan 形容詞 (Keiyoushi). Kata sifat Bahasa Jepang dibagi menjadi kata sifat-i dan kata sifat-na. Perbedaannya yaitu kata sifat-i menggunakan tambahan huruf i untuk menerangkan kata benda, sedangkan kata sifat-na menggunakan tambahan huruf na. Namun terdapat beberapa kata sifat yang menggunakan tambahan huruf i, tapi termasuk ke dalam kata sifat-na. Salah satu contohnya yaitu Kirai (Benci).

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

Huruf Hiragana dibagi menjadi beberapa kelompok yaitu Hiragana dasar (Seion), Hiragana Tenten (Dakuon), Hiragana Maru (Handakuon), Hiragana campuran (Youon), dan Hiragana khusus (Tokushuon). Yang saya sebutkan kali ini hanya Hiragana dasar saja yang terdiri dari 48 huruf, sebagai berikut.

あ い う え お (a - i - u - e - o)か き く け こ (ka - ki - ku - ke - ko)さ し す せ そ (sa - shi - su - se - so)た ち つ て と (ta - chi - tsu -te - to)な に ぬ ね の (na - ni - nu - ne - no)は ひ ふ へ ほ (ha - hi - fu - he - ho)ま み む め も (ma - mi - mu - me - mo)や ゆ よ (ya - yu - yo)ら り る れ ろ (ra - ri - ru - re - ro)わ ゐ ゑ を (wa - wi - we - wo)ん (n)

Untuk huruf ゐ (Wi) dan ゑ (We) sudah jarang digunakan di Jepang, dan sekarang penulisannya diubah menjadi うぃ (Wi) dan うぇ (We).

- - - - - - - - -

青い / あおい (Aoi) memiliki arti "Biru", dan kata ini termasuk ke dalam kelompok kata sifat-i. Kata sifat-i merupakan kelompok kata sifat yang terdapat tambahan huruf i di akhir kata nya. Contohnya seperti kata Aoi, yang awalnya hanya Ao kemudian ditambahkan huruf i. Menambahkan huruf i di akhir kata sifat-i ini fungsinya adalah untuk menerangkan kata benda nya. Contoh kalimatnya sebagai berikut.

青い鞄 (Aoi kaban) = Tas biru青い服 (Aoi fuku) = Baju biru青い空 (Aoi sora) = Langit biru

Beberapa kata sifat-i jika tidak terdapat tambahan huruf i nya, akan menjadi kata benda. Contohnya seperti kata Aoi ini. Aoi merupakan kata sifat, sedangkan Ao merupakan kata benda. Contoh kalimat yang menggunakan kata Ao sebagai berikut.

私の鞄は青 (Watashi no kaban wa ao) = Tas saya biru私の服は青 (Watashi no fuku wa ao) = Baju saya biru私の髪は青 (Watashi no kami wa ao) = Rambut saya biru

27. Ada yang bisa menjelaskan mengapa [tex]\frac{1}{(Ao-x)(Po+x)}=\frac{1}{Ao+Po} (\frac{1}{Ao-x}+\frac{1}{Po+x})[/tex] ?


Penjelasan

Diberikan persamaan:

[tex]\begin{aligned}\frac{1}{(A_0-x)(P_0+x)}=\frac{1}{A_0+P_0}\left(\frac{1}{A_0-x}+\frac{1}{P_0+x}\right)\end{aligned}[/tex]

Ruas kiri dapat dianggap sebagai perkalian dua pecahan. Perhatikan bahwa:

[tex]\begin{aligned}&\frac{1}{PQ}=\frac{1}{P}\cdot\frac{1}{Q}\end{aligned}[/tex]

Sedangkan pada ruas kanan, perkalian tersebut diubah menjadi penjumlahan antara [tex]1/P[/tex] dan [tex]1/Q[/tex]. Oleh karena itu, dibutuhkan faktor pengali [tex]K[/tex].
Kita cari nilai [tex]K[/tex] tersebut.

[tex]\begin{aligned}\frac{1}{PQ}&=K\left(\frac{1}{P}+\frac{1}{Q}\right)\\&\quad\rightarrow \textsf{samakan penyebut}\\\frac{1}{\cancel{PQ}}&=K\left(\frac{P+Q}{\cancel{PQ}}\right)\\1&=K(P+Q)\\\Rightarrow K&=\frac{1}{P+Q}\end{aligned}[/tex]

sehingga diperoleh:

[tex]\boxed{\vphantom{\Bigg|}\,\frac{1}{PQ}=\frac{1}{P+Q}\left(\frac{1}{P}+\frac{1}{Q}\right)\,}[/tex]

Dengan [tex]P=A_0-x[/tex] dan [tex]Q=P_0+x[/tex].

[tex]\begin{aligned}K&=\frac{1}{P+Q}\\&=\frac{1}{A_0-x+P_0+x}\\K&=\frac{1}{A_0+P_0}\end{aligned}[/tex]

Oleh karena itu:

[tex]\begin{aligned}\frac{1}{PQ}&=K\left(\frac{1}{P}+\frac{1}{Q}\right)\\\frac{1}{(A_0-x)(P_0+x)}&=\frac{1}{A_0+P_0}\left(\frac{1}{A_0-x}+\frac{1}{P_0+x}\right)\end{aligned}[/tex]

36. Jika AO = i + 2j, OB = 4i + 2j, dan sudut antara AO dan OB = ø maka tan ø ?


Jika AO = i + 2j, OB = 4i + 2j,

AO = i + 2j = (1, 2)
OB = 4i + 2j = (4, 2)

AO .OB = (1, 2) × (4, 2) = 4 + 4 = 8
|AO| = √(1)² + (2)² = √5
|OB| = √(4)² + (2)² = √20 = 2√5

Cos α = AO . OB / |AO| . |OB|
Cos α = 8 / (√5) × (2√5)
Cos α = 4/5

Tripel phytagoras 3, 4, 5

maka tan ø = 3/4
