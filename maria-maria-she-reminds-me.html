11. Maria menabung di bank sebesar Rp2.500.000 dengan suku bunga 16% pertahun . berapa lama maria menabung agar tabungan maria menjadi Rp 2.600.000​


Jawaban:

tabungan awal = Rp. 2.500.000,00.

dalam 1 tahun akan mendapat bunga sebesar :

= 16% x 2.500.000

= Rp. 400.000,00.

besar tabungan akhir = Rp. 2.600.000,00. sehingga bunga yang di dapat:

= 2.6000.000 - 2.500.000

= Rp. 100.000,00

lama waktu menabung:

[tex] = \frac{100.000}{400.000} \\ = \frac{1}{4} \: bulan[/tex]

jadi lama waktu Maria adalah 1/4 bulan.

v : 15,25 km/jam
t : 1,5 jam

ditanya:
s ( jarak )

jawab:
s = v / t
= 15,25 / 1 1/2
= 15,25 × 3 /2
= 45,75 / 2
= 22,875

jadi jarak yang sudah ditempuh Maria adalah 22,875 km

Jawaban:

Penjelasan dengan langkah-langkah:

Dik: Perbandingan Umur Maria dan

Debora adalah 3:4.

selisih umur Maria dan Debora

adalah 8 th

Dit : Umur Maria

Jawab:

Maria = M

Debora=D

M: D = 3:4

M = 3/4 × D

selisih M dan D:

D–M = 8

D–3/4D =8

¼D = 8

D = 32

umur Debora adalah 32 tahun

umur Maria:

D–M = 8

D–8 = M

32–8=M

24 = M

umur Maria adalah 24 tahun

34. To learn - did not - maria - know - was - but - eager - she


REARRANGE THESE JUMBLED WORDS INTO A GOOD SENTENCE!

To learn - did not - maria - know - was - but - eager - she

____________________

JAWABAN :

Maria did not know but she was eager to learn.

{ Maria tidak mengerti/tidak paham tetapi dia bersemangat untuk belajar }

PENJABARAN KATA-KATA :

Maria => Subyek kalimat

Did not => Auxiliary verb negative bentuk lampau ( tidak )

Know => Kata kerja ( tahu/paham)

But => Subordinate conjunction makna kontras ( tetapi )

She => pronoun / kata ganti orang sbg subyek klausa kedua

Was. => To be bentuk lampau

Eager => Adjective / kata sifat { bersemangat }

To learn => kt kerja to infinitive modifikasi tujuan /alasan{ untuk belajar }

________________
