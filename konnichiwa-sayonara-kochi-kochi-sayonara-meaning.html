2. Apa arti konnichiwa Arigatou gozaimasu Dan sayonara


Apa arti konnichiwa?

こんにちは (konnichiwa)

Arti dari salam ini adalah selamat siang atau sore. Namun, dapat digunakan untuk mengucapkan hai. Waktu yang diberikan untuk menggunakan salam ini adalah sekitar 11 pagi - 5 sore. Rata-rata beberapa orang mengucapkan konnichiwa diluar jangka waktu tersebut

Apa arti Arigatou gozaimasu?

ありがとうございます

[ Arigatou Gozaimasu ]

Arigatou Gozaimasu memiliki Arti yaitu Terimakasih , Biasanya untuk mengucapkan terimakasih anda bisa mengucapkan Arigatou Gozaimasu , kalimat ini termasuk dalam Ungkapan yang sopan.

Apa arti sayonara?

さよなら ( sayonara )

Artinya adalah sampai jumpa lagi. Digunakan untuk orang yang akan berpisah namun tidak tahu pasti waktunya.

Yamete Kudasai (やめて ください) = Tolong hentikan / berhenti

Yamete kudasai (やめて ください) dalam Bahasa jepang jika diterjemahkan kedalam bahasa Indonesia artinya "Tolong berhenti/Hentikan" ada frasa yang hampir sama dengan Yamete kudasai yaitu Yameru (やめる)

Apa Bedanya ?

Yamete kudasai(やめて ください) ialah kata kerja (verb) bentuk -te (て) , sedangkan jika Yameru (やめる)

Menggunakan kata dasar yaitu -ru yang artinya berhenti. Ketika kata kerja ini diubah ke bentuk -te, maka kata kerja bisa kita hubungkan dengan kata yameru dengan kudasai (ください)

Contoh Kalimat yang menggunakan kata "Kudasai" (ください)

9. Kochi asobo yo.Artinya??​


こっち遊ぼうよ (Kocchi asobou yo) artinya adalah "Ayo main ke sini dong!"

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pendahuluan :

Shiji-shi「指示詞・しじし」

Merupakan istilah umum untuk "Kata tunjuk" dalam Bahasa Jepang. Kata tunjuk ini bisa digunakan untuk menunjukkan berbagai macam hal, seperti benda, orang, tempat, dan lain sebagainya. Kata tunjuk dalam Bahasa Jepang dibagi menjadi empat berdasarkan awalan bunyinya, yaitu :

Kata tunjuk berawalan bunyi KoKata tunjuk berawalan bunyi SoKata tunjuk berawalan bunyi AKata tunjuk berawalan bunyi Do

Kata tunjuk berawalan bunyi Ko, digunakan untuk menunjukkan sesuatu yang dekat dengan pembicara. Kata tunjuk berawalan bunyi So, digunakan untuk menunjukkan sesuatu yang dekat dari lawan bicara. Kata tunjuk berawalan bunyi A, digunakan untuk menunjukkan sesuatu yang jauh dari pembicara dan lawan bicara. Dan kata tunjuk berawalan bunyi Do, digunakan untuk menanyakan sesuatu yang ditunjuk.

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

Kata Tunjuk Arah

Untuk yang formal bisa menggunakan Kochira, Sochira, Achira, dan Dochira. Sedangkan untuk yang informal bisa menggunakan Kocchi, Socchi, Acchi, dan Docchi. Kali ini saya hanya membahas yang informal saja, sebagai berikut.

- - - - - - -

Kocchi「こっち」= Dapat diartikan dengan "(Ke) sini". Kata ini digunakan untuk menunjukkan arah yang mendekat ke pembicara.

Socchi「そっち」= Dapat diartikan dengan "(Ke) situ". Kata ini digunakan untuk menunjukkan arah yang mendekat ke lawan bicara.

Acchi「あっち」= Dapat diartikan dengan "(Ke) sana". Kata ini digunakan untuk menunjukkan arah yang menjauh dari pembicara dan lawan bicara.

Docchi「どっち」= Dapat diartikan dengan "(Ke) mana" atau "Yang mana". Kata ini bisa digunakan untuk menanyakan arah mana yang dituju, dan bisa juga digunakan untuk menanyakan benda apa yang ditunjuk. Kata ini digunakan untuk menanyakan salah satu dari dua arah atau benda yang dituju atau ditunjuk.

- - - - - - -

Asobou「遊ぼう・あそぼう」

Kata ini berarti "Ayo main" atau "Main yok". Kata ini merupakan kata kerja bentuk ajakan, yang digunakan di situasi informal saja. Penggunaannya bisa ke teman, saudara, dan orang-orang lainnya yang dekat dan akrab. Bentuk awal dari kata ini adalah Asobu, yang artinya bermain.

Yo「よ」

Kata ini tidak memiliki arti khusus, tapi di sini saya mengartikannya dengan akhiran "...dong". Kata ini merupakan partikel atau kata bantu kalimat, yang diletakkan di akhir kalimat, dengan tujuan untuk menegaskan isi kalimatnya. Dalam beberapa situasi, partikel ini juga bisa diartikan dengan akhiran "...kok" dan "...loh".

22. "Sayonara".artinya adalah?


さよなら ( sayonara) artinya sampai jumpa.

Pembahasan

さよなら sering digunakan masyarakat Jepang untuk mengucapkan perpisahan dengan seseorang. Ada juga kalimat yang mirip dengan itu yakni また明日 ( mata ashita)

—

さよなら ( sayonara )

Artinya adalah sampai jumpa lagi. Digunakan untuk orang yang akan berpisah namun tidak tahu pasti waktunya.

=> なら ( nara) menunjukkan bahwa kalimat ini sebuah pengandaian.

—

また明日 ( mata ashita )

Artinya adalah sampai jumpa besok. Digunakan untuk orang berpisah kurang dari satu hari.

=> また ( mata) artinya lagi atau juga.

=> あした ( ashita) artinya besok.

Adalah sebutan untuk daerah di indonesia yang dijadikan sebagai kochi, kochi sendiri bersasal dari bahasa jepang yang memliki arti tak lain adalah daerah yang istimewa, sehingga daerah yan terpilih adalah daerah yang istimewa. Daerah yang menjadi kochi tak lain adalah Yogyakarta dan Surakarta.

27. apa arti sayonara? :v ​


Bahasa Indonesia dari さようなら (Sayonara) adalah Selamat tinggal.

▃▃▃▃▃▃▃▃▃▃▃▃▃▃

Pembahasan

▃▃▃▃▃▃▃▃▃▃▃▃▃▃

Jika seseorang akan berpisah maka salah satu orang tersebut mengucapkan さようなら atau Sayonara yang dalam bahasa Indonesia artinya adalah sebagai Selamat tinggal, maka dari itu frasa ini termasuk salam perpisahan. Selain itu adapun じゃまた yang artinya 'dah'. Berikut adalah penjelasannya :

さようなら atau cara membacanya adalah 'Sayonara', yang dalam bahasa Indonesia artinya adalah selamat tinggal, jika mengucapkan さようなら (Sayonara) maka orang tersebut akan pergi dengan lama. Maka dari itu frasa ini termasuk salam perpisahan.

▃▃▃▃▃▃▃▃▃▃▃▃▃▃▃▃▃▃▃▃▃▃▃▃

じゃまた atau cara membacanya adalah 'Ja mata', yang dalam bahasa Indonesia artinya adalah sebagai 'dah'. Kita sering mendengar kata 'dah' dalam bahasa Indonesia ketika seseorang akan berpisah, dan kata 'dah' tersebut yang dimaksud. Jika seseorang mengucapkan じゃまた (Ja mata), biasanya sambil melambaikan tangan.

Selamat Datang :

Irasshaimase;

Irasshaimase Memiliki arti yaitu ‘Selamat Datang’ , Irasshaimase merupakan Sapaan yang Paling tepat Untuk Tamu Yang berkunjung ke tempat anda sendiri ( Contoh : Rumah Sendiri ) , Irasshaimase biasanya diucapkan oleh Pelayan Restoran kepada Tamu , Dan secara langsung akan memengaruhi tamu yang berkunjung , Karena kalimat ini dianggap sebagai Kalimat Yang sangat Ramah

Youkoso;

Youkoso memiliki arti "Selamat datang" , Youkoso merupakan Sapaan yang paling umum diucapkan ketika acara resmi. Contoh penggunaan Youkoso yang paling sering diucapkan;

ようこそへマナド ( Youkoso e Manado ); Selamat datang di Manado

__________________

Selamat Tinggal :

Sayounara Merupakan salam Perpisahan , Sayonara Memiliki arti yaitu 'Selamat Tinggal' , Biasanya digunakan ketika Seseorang akan Pergi dalam Jangka waktu yang lama

Mata A shi ta :

Mata A shi ta , Memiliki arti yaitu 'Selamat Jumpa lagi Besok' , Sedangkan A shi ta artinya yaitu Besok. Tetapi sebenarnya Penggunaan Sayounara dalam bahasa Jepang Jumlah nya terbatas ! tidak sama dengan Sampai Jumpa dalam bahasa Indonesia

Selain Itu Ucapan Selamat ( Salam ) Bukan hanya Sayounara saja , Contoh nya ada : Selamat Pagi , Selamat Malam , Selamat Tidur , Selamat Siang. , Dan Lain lain
