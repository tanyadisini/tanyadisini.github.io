3. apa itu mikrotik dan cisco


Mikrotik adalah sistem operasi dan perangkat lunak yang dapat digunakan untuk menjadikan komputer manjadi router network yang handal, mencakup berbagai fitur yang dibuat untuk IP network dan jaringan wireless, cocok digunakan oleh ISP, provider hotspot dan warnet.

sedangkan  Cisco adalah peralatan utama yang banyak digunakan pada Jaringan Area Luas atau Wide Area Network (WAN). Dengan cisco router, informasi dapat diteruskan ke alamat-alamat yang berjauhan dan berada di jaringan computer yang berlainan. Yang bertujuan untuk dapat meneruskan paket data dari suatu LAN ke LAN lainnya, Cisco router menggunakan tabel dan protocol routing yang berfungsi untuk mengatur lalu lintas data. Paket data yang tiba di router diperiksa dan diteruskan ke alamat yang dituju. Agar paket data yang diterima dapat sampai ke tujuannya dengan cepat, router harus memproses data tersebut dengan sangat tepat. Untuk itu, Cisco Router menggunakan Central Processing Unit (CPU) seperti yang digunakan di dalam komputer untuk memproses lalu lintas data tersebut dengan cepat. Seperti komputer, cisco router juga mempunyai sejumlah jenis memori yaitu ROM, RAM, NVRAM dan FLASH, yang berguna untuk membantu kerjanya CPU. Selain itu dilengkapi pula dengan sejumlah interface untuk berhubungan dengan dunia luar dan keluar masuk data. Sistem operasi yang digunakan oleh cisco router adalah Internetwork Operating System (IOS).

11. sebutkan hierarki konfigurasi pada cisco IPS? ​


Jawaban:

Hierarki konfigurasi pada Cisco IPS (Intrusion Prevention System) melibatkan beberapa level atau lapisan yang harus dikonfigurasi secara berurutan untuk menjalankan perangkat IPS dengan benar. Berikut hierarki konfigurasi pada Cisco IPS:

1. Sistem (System Configuration): Ini adalah tingkat konfigurasi teratas di mana Anda mengatur parameter sistem umum, seperti informasi identitas perangkat IPS, koneksi jaringan, pengaturan saat boot, dan opsi manajemen jarak jauh.

2. Sumber (Interface Configuration): Pada tingkat ini, Anda mengkonfigurasi antarmuka fisik pada perangkat IPS, seperti antarmuka Ethernet atau VLAN yang terhubung ke jaringan. Anda juga dapat mengkonfigurasi parameter jaringan, seperti alamat IP dan subnet mask.

3. Zona (Zone Configuration): Konfigurasi zona melibatkan mengelompokkan antarmuka fisik ke dalam zona yang berbeda. Ini membantu dalam mengatur kebijakan akses antara zona yang berbeda dalam jaringan.

4. Virtual Sensor (VS Configuration): Jika Anda memiliki beberapa sensor virtual di perangkat IPS, Anda dapat mengkonfigurasi setiap sensor virtual secara terpisah di tingkat ini. Setiap sensor virtual dapat memiliki kebijakan dan konfigurasi yang berbeda.

5. Kebijakan (Policy Configuration): Di tingkat ini, Anda mengatur kebijakan pemantauan dan penanggulangan serangan. Anda dapat menentukan tindakan yang diambil ketika serangan terdeteksi, seperti memblokir lalu lintas, mengirimkan peringatan, atau melakukan tindakan lain sesuai dengan kebijakan Anda.

6. Pengaturan Tanda Tangan (Signature Configuration): Ini melibatkan konfigurasi tanda tangan atau pola serangan yang akan diawasi oleh perangkat IPS. Anda dapat memilih tanda tangan yang ingin Anda aktifkan dan mengkonfigurasi parameter terkait.

7. Pengaturan Logging dan Pelaporan (Logging and Reporting Configuration): Di tingkat ini, Anda mengatur bagaimana data log dan laporan serangan akan dikelola, termasuk pengiriman log ke server SIEM (Security Information and Event Management) atau alat lainnya.

23. Apa itu vlan? Dan bagaimana penerapannya dalam cisco packet treser?


Virtual LAN atau disingkat VLAN merupakan sekelompok perangkat pada satu LAN atau lebih yang dikonfigurasikan (menggunakan perangkat lunak pengelolaan) sehingga dapat berkomunikasi seperti halnya bila perangkat tersebut terhubung ke jalur yang sama, padahal sebenarnya perangkat tersebut berada pada sejumlah segmen LAN yang berbeda. Vlan dibuat dengan menggunakan jaringan pihak ke tiga. VLAN merupakan sebuah bagian kecil jaringan IP yang terpisah secara logik. VLAN memungkinkan beberapa jaringan IP dan jaringan-jaringan kecil (subnet) berada dalam jaringan switched switched yang sama. Agar computer bisa berkomunikasi pada VLAN yang sama, setiap computer harus memiliki sebuah alamat IP dan Subnet Mask yang sesuai dengan VLAN tersebut. Switch harus dikonfigurasi dengan VLAN dan setiap port dalam VLAN harus didaftarkan ke VLAN. Sebuah port switch yang telah dikonfigurasi dengan sebuah VLAN tunggal disebut sebagai access port.Sebuah VLAN memungkinkan seorang Administrator untuk menciptakan sekelompok peralatan yang secara logic dihubungkan satu sama lain. Dengan VLAN, kita dapat membagi jaringan switch secara logik berdasarkan fungsi, departemen atau project team .

Jawaban:

ada 6 yaitu:

Setup Mode. User Mode. Privileged Mode. Global Configuration Mode. Interface Mode. Rommon Mode.

Penjelasan:

Dari semua mode diatas, berikut adalah penjelasan singkatnya :

1. Setup Mode

Setup mode ini terjadi saat NVRAM pada Router Cisco masih kosong atau saat tidak ada konfigurasi. Biasanya terjadi ketika kita mengaktifkan Router baru atau pada saat melakukan reset konfigurasi.

2. User Mode

Hanya terdapat beberapa command untuk monitoring

Command show terbatas, ping dan traceroute

Ditandai dengan tanda > setelah nama device

3. Privileged Mode.

Terdapat beberapa command monitoring dan troubleshooting

Terdapat semua command show, ping, trace, copy, erase

Ditandai dengan # setelah nama device

4. Global Configuration Mode.

Untuk mensetting keseluruhan router misalnya hostname, konfigurasi routing

Semua konfigurasi yang kita inputkan berefek global di router

Ditandai dengan (config)# setelah nama device

5. Interface Mode.

Untuk konfigurasi interface secara spesifik, misal Interface fa0/0, Interface Fa0/1

Ditandai dengan : Router(config-if)#

6. Rommon Mode.

Untuk recovery password

Jika lupa password console dan telnet, atau lupa password enable maka gunakan rommon mode untuk melakukan recovery password dengan mengubah nilai confreg

Setelah memahami Macam-macam Mode dalam Router Cisco, selanjutnya bisa belajar mengenai 23 Command Dasar Cisco Router Yang Wajib Diketahui dan Dipahami untuk Pemula. Tulisan ini sengaja dibuat berurutan supaya dapat dipelajari secara berurutan supaya mudah mempelajarinya.
