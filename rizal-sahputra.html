14. Rizal menabung di bank sebesar Rp2.100.000,00 dengan suku bunga tunggal sebesar 8% pertahun. Saat diambil, tabungan Rizal menjadi Rp2.282.000,00. Lama Rizal menabung adalah ....​


tabungan awal = Rp2.100.000

tabungan akhir = Rp2.282.000

p = 8%

tabungan akhir = tabungan awal + n/12 x p x tabungan awal

2.282.000 = 2.100.000 + n/12 x 8/100 x 2.100.000

2.282.000 - 2.100.000 = n/12 x 8 x 21.000

182.000 = n/3 x 2 x 21.000

182.000 = n/3 x 42.000

182.000 x 3 / 42.000 = n

13 = n

Jadi, lama Rizal menabung adalah 13 bulan

#MathIsBeautiful
