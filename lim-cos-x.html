14. lim x→0 cos 3x • cos x / x²​


Untuk menghitung limit tersebut, kita perlu menggunakan aturan identitas trigonometri dan aturan limit:

Aturan identitas trigonometri:

cos A • cos B = (1/2) • (cos(A + B) + cos(A - B))

Aturan limit:

lim x→0 sin x / x = 1 (limit trigonometri umum)

Mari kita aplikasikan aturan identitas trigonometri pada ekspresi cos 3x • cos x:

cos 3x • cos x = (1/2) • (cos(3x + x) + cos(3x - x))

cos 3x • cos x = (1/2) • (cos(4x) + cos(2x))

Selanjutnya, kita bagi dengan x^2 dan hitung limit saat x mendekati 0:

lim x→0 (cos 3x • cos x / x^2) = lim x→0 [(1/2) • (cos(4x) + cos(2x)) / x^2]

Pertama-tama, kita aplikasikan aturan limit untuk menghilangkan x di penyebut:

lim x→0 (cos 3x • cos x / x^2) = (1/2) • lim x→0 [(cos(4x) + cos(2x)) / x^2]

Sekarang, kita aplikasikan aturan limit trigonometri umum:

lim x→0 (cos 3x • cos x / x^2) = (1/2) • (1 + 1) = (1/2) • 2 = 1

Jadi, hasil dari limit tersebut adalah 1.

a. Untuk menghitung nilai lim x→0 (1 - cos(4x))/(x² cos(6x) - 1), kita akan menggunakan aturan limit trigonometri:

Pertama, kita evaluasi masing-masing bagian lim:

- Bagian pembilang (1 - cos(4x)) saat x mendekati 0:

- Bagian penyebut (x² cos(6x) - 1) saat x mendekati 0:

- Bagian penyebut (lim x→0 (x² - x²) - 3x²) saat x mendekati 0:

38. lim cos x - cos 3x/1 - cos x x->0​


Hasil dari [tex]\displaystyle{ \lim_{x \to 0} \frac{cosx-cos3x}{1-cosx} }[/tex] adalah 8.

PEMBAHASAN

Teorema pada limit adalah sebagai berikut :

[tex](i)~\lim\limits_{x \to c} f(x)=f(c)[/tex]

[tex](ii)~\lim\limits_{x \to c} kf(x)=k\lim\limits_{x \to c} f(x)[/tex]
