6. Change the active to the passive by supplying the correct form of beExample: Tom opens the door : The door is opened by Tom1. Tom is opening the door : The door ... opened by Tom2. Tom has opened the door : The door ... opened by Tom3. Tom opened the door : The door ... opened by Tom4. Tom was opening the door : The door ... opened by Tom5. Tom had open the door : The door ... opened by Tom6. Tom will open the door : The door ... opened by Tom7. Tom is going to open the door : The door ... opened by Tom8. Tom will have opened the door : The door ... opened by Tom​


Bentuk passive nya adalah..

Answer :

1). The door is being opened by Tom. (Present continuous)

2). The door has been opened by Tom. (Present perfect)

3). The door was opened by Tom. (Simple past)

4). The door was being opened by Tom. (Past continuous)

5). The door had been opened by Tom. (Past perfect)

6). The door will be opened by Tom. (Simple future)

7). The door is going to be opened by Tom. (Simple future "be going to")

8). The door will have been opened by Tom. (Future perfect)

Penjelasan :

Dalam mengubah kalimat aktif ke pasif, maka perhatikan :

Tenses yang digunakanPronounSubject & Object yang digunakan

Berikut ini rumus yang digunakan dari kalimat di atas

Present continuous

Active : S + To be (am/are/is) + V-ing + O

Passive : S + To be (am/are/is) + being + V3 + by..

Present perfect

Active : S + Have/has + V3 + O

Passive : S + Have/has + been + V3 + by..

Simple past

Active : S + V2 + O

Passive : S + To be (was/were) + V3 + by..

Past continuous

Active : S + To be (was/were) + V-ing + O

Passive : S + To be (was/were) + being + V3 + by..

Past perfect

Active : S + Had + V3 + O

Passive : S + Had + been + V3 + by..

Simple future

Active : S + will + V1 + O

Passive : S + will + be + V3 + by .

Simple future "be going to"

Active : S + To be (am/are/is) + going to + V1 + O

Passive : S + To be (am/are/is) + going to + be + V3 + by..

Future perfect

Active : S + will + have + V3 + O

Passive : S + will + have + been + V3 + by..

Rumus Passive Voice:

-Pattern = Subject + have/has + been + Verb3

-Pattern = Subject + to be (was – were) + Verb 3

-Pattern = Subject + to be (was – were) + Being + Verb3

6. Passive Voice [Past Perfect Tense]

-Pattern = Subject + had + been + Verb3

7. Passive Voice [Simple Future Tense]

-Pattern = Subject + will + be + Verb3

8. Passive Voice [Future Continuous Tense]

-Pattern = Subject + will be + being + Verb3

9. Passive Voice [Future Perfect Tense]

-Pattern = Subject + will have + been + Verb3

39. ubah kalimat aktif menjadi pasif.1.tom has opening the door.2.tom opened the door.3.tom was opening the door.4.tom had open the door.5.tom will open the door.6.tom is going to open the door.ka tolong lah.​


Task

Ubah kalimat aktif menjadi pasif.

1.Tom has opened the door.

2. Tom opened the door.

3. Tom was opening the door.

4. Tom had opened the door.

5. Tom will open the door.

6. Tom is going to open the door.

---------

Answer

1. The door has been opened by Tom.

2. The door was opened by Tom.

3. The door was being opened by Tom.

4. The door had been opened by Tom.

5. The door will be opened by Tom.

6. The door is going to be opened by Tom.

---------

Explanation

Kalimat pasif adalah kalimat di mana subjeknya dikenai aksi yang didefinisikan oleh kata kerja dalam kalimat tersebut.

Konstruksi umum kalimat pasif adalah :

S + be + past participle

Di mana subjek dari kalimat pasif merupan objek dalam kalimat aktifnya.

Sekarang kita cari mana subject, verb, dan object kalimat

Subject: Tom → menjadi pelaku (by Tom)

Verb: open → berubah ke bentuk past participle opened

Object: the door → menjadi subject
