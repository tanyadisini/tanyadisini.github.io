10. Sin 31°=p hasil sin 76° + sin 14° =​


[tex] 2 \sqrt{1-p^2} [/tex]

DIKETAHUI

[tex] \sin 31^{\circ} = p [/tex]

DITANYA

[tex] \sin 76^{\circ} + \sin 14^{\circ} = [/tex] ?

JAWAB

Ingat kembali rumus berikut

[tex] \sin A + \sin B = 2 \sin \left( \frac{A + B}{2} \right) \cos \left( \frac{A - B}{2} \right) [/tex]

Menggunakan rumus tersebut, didapat

[tex] \sin 76^{\circ} + \sin 14^{\circ} = 2 \sin \left( \frac{76^{\circ} + 14^{\circ}}{2} \right) \cos \left( \frac{76^{\circ} - 14^{\circ}}{2} \right) \\ = 2 \sin \left( \frac{90^{\circ}}{2} \right) \cos \left( \frac{62^{\circ}}{2} \right) \\ = 2(1) \cos 31^{\circ} \\ = 2 \cos 31^{\circ} [/tex]

Lalu, gunakan identitas trigonometri berikut

[tex] \sin^2 x + \cos^2 x = 1 [/tex]

didapat

[tex]\cos^2 x = 1 - \sin^2 x \\ \Rightarrow \cos^2 31^{\circ} = 1 - \sin^2 31^{\circ} = 1 - p^2 \\ \Leftrightarrow \cos 31^{\circ} = \pm \sqrt{1-p^2} \\ \Leftrightarrow \cos 31^{\circ} = \sqrt{1-p^2} \lor \cos 31^{\circ} = -\sqrt{1-p^2}[/tex]

Oleh karena [tex] 31^{\circ} [/tex] berada di kuadran I, [tex] \cos 31^{\circ} [/tex] pasti bernilai positif. Dengan demikian, nilai [tex] \cos 31^{\circ} [/tex] yang memenuhi adalah [tex] \sqrt{1-p^2} [/tex] sehingga [tex] \sin 76^{\circ} + \sin 14^{\circ} = 2 \cos 31^{\circ} = 2 \sqrt{1-p^2} [/tex]

KESIMPULAN

Jadi, hasil dari [tex] \sin 76^{\circ} + \sin 14^{\circ} [/tex] adalah [tex] 2 \sqrt{1-p^2} [/tex]

14. jika 3 cos 2x-14 sin X+9=0, Maka sin X?


Jawab

jika 3cos2x-14sinx+9 = 0, Maka nilai sinx adalah 2/3

Pembahasan Ingat Kembali

ok saya akan menjelaskan beberapa materi matematika yang berkaitan dengan soal ini

-Substitusi

{adalah cara menggabungkan dua persamaan dalam matematika dengan mengganti variabel dalam persamaan 1 dengan persamaan 2

misal

// persamaan 1 //

2y = 30

// persamaan 2 //

y = 7-x

//y pada persamaan 1 bisa diganti dengan 7-x pada persamaan 2 //

    2y = 30

2(7-x) = 30}

-Pindah Ruas

{Pada saat bilangan pindah ruas bilangan positif menjadi negatif, bilangan negatif menjadi positif, di kali menjadi di bagi dan di bagi menjadi di kali, pangkat menjadi akar dan akar menjadi pangkat dll

khusus untuk pangkat suatu variabel yang berkelipatan 2, ketika pindah ruas ditambah tanda plus minus di depannya([tex]\pm[/tex])

misal :

x² = 9

x = ±√9

// sehingga nilai x adalah: //

x = -3 atau x = 3

NB: tanda plus minus tersebut karena -3 dan 3 jika dikuadratkan hasilnya sama sama 9 }

-Trigonometri(rumus identitas)

{untuk menyederhanakan soal trigonometri ini, dapat dilakukan dengan mensubstitusikan rumus rumus trigonometri yang ada, berikut rumus rumus tersebut :

[rumus berkebalikan]

sinα = 1/cosecα

cosα = 1/secα

tanα = 1/cotanα

[rumus perbandingan]

   tanα = sinα/cosα

cotanα = cosα/sinα

[rumus penjumlahan]

sin²α+cos²α = 1

       tan²α+1 = sec²α

   cotan²α+1 = cosec²α  }

-Trigonometri(Sudut Rangkap)

trigonometri sudut rangkap digunakan untuk mencari sudut sudut yang bukan istimewa, seperti 120° dengan mengubahnya menjadi 2(60°) sudut tersebut bisa diketahui

berikut rumus-rumus sudut rangkap trigonometri

sin2α = 2sinαcosα

cos2α = cos²α-sin²α

tan²α = 2tanα/(1-tan²α)  // untuk yang ini tidak berlaku jika α = 90° //

Penyelesaian

// dari soal dapat kita simpulkan //

soal terdiri dari cos dan sinyang ditanya adalah nilai sinx, sehingga untuk mencari sinx kita ubah cos2x menjadi sinx dengan mensubstitusikan rumus sudut rangkap dan rumus penjumlahan

// jabarkan rumus penjumlahan sin dan cos agar mudah digunakan //

sin²x+cos²x = 1

\\ pindah ruas \\

         cos²x = 1-sin²x

// substitusikan cos²x //

cos2x = cos²x-sin²x

          = (1-sin²x)-sin²x

          = 1-2sin²x

// kembali ke soal //

   3cos2x-14sinx+9 = 0

\\ substitusikan cos2x \\

3(1-2sin²x)-14sinx+9 = 0

  3-6sin²x-14sinx+9 = 0

   -6sin²x-14sinx+12 = 0

     3(sinx)²+7sinx-6 = 0

\\ kita substitusikan sinx dengan y, dimana y = sinx \\

3y²+7y-6 = 0

\\ faktorkan, dan cari nilai y \\

  3y²+9y-2y-6 = 0

3y(y+3)-2(y+3) = 0

     (y+3)(3y-2) = 0

\\ sehingga nilai y adalah : \\

y = -3

y = 2/3

\\ substitusikan y dengan sinx kembali, untuk menentukan nilai sinx \\

sinx = -3

sinx = 2/3

\\ karena nilai sin harus di antara 0 dan 1 atau sama dengan 0 atau 1, sehingga sinx = -3 tidak memenuhi(ini karena tidak ada nilai x yang membuat nilainya -3) \\

\\ maka yang memenuhi hanyalah sinx = 2/3 \\

Jadi nilai sinx adalah 2/3
