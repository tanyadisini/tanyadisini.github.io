10. Materi : Dimensi TigaJarak titik F ke bidang PDR adalah...


maaf terlalu panjang tapi ini caranya..PH = PR = 12/2 = 6 cm
PR = 6√2 cm
PD = √6²+12² = √36+144 = √180 = 6√5 cm
tinggi PRD = √(6√5)²-(6√2/2)² = √180-18 = √162 = 9√2 cm
FD = 12√3 cm

F ke bidang PRD = √(12√3)² - (9√2)² = √432-162 = √270 = 9√30 cm
