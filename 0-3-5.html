31. (i) 2×² -5=0 (ii) 2×² +3×³=0 (iii) 3× +6=0 (iv) 3×² +5 × +9=0 Yang merupakan perasamaan kuadrat adalah?


-Persamaan Kuadrat-

Bentuk umum: a² + bx + c = 0

Variabel paling tinggi adalah pangkat 2 dengan c sebagai konstanta.

(I) 2x² - 5 = 0 (Persamaan Kuadrat, meskipun tidak ada konstanta)

(II) 2x² + 3x³ = 0 (Bukan, pangkat melebihi angka maksimum)

(III) 3x + 6 = 0 (Bukan, tidak memiliki Kuadrat)

(IV) 3x² + 5x + 9=0 (Persamaan Kuadrat sempurna)

Maka, (I) dan (IV)
