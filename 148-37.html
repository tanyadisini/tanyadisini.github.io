4. A.106°B.37°C.148°D74°​


Jawaban:

A. 106°

Penjelasan dengan langkah-langkah:

<P4 = 74° (Sama dibawahnya)

<P2 = <P4 = 74° (Bertolak belakang)

<P1 = <P3 (Bertolak belakang) -> dimisalkan x

Total <P1 + <P2 + <P3 + <P4 itu adalah 360°

360° = <P1 + <P2 + <P3 + <P4

360° = x + 74° + x + 74°

360° = 148 + 2x

2x = 360° - 148°

2x = 212

x = 106 (<P1 dan <P3)

Jawabannya :

<P1 = 106°

<P2 = 74°

<P3 = 106°

<P4 = 74°

7. Nilai P3 adalah a. 37 b. 74 c. 106 d. 148


Jawab:

C.) 106°

Penjelasan dengan langkah-langkah:

Materi :      [Sudut & Garis]

Kelas :       8 SMP

Diketahui :    Suatu sudut 74°

(merupakan pasangan sudut luar berseberangan dengan P2)

*74° = P2

Ditanya :  P3 = ?

Jawab :

180° = P2 + P3

180° = 74° + P3

P3 = 180° - 74° = 106°
