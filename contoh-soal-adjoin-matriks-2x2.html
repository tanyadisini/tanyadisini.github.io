24. [124] 4. (Soal Teori Matriks) Jika diketahui matriks A = 3 -2 1 2 0 3 matriks A tersebut dengan Metode Adjoin! Tentukan Invers dari​


Penjelasan dengan langkah-langkah:

Untuk menentukan invers dari matriks A menggunakan metode adjoin, langkah-langkahnya adalah sebagai berikut:

1. Hitung determinan matriks A.

Determinan matriks A dapat dihitung dengan menggunakan aturan:

det(A) = (3 * 0 * 3) + (-2 * 2 * 1) + (1 * 2 * 0) - (1 * 0 * 1) - (3 * 2 * 0) - (0 * (-2) * 3)

= 0 + (-4) + 0 - 0 - 0 - 0

= -4

2. Hitung matriks adjoin A.

Matriks adjoin A diperoleh dengan menukar elemen-elemen diagonal utama (a dan d) dan mengubah tanda elemen-elemen di luar diagonal utama.

Matriks adjoin A dari matriks A = 3 -2 1 2 0 3 adalah:

adjoin(A) = 3 2 -1 -2 3 0

3. Hitung matriks invers A.

Matriks invers A dapat diperoleh dengan membagi matriks adjoin A dengan determinan matriks A.

Matriks invers A dari matriks A = 3 -2 1 2 0 3 adalah:

inverse(A) = (1/det(A)) * adjoin(A)

= (1/(-4)) * 3 2 -1 -2 3 0

= -3/4 -1/2 1/4 1/2 -3/4 0

Jadi, invers dari matriks A adalah:

inverse(A) = -3/4 -1/2 1/4

1/2 -3/4 0

= 0
