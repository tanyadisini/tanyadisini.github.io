6. Selisih panjang pita Hasna dan Vira 1,25 meter. Jika perbandingan panjang pita Hasna dan Vira 4 : 5, maka berapa meter panjang pita Vira?


Panjang pita Vira adalah 6,25 m.

Pembahasan

Misal perbandingan A dan B adalah m : n

m berarti rasio untuk A, dan n adalah rasio untuk B

Jumlah perbandingan (jumlah rasio) = m + n

Selisih perbandingan (selisih rasio) = m - n dengan m > n

Jumlah A dan B adalah K

Selisih A dan B adalah L

Maka berlaku rumus ↓

A = [tex]\frac{m}{m+n}[/tex] x K

B = [tex]\frac{n}{m+n}[/tex] x K

A = [tex]\frac{m}{m-n}[/tex] x L

B = [tex]\frac{n}{m-n}[/tex] x L

A = [tex]\frac{m}{n}[/tex] x B

B = [tex]\frac{n}{m}[/tex] x A

K = [tex]\frac{m+n}{m}[/tex] x A

K = [tex]\frac{m+n}{n}[/tex] x B

L = [tex]\frac{m-n}{m}[/tex] x A

L = [tex]\frac{m-n}{n}[/tex] x B

K = [tex]\frac{m+n}{m-n}[/tex] x L

L = [tex]\frac{m-n}{m+n}[/tex] x K

Penyelesaian Soal

Diketahui:

Selisih panjang pita Hasna dan Vira = 1,25 m

Perbandingan panjang pita Hasna dan Vira = 4 : 5

Ditanya:

Panjang pita Vira = ... ?

Jawab:

Panjang pita Vira = [tex]\frac{5}{5-4}[/tex] x 1,25 m

                             = [tex]\frac{5}{1}[/tex] x 1,25 m

                             = 5 x 1,25 m

                             = 6,25 m

Panjang pita Vira adalah 6,25 m.

Bu Vira akan mendapatkan keuntungan sebesar Rp 300.000,00. Soal tersebut merupakan soal tentang aritmetika sosial.

Penjelasan dengan langkah-langkah

Soal tersebut merupakan soal matematika yang membahas tentang aritmetika sosial. Aritmetika sosial merupakan cabang ilmu matematika yang membahas tentang keuangan atau biasanya digunakan untuk menghitung potongan harga.

Persamaan mencari harga beli per biji

b = B : n

Persamaan mencari harga jual per biji

j = J : n

Persamaan mencari keuntungan

U = (j-b) x n

Penyelesaian soal

Diketahui:

1 kodi = 20 buah1 lusin = 12 buahB = Rp 1.800.000/3 kodiJ = 420.000/ lusin

Ditanyakan:

Tentukan keuntungan yang didapat Bu Vira!

Jawab:

Mencari harga beli per biji

b = B : nb = Rp 1.800.000 : 60b = Rp 30.000/pcs

Mencari harga jual per biji

j = J : nj = Rp 420.000 : 12j = Rp 35.000/pcs

Mencari keuntungan

U = (j-b) x nU = (35.000 - 30.000) x 60U = 5.000 x 60U = Rp 300.000,00

Jadi, Bu Vira akan mendapatkan keuntungan sebesar Rp 300.000,00.

12. Selisih panjang pita Hasna dan Vira 1,25 meter jika perbandingan panjang pita Hasna dan Vira 4:5 maka berapa meter panjang pita Vira​


Jawaban:

6,25 meter

Penjelasan:

Diketahui:

Perbandingan panjang pita Husna dan Vira = 4 : 5 Selisih panjang pita keduanya adalah 1,25 meter

Ditanya:

Panjang pita Vira

Penyelesaian:

Angka banding Husna = 4

Angka banding Vira = 5

Panjang pita Vira =  [tex]\frac{5}{5-4} \times 1,25 \ meter[/tex]

Jadi panjang pita Vira = 6,25 meter.

Panjang pita Husna = [tex]\frac{4}{5-4} \times 1,25 \ meter[/tex]

Dan panjang pita Husna = 5 meter.

Bukti:

Pita Vira lebih panjang dari pada pita Husna, dengan selisih 6,25 m - 5 m = 1,25 m.

1 kodi = 20 buah

3 kodi = 60 buah = 5 lusin

harga beli 3 kodi = 1.800.000,00

harga jual 5 lusin ( 3 kodi ) = Rp.420.000,00 × 5

Untung = harga jual - harga beli
