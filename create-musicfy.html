Jawaban:

B. Create database, drop database, create table, drop table, insert, select, update, delete.

Penjelasannya:

Struktur dari basis data adalah Create database, drop database, create table, drop table, insert, select, update, delete. Struktur ini merupakan struktur dasar dari basis data yang digunakan untuk mengelola data dalam sebuah sistem basis data. Create database dan drop database digunakan untuk membuat dan menghapus database. Create table dan drop table digunakan untuk membuat dan menghapus tabel dalam database. Insert digunakan untuk memasukkan data ke dalam tabel. Select digunakan untuk mengambil data dari tabel. Update digunakan untuk memperbarui data yang sudah ada dalam tabel. Delete digunakan untuk menghapus data dari tabel. Oleh karena itu, jawaban yang tepat untuk pertanyaan ini adalah B. Create databese, drop database, create table, drop table, insert, select, update, delete.

34. Fasilitas apa saja yang terdapat pada panel create


Panel create dalam aplikasi 3D Max berisi fasilitas yang memungkinkan kita untuk membuat objek-objek tertentu. Objek tersebut terbagi dalam beberapa kategori, dimana masing-masingnya memiliki tombol sendiri.

Pembahasan:

Berikut ini adalah 7 kategori object yang bisa kamu ciptakan melalui panel create. Diantaranya adalah:

Geometry, berisi fasilitas untuk membuat berbagai bidang geometri. Ada box, sphere, pyramid dan bentuk lain yang lebih kompleks seperti booleans, lofts, tangga pintu dan masih banyak lagi.Shape, berisi fasilitas untuk membuat bentuk-bentuk ruang 2 dan 3 dimensi seperti kotak dan garis dan helix.Lights, berisi fasilitas untuk membentuk pencahayaan agar sebuah objek lebih terlihat realistis dan nyata.Camera, berisi fasilitas untuk mengatur angle atau sudut pandang agar object bisa ditampilkan dari berbagai sudut yang diinginkan.Helper, berisi fasilitas yang membantu kita untuk mengatur posisi objectSpace Warps, adalah fasilitas untuk membuat efek distorsi diantara ruang kosong antara object sat dengan object yang lainnyaSystems, yaitu fasililtas untuk mengontrol hirarki object berdasarkan perilakunya. Selain itu terdapat pula kontrol pencahayaan.
