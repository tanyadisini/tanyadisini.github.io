34. bukakan bahwa n3 +2n habis di bagi 3​


Jawaban:

Kita dapat membuktikan ini dengan menggunakan induksi matematika. Mari kita buktikan bahwa (n³ + 2n) habis dibagi 3 untuk setiap bilangan bulat n.

**Langkah Induksi:**

**Langkah Basis:**

Untuk n = 1, kita memiliki (1³ + 2.1 = 3), yang memang habis dibagi 3.

**Langkah Induksi:**

Assumsikan untuk suatu bilangan bulat positif (k), yaitu (k³ + 2k) habis dibagi 3.

**Langkah Induksi Maju:**

Kita ingin membuktikan untuk (k+1), yaitu ((k+1)³ + 2(k+1)) habis dibagi 3.

(k+1)³ + 2(k+1) = k³ + 3k² + 3k + 1 + 2k + 2 = (k³ + 2k) + 3k² + 3k + 3

Kita tahu bahwa (k³ + 2k) habis dibagi 3 berdasarkan asumsi induksi. Selain itu, (3k² + 3k + 3) juga habis dibagi 3 karena setiap suku memiliki faktor 3. Oleh karena itu, keseluruhan ekspresi juga habis dibagi 3.

Dengan demikian, berdasarkan prinsip induksi matematika, kita telah membuktikan bahwa (n³ + 2n) habis dibagi 3 untuk setiap bilangan bulat (n).
