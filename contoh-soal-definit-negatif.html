15. Diketahui fungsi f(x)=(a+1)x2-2ax+(a-2) definit negatif . Nilai a yang memenuhi adalah


Diketahui fungsi f(x) = (a + 1)x² – 2ax + (a – 2) definit negatif. Nilai a yang memenuhi adalah a < –2. Bentuk umum fungsi kuadrat adalah f(x) = Ax² + Bx + C, dengan a ≠ 0. Diskriman pada fungsi kuadrat berfungsi untuk menentukan apakah fungsi kuadrat tersebut memotong sumbu x atau tidak. Rumus diskriminan: D = B² – 4AC  

Jika D > 0, maka grafik fungsi kuadrat memotong sumbu x di dua titik Jika D = 0 maka grafik fungsi kuadrat memotong sumbu x di satu titik (menyinggung sumbu x) Jika D < 0 maka grafik fungsi kuadrat tidak memotong sumbu x (definit)

Ada dua jenis definit, yaitu

Definit positif jika D < 0 dan A > 0 Definit negatif jika D < 0 dan A < 0
Pembahasan

f(x) = (a + 1)x² – 2ax + (a – 2)

A = (a + 1) B = –2a  C = (a – 2)

f(x) definit negatif jika A < 0 dan D < 0


Syarat 1

A < 0

(a + 1) < 0

a < –1


Syarat 2

D < 0

B² – 4AC < 0

(–2a)² – 4(a + 1)(a – 2) < 0

4a² – 4(a² – 2a + a – 2) < 0

4a² – 4(a² – a – 2) < 0

4a² – 4a² + 4a + 8 < 0

4a < –8

a < –2


Irisan syarat 1 dan syarat 2

xxxxxxxxxxxxxx (–1) ..............

xxxxx (–2) .............................

a < –2

Jadi nilai a yang memenuhi agar f(x) definit negatif adalah a < –2
