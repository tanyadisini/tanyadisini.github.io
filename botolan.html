4. 14. Suatu produk air mineral dikemas dalam tiga jenis kemasan, yaitu botol kecil, botol sedang, dan botol besar. Perbandingan volume antara botol kecil dan botol sedang adalah 5 : 6. Perbandingan volume antara botol sedang dan botol besar adalah 3 : 5. Jika jumlah volume air dalam 1 botol kecil dan 1 botol besar 1.500 ml, tentukann volume air dalam setiap botol sedangpke cara ya kak​


Jawaban:

jawabannya adalah 600 ml

Konsep :

kita gunakan metode substitusi

Pembahasan

Misalkan x = botol kecil, y = botol sedang, dan z = botol besar.

Diketahui bahwa perbandingan antara volume botol kecil dan botol sedang adalah 5:6, maka dapat ditulis sebagai

x/y = 5/6

x = 5/6 y ... (1)

perbandingan antara volume botol sedang dan botol besar adalah 3:5, maka dapat ditulis sebagai

y/z = 3/5

y = 3/5 z ... (2)

jumlah volume air dalam 1 botol kecil dan 1 botol besar 1.500ml

x + z = 1.500 ... (3)

Langkah pertama kita substitusikan persamaan 2 ke persamaan 1, maka

x = 5/6 y

= 5/6 (3/5 z)

= (5 x 3)/(6 x 5) z

= 1/2 z

Kemudian kita substitusikan ke persamaan 3, maka

x + z = 1.500

1/2 z + z = 1.500

z + 2z = 3.000

3z = 3.000

z = 3.000/3

z = 1.000

kemudian lanjut dengan substitusikan ke persamaan 2, maka

y = 3/5 z

= 3/5 (1.000)

= 3.000/5

= 600

Sehingga dapat disimpulkan bahwa volume air dalam 1 botol sedang adalah 600 ml.

18. Sebuah produk air mineral dikemas dalam tiga jenis kemasan yaitu botol kecil, botol sedang, dan botol besar. Perbandingan antara volume botol kecil dan botol sedang adalah 5 : 6. Jawab:

600 ml

Penjelasan dengan langkah-langkah:

botol kecil : botol sedang = k : s = 5 : 6

--> s = 6/5 k

botol sedang : botol besar = s : b = 3 : 5

--> 6/5 k : b = 3 : 5

    b = 6/5 k x 5/3  

    b = 2 k

botol kecil + botol besar = k + b = 1.500 ml

k + 2k = 1.500 ml

k = 500 ml

Volume air dalam 1 botol sedang

= 6/5 k

= 6/5 x 500 ml

= 600 ml

40. 5 botol soda Dan 3 botol Mineral =Rp.61.500 3 botol soda Dan 5 botol mineral = Rp. 54.500 Ditanyakan = 1 botol soda Dan 1 botol mineral?


Jawaban:

semoga jawabannya membantu

Jawaban:

Diketahui :

5 botol soda + 3 botol mineral => Rp 61.500

3 botol soda + 5 botol mineral => Rp 54.500

Ditanyakan :

Harga 1 botol soda + 1 botol mineral ?

Jawab :

SPLDV (Sistem persamaan linear dua variabel), kita kerjakan dengan metode eliminasi.

Model matematika :

misal :

x => Botol Soda

y => Botol mineral

maka,

5x + 3y = 61.500

3x + 5y = 54.500

jadi, kita bisa mengeliminasi variabel x untuk mendapatkan nilai y

5x + 3y = 61.500 |×3|

3x + 5y = 54.500 |×5|

15x + 9y = 184.500

15x + 25y = 272.500

__________________ _

-16y = -88.000

y = (-88.000) / (-16)

y = 5.500

subsitusikan y = 5.500 ke persamaan :

3x + 5y = 54.500

3x + 5(5.500) = 54.500

3x + 27.500 = 54.500

3x = 54.500 - 27.500

3x = 27.000

x = 27.000 / 3

x = 9.000

Jadi, harga :

1 botol soda (x) = Rp 9.000

1 botol mineral (y) = Rp 5.500

maka, harga total keseluruhan :

= 1 botol soda + 1 botol mineral

= x + y

= (9.000) + (5.500)

= Rp 14.500
