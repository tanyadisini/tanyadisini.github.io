17. cara mengatasi laptop yang black screen gimana ya? please bantu ya.


1Masuk ke Safe Mode

Solusi awal yang bisa dilakukan sebagai cara mendapatkan Windows yang baik kembali adalah dengan masuk ke mode aman sistem  atau safe mode windows 7. Langkah untuk masuk ke Safe mode Windows 7 sendiri adalah sbb :

Nyalakan laptop;

Sebelum masuk tampilan logo Windows 7, tekan tombol F8, beberapa kali, hingga masuk menu Boot Option Windows 7;

Dari opsi yang ada, pilih Safe Mode;

Setelah tampil safe mode, tunggu sekitar 5-10 menit lalu restart laptop dan nyalakn kembali laptop, langkah ini pernah diuji dan memang jika masalahnya tidak parah bisa menjadikan laptop menyala normal kembali;

Biasanya masalah seperti ini diakibatkan oleh service Windows yang bermasalah atau timing service yang delay.


2. Instal ulang graphic (display) adapter

Fungsi dari graphic adapter sendiri adalah menampilkan output hasil processing ke layar monitor, dan rendering, meski sekilas tak ada kaitan langsung antara blank screen dengan kerja graphic adapter, tetapi sebenarnya masalah pada driver graphic bisa mengakibatkan sistem menjadi unstable dan menjadi penyebab blue screen  misalnya. Adapun untuk instal ulang graphic adapter langkahnya adalah sbb :

Nyalakan laptop;

Masuk ke safe mode seperti langkah diatas;

Tekan tombol keyboard CTRL + ALT + DELETE untuk memunculkan Task Manager;

Pada jendela taskmanager, tekan CTRL + N atau klik menu File > New kemudian ketik explorer.exe dan tekan Enter;

Setelah Windows Explorer muncul, klik ikon Computer pada panel kiri, dan klik Manage;

Pada jendela Computer management yang muncul, pada panel kiri, klik Device Manager;

Pada panel kanan, expand section Display adapter, klik kanan pada driver yang muncul dan klik Properties;

Pada jendela yang muncul, klik Update driver software, ikuti langkah yang diberikan, hingga selesai;

Pada tahap ini, jika instalasi driver sebelumnya memang memerlukan CD-ROM driver bawaan adapter atau dapat unduh dari internet, maka ada opsi untuk mengambil source instalasi driver dari disk;

Setelah proses selesai, restart laptop, jika masalah driver graphic selesai, seharusny Windows 7 sudah masuk Desktop.

Cara Pertama

Anda coba menekan tombol dengan logo Windows + P, setelah itu tekan tombol yang bergambar panah.

Kemudian lihat dan perhatikan pada windows laptop Anda, apakah kursor di layar keluar, apabila hal tersbut terjadi, tekan dan tahan tombol Spasi dan juga ketik pasword atau sandi, lalu tekan Enter.

Cara Kedua

Lepaskan atau copot Baterai Laptop Anda.

Cabut Casan Laptop Anda.

Tekan dengan menahan tombol Power sampai 10 detik.

Copt tombol Powernya.

Setelah itu pasang semua perangkat yang dilepaskan tadi, kemudian nyalakan kembali.

Cara Ketiga

Pada saat laptop Anda windows nya berlayar hitam atau blackscreen Anda coba dengan cara ini yaitu : Tahan dan tekan tombol ALT + DEL + CTRL dengan bersamaan, yang bertujuan untuk membuka Task Manager, apabila sudah terbuka, Klik File ~ New ~ lalu ketik explorer.exe

Apabila berhasil dengan selesai maka Anda harus melakukan System pada windows untuk di Restore dan aplikasi yang membuat tidak kompatible segera di Uninstal.

Cara Keempat

Yang pertama pada pengaturan windows Background Show harus aktif (Anda bisa lihat di settingan)

Setelah itu masuk ke setting dengan cara menekan tombol logo Window + I atau juga bisa ketik Setting di Menu Start

Kemudian Anda pindahkan jendelanya di Ease of access ~ Other Options. Dengan cara seperti ini yaitu buka Settings — > Lalu Ease of access — > dan Other Options.

Berikutnya harus menyala pada sklar Windows Background Show

Lalu Anda harus memilih thema sesuai keinginan Anda

Yang terakhir Anda harus selesaikan terlebih dahulu kerjaannya, setelah itu keluar dengan ketik Sign out dari laptop windows 10, setelah itu masuk kemabli dnegan Sign In, maka dengan ini layar dekstop sobat akan kembali normal.

27. bagai mana cara menscreen pada laptop asus ?


Menggunakan Tombol Print Screen

Pertama, Gadgeter dapat menentukan area yang akan di-screenshot pada layar laptop ASUS

Selanjutnya, Gadgeter dapat menekan tombol prt sc sysrq yang berada di keyboard

Nantinya secara otomatis gambar screenshot akan dihasilkan

Kemudian, Gadgeter dapat membuka aplikasi Microsoft Paint atau Adobe Photoshop atau pengolah gambar lainnya

Setelah aplikasi terbuka maka Gadgeter tinggal melakukan paste dengan kombinasi Ctrl + V di keyboard

Nantinya gambar hasil screenshot akan muncul dan dapat langsung diedit atau disimpan dengan format jpg, png, bmp, dan masih banyak lagi di Harddisk.

Menggunakan Kombinasi Tombol Win + Print screen Pertama, Gadgeter dapat menentukan area yang akan di-screenshot pada layar laptop ASUS

Kemudian, Gadgeter bisa menekan tombol berlogo Windows + prt sc sysrq secara bersamaan dalam waktu beberapa detik

Hasilnya secara otomatis akan disimpan dalam folder Screenshots yang berada di dalam folder Pictures yang bisa diakses dengan Windows Explorer

Menggunakan software Snipping Tool

Snipping Tool merupakan software atau tool bawaan yang sudah hadir dalam Windows. Dengan menggunakan tool ini, Gadgeter dapat melakukan berbagai macam model screenshot dibandingkan kedua cara di atas.

Pertama, Gadgeter dapat menentukan area yang akan di-screenshot pada layar laptop ASUS

Selanjutnya, Gadgeter dapat mencari software Snipping Tool di Start Menu

Setelah software ini dibuka, Gadgeter akan dihadapkan pada menu Snipping Tool

Untuk memulai screenshot, Gadgeter dapat memilih menu mode dan akan dihadapkan pada 4 mode sekaligus yang terdiri dari Free-form Snip, Rectangular Snip, Windows Snip, dan Full-screen Snip

Setelah memilih mode screenshot yang diinginkan maka Gadgeter dapat langsung melakukan screenshot

Hasilnya akan muncul di software Snipping Tool

Nantinya Gadgeter hanya tinggal memilih menu File > Save as dan hasil screenshot dapat langsung disimpan di folder yang telah ditentukan

30. beberapa tombol keyboard seperti enter,shift,capslock,numlock,spacebar,tab dan print screen merupakan bagian dari


beberapa tombol keyboard yang sebutkan merupakan bagian dari tombol-tombol utama pada keyboard komputer. Berikut ini adalah penjelasan tentang masing-masing tombol tersebut:

Enter: Tombol Enter digunakan untuk memberi isyarat ke sistem bahwa Anda telah selesai mengetik sebuah perintah atau teks, sehingga sistem akan memproses perintah atau teks tersebut.Shift: Tombol Shift digunakan untuk mengaktifkan huruf kapital (uppercase) atau karakter khusus pada tombol-tombol lain yang terdapat di sebelah kiri tombol Shift.Caps Lock: Tombol Caps Lock digunakan untuk mengaktifkan huruf kapital (uppercase) secara permanen. Sehingga semua huruf yang diketik setelah tombol Caps Lock diaktifkan akan ditulis dengan huruf kapital.Num Lock: Tombol Num Lock digunakan untuk mengaktifkan atau menonaktifkan penggunaan tombol-tombol angka di bagian kanan keyboard. Jika tombol Num Lock diaktifkan, tombol-tombol angka tersebut akan berfungsi sebagai tombol angka, sedangkan jika tombol Num Lock dinonaktifkan, tombol-tombol tersebut akan berfungsi sebagai tombol kursor.Spacebar: Tombol Spacebar digunakan untuk memberikan jarak antar kata atau untuk memindahkan kursor ke kanan sejauh satu spasi.Tab: Tombol Tab digunakan untuk memindahkan kursor ke kanan sejauh satu tab atau untuk mengatur tingkat paragraf dalam dokumen teks.Print Screen: Tombol Print Screen digunakan untuk mengambil tangkapan layar (screenshot) dari layar komputer Anda. Hasil tangkapan layar tersebut dapat disimpan sebagai file gambar atau dicetak langsung.

33. Jelaskan secara singkat tentang silk screen woodcut Intaglio print​


Silk screen, woodcut, dan intaglio print adalah teknik cetak yang berbeda-beda.

Silk screen printing adalah teknik cetak di mana tinta ditekan melalui kain jala halus (screen) yang ditempatkan di atas media cetak seperti kertas atau kain. Pola atau gambar hasil cetak terbentuk melalui bagian yang tidak tertutup oleh jala.

Woodcut printing adalah teknik cetak di mana gambar diukir pada permukaan kayu, dan bagian yang tidak diukir diisi dengan tinta. Kemudian, permukaan cetak kayu ditekan pada media cetak untuk mencetak gambar.

Intaglio printing adalah teknik cetak di mana gambar diukir atau digores pada permukaan logam atau piringan akrilik. Tinta kemudian dioleskan ke permukaan cetak dan digosok dengan kain lembut untuk mengisi goresan. Kemudian, media cetak ditekan ke permukaan cetak untuk mencetak gambar.

Semua teknik cetak ini memiliki keunikan dan kelebihan masing-masing, dan biasanya dipilih berdasarkan efek visual yang diinginkan dan media cetak yang digunakan.
