13. Jelaskan definisi AFC, AVC, MC, TC dan TR !


Jawaban:

Penjelasan:

Biaya Produksi

Biaya Total (TC = Total Cost) = biaya variable total (TVC) + biaya tetap total (TFC)

a. TVC : biaya yang berubah seiring perubahan output yang diproduksi. Contoh : pembayaran BAHAN BAKU, BAHAN BAKAR, TENAGA KERJA

b. TFC : biaya yang tidak berubah karena perubahan output. Contoh : pembayaran SEWA, PENYUSUTAN GEDUNG PERALATAN, PREMI ASURANSI, GAJI MANAJER PUNCAK

Contoh :

a. TFC : Rp. 300.000

b. TVC : 3Q + ½ Q2

c. TC : ½ Q2 +3Q + 300.000

d. Contoh kasus: perusahaan memproduksi sebanyak 500 satuan (Q). Berapa biaya total nya (TC) ?  

TC = ½ (500)2  + 3 (500) + 300.000

e. Contoh kasus: tidak berproduksi. Berapa TC ?

ganti Q dengan 0  

Biaya tetap rata-rata (AFC) : AFC =TFC/Q

Biaya variabel rata-rata (AVC) : AVC= TVC/Q

Pendapatan produksi

Laba Maks dalam Pasar :

1. TR (Total Revenue) = P x Q (hasil penjualan produk)

2. AR (Average Revenue) = TR/Q

3. MR (Marginal Revenue) : tambahan penerimaan sebagai hasil penjualan satu unit produk lagi  

MR = TRn – TRn-1  

4. MC (Marginal Cost) = TCn – TCn-1

5. ATC (Average Total Cost) = TC/Q

28. diketahui TR = Q kuadrat + 14Q , TC = 2Q kuadrat + 2 Q + 4 ditanya AC, TC, TR, MC


Diketahui :
TR = Q² + 14Q
TC = 2Q² + 2Q + 4

Ditanya : 
AC, TC, TR, MC

Jawab :
Cari terlebih dahulu fungsi MR & MC untuk mencari berapa Q (kuantitas/unit)

MR = TR'
MR = 2Q + 14 

MC = TC'
MC = 4Q + 2

    MR = MC
2Q + 14 = 4Q + 2
14 - 2 = 4Q - 2Q
     12 = 2Q
       6 = Q

TR = Q² + 14Q
      = (6)² + 14(6)
      = 36 + 84
      = 120

TC = 2Q² + 2Q + 4
      = 2(6)² + 2(6) + 4
      = 72 + 12 + 4
      = 88

AC = TC ÷ Q
      = 88 ÷ 6
      = 14,67

MC = 4Q + 2
       = 4(6) + 2
       = 24 + 2
       = 26

Jadi, AC = 14,67 ; TC = 88 ; TR = 120 ; MC = 26


*Catatan 
Tanda (') = Turunan
