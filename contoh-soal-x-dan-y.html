36. tlg kasi contoh soal cos (x+y) plz urgent


Jawab:

Berikut adalah contoh soal dan jawaban dari cos(x + y):

Penjelasan dengan langkah-langkah:

Contoh Soal:

Hitung nilai dari cos(x + y) jika diketahui bahwa cos(x) = 3/5 dan cos(y) = 4/5.

Jawaban:

Untuk menyelesaikan soal ini, kita dapat menggunakan rumus identitas trigonometri, yaitu:

cos(x + y) = cos(x) * cos(y) - sin(x) * sin(y)

Langkah-langkahnya adalah sebagai berikut:

Diketahui cos(x) = 3/5 dan cos(y) = 4/5.

Kita perlu mencari nilai sin(x) dan sin(y). Untuk itu, kita dapat menggunakan rumus Pythagoras:

sin^2(x) = 1 - cos^2(x)

sin^2(y) = 1 - cos^2(y)

sin^2(x) = 1 - (3/5)^2 = 1 - 9/25 = 16/25

sin^2(y) = 1 - (4/5)^2 = 1 - 16/25 = 9/25

sin(x) = √(16/25) = 4/5

sin(y) = √(9/25) = 3/5

Sekarang, kita dapat menghitung nilai cos(x + y) menggunakan rumus identitas trigonometri:

cos(x + y) = cos(x) * cos(y) - sin(x) * sin(y)

cos(x + y) = (3/5) * (4/5) - (4/5) * (3/5)

cos(x + y) = 12/25 - 12/25

cos(x + y) = 0

Jadi, nilai dari cos(x + y) adalah 0.
