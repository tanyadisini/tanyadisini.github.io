30. Sebuah perusahaan memproduksi X unit barang perhari dengan biaya x3 - 600x2 + 112.500x rupiah.Banyaknya unit barang yang harus diproduksi setiap harinya supaya biaya produksi menjadi minimal adalah


biaya produksi : p(x) = x³ - 600x² + 112.500x

biaya produksi minimum :
p'(x) = 0

p'(x) = 3x² - 1200x + 112.500
0 = 3x² - 1200x + 112.500
0 = x² - 400x + 37.500
0 = (x - 250)(x - 150)
x = 250 atau x = 150

• untuk x = 250
p(250) = (250)³ - 600(250)² + 112.500(250)
p(250) = 6.250.000

• untuk x = 150
p(150) = (150)³ - 600(150)² + 112.500(150)
p(150) = 6.750.000

maka, agar biaya produksi menjadi minimal, maka banyaknya unit barang yg harus diproduksi adalah 250

Operasi hitung campuran adalah operasi hitung yang terdiri dari perkalian, pembagian, penjumlahan dan pengurangan.

Dalam operasi hitung campuran dahulukan yang dalam kurung, perkalian dan pembagian. Penjumlahan dan pengurangan itu sama kuat, jadi kerjakan mulai dari kiri.

Perkalian dan pembagian juga sama-sama kuat, jadi kerjakan mulai dari kiri dahulu.
